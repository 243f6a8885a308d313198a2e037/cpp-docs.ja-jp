# <a name="endcolumnmap"></a>END_COLUMN_MAP
列マップ エントリの末尾をマークします。  
  
## <a name="syntax"></a>構文  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## <a name="remarks"></a>コメント  
 行セットで、1 つのアクセサーを持つために使用します。 `BEGIN_COLUMN_MAP` マクロは `END_COLUMN_MAP` マクロで終了します。
