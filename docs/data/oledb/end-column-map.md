# END_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

列マップ エントリの終了位置を示します。  
  
## 構文  
  
```  
  
END_COLUMN_MAP( )  
  
```  
  
## 解説  
 この行セットの一つのアクセサーとともに使用されます。  `BEGIN_COLUMN_MAP` マクロは `END_COLUMN_MAP` マクロについて説明します。  
  
## 使用例  
 [BEGIN\_COLUMN\_MAP](../Topic/BEGIN_COLUMN_MAP.md)を参照してください。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h
