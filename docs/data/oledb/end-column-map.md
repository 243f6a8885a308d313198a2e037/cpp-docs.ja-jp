# <a name="endcolumnmap"></a>END_COLUMN_MAP
列マップ エントリの末尾をマークします。  
  
## <a name="syntax"></a>構文  
  
```cpp
END_COLUMN_MAP()  
  
```  
  
## <a name="remarks"></a>コメント  
 行セットで、1 つのアクセサーを持つために使用します。 `BEGIN_COLUMN_MAP` マクロは `END_COLUMN_MAP` マクロで終了します。

 参照してください[BEGIN_COLUMN_MAP](../../data/oledb/begin-column-map.md)です。
