# CBookmark::GetBuffer
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ブックマーク バッファーへのポインターを取得します。  
  
## 構文  
  
```  
  
virtual BYTE* GetBuffer( ) const throw( );  
  
```  
  
## 戻り値  
 ブックマーク バッファーへのポインター。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h
