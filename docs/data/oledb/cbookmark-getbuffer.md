# <a name="cbookmarkgetbuffer"></a>CBookmark::GetBuffer
ブックマーク バッファーへのポインターを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp
virtual BYTE* GetBuffer() const throw();  
  
```  
  
## <a name="return-value"></a>戻り値  
 ブックマーク バッファーへのポインター。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** atldbcli.h
