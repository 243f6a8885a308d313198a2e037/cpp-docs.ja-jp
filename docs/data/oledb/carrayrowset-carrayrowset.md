# <a name="carrayrowsetcarrayrowset"></a>CArrayRowset::CArrayRowset
新しい `CArrayRowset` オブジェクトを作成します。  
  
## <a name="syntax"></a>構文  
  
```cpp
      CArrayRowset(int nMax = 100000);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `nMax`  
 [in]行セット内の行の最大数。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** atldbcli.h
