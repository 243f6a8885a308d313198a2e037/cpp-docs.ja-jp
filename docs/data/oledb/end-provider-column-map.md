# END_PROVIDER_COLUMN_MAP
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プロバイダーの列マップ エントリの終了位置を示します。  
  
## 構文  
  
```  
  
END_PROVIDER_COLUMN_MAP( )  
  
```  
  
## 使用例  
 [BEGIN\_PROVIDER\_COLUMN\_MAP](../../data/oledb/begin-provider-column-map.md)を参照してください。  
  
## 必要条件  
 **ヘッダー:** atldb.h
