# CAccessorBase::GetNumAccessors
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

クラスによって作成されたアクセサーの数を取得します。  
  
## 構文  
  
```  
  
ULONG GetNumAccessors( ) const;  
  
```  
  
## 戻り値  
 クラスによって作成されたアクセサーの数。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h
