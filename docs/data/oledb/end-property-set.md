# END_PROPERTY_SET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プロパティ セットの終了位置を示します。  
  
## 構文  
  
```  
  
END_PROPERTY_SET(  
guid   
)  
  
```  
  
#### パラメーター  
 `guid`  
 \[\]プロパティの GUID。  
  
## 使用例  
 [BEGIN\_PROPSET\_MAP](../Topic/BEGIN_PROPSET_MAP.md)を参照してください。  
  
## 必要条件  
 **ヘッダー:** atldb.h
