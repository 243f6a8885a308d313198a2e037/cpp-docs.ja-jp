# <a name="endpropertyset"></a>END_PROPERTY_SET
プロパティ セットの末尾をマークします。  
  
## <a name="syntax"></a>構文  
  
```cpp
END_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>パラメーター  
 `guid`  
 [in]プロパティの GUID です。  
  
## <a name="example"></a>例  
 「 [BEGIN_PROPSET_MAP](../../data/oledb/begin-propset-map.md)」を参照してください。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** atldb.h
