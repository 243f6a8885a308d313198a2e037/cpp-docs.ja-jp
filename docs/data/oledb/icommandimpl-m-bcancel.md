# <a name="icommandimplmbcancel"></a>ICommandImpl::m_bCancel
コマンドが取り消されたかどうかを示します。  
  
## <a name="syntax"></a>構文  
  
```cpp
unsigned m_bCancel:1;  
  
```  
  
## <a name="remarks"></a>コメント  
 この変数を取得することができます、 **Execute**コマンド クラスと適切なキャンセルのメソッドです。

 **ヘッダー:** atldb.h
