# <a name="crowsetgetdatahere"></a>CRowset::GetDataHere
現在の行からデータを取得し、指定されたバッファーに配置します。  
  
## <a name="syntax"></a>構文  
  
```  
  
      HRESULT GetDataHere(   
   int nAccessor,   
   void* pBuffer    
) throw( );  
```  
  
#### <a name="parameters"></a>パラメーター  
 `nAccessor`  
 [in]データにアクセスするのに使用するアクセサーのインデックス番号します。  
  
 `pBuffer`  
 [out]現在のレコードのデータを配置する先のバッファー。  
  
## <a name="return-value"></a>戻り値  
 標準の `HRESULT`。
