# CRowset::Close
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

リリース [IRowset](https://msdn.microsoft.com/en-us/library/ms720986.aspx) 行と現在のインターフェイス。  
  
## 構文  
  
```  
  
void Close( ) throw( );  
  
```  
  
## 解説  
 このメソッドは、行セット内のすべての行を現在解放します。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h
