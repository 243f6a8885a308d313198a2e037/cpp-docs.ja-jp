# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プロパティ ID を設定してプロパティ ID を追加します。  
  
## 構文  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### パラメーター  
 `propid`  
 \[\]プロパティ ID 設定に追加するプロパティ ID。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h
