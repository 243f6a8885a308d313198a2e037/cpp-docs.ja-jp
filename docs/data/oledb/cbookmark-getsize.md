# <a name="cbookmarkgetsize"></a>CBookmark::GetSize
ブックマークのバッファーのサイズを取得します。  
  
## <a name="syntax"></a>構文  
  
```cpp
virtual DBLENGTH GetSize() const throw();  
  
```  
  
## <a name="return-value"></a>戻り値  
 バッファーのサイズ (バイト単位)。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** atldbcli.h
