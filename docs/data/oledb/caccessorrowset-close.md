# <a name="caccessorrowsetclose"></a>CAccessorRowset::Close
アクティブなアクセサーと行セットを解放します。  
  
## <a name="syntax"></a>構文  
  
```cpp
void Close();  
  
```  
  
## <a name="remarks"></a>コメント  
 関連付けられているメモリを解放します。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** atldbcli.h
