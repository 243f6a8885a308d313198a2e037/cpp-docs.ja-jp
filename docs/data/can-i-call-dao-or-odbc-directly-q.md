# DAO または ODBC の直接呼び出し
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC の通例として、DAO や ODBC をクラスからアクセスするだけでなく直接呼び出すこともできます。直接呼び出す方が、細かく制御できます。  MFC は Windows のプログラミングを簡略化することを目的としていますが、基礎となる API への直接アクセスを妨げることはありません。
