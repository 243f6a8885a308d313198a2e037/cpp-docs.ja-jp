# データベース トピックに関する Microsoft サポート技術情報の文書の検索
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC の ODBC に関するサポート技術情報の文書を検索するには、MFCDATABASE というキーワードで検索します。  
  
 MFC の DAO に関するサポート技術情報の文書を検索するには、MFCDAO というキーワードで検索します。
