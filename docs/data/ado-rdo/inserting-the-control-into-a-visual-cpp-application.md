# Visual C++ アプリケーションにコントロールを挿入する
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ツール バーまたは [&#91;ActiveX コントロールの挿入&#93;](../Topic/Insert%20ActiveX%20Control%20Dialog%20Box.md) ダイアログ ボックスを使用して、ActiveX コントロールをダイアログ ボックスに挿入できます。  
  
### ツールボックスで ActiveX コントロールを挿入するには  
  
1.  ツールボックス内の何もない領域上で右クリックします。  
  
2.  ショートカット メニューの \[ツールボックスのカスタマイズ\] をクリックし、必要なコントロールを選択します。  
  
3.  ダイアログ エディターで、ダイアログ ボックスにコントロールをドラッグします。  
  
### ダイアログ エディターで ActiveX コントロールを挿入するには  
  
1.  ダイアログ ボックスを右クリックします。  
  
2.  ショートカット メニューの [&#91;ActiveX コントロールの挿入&#93;](../Topic/Insert%20ActiveX%20Control%20Dialog%20Box.md) をクリックします。  
  
    > [!NOTE]
    >  **\[ActiveX コントロールの挿入\]** を使用してプロジェクトに ActiveX コントロールを挿入する場合、[ラッパー クラス](../../data/ado-rdo/wrapper-classes.md)はプロジェクトに含まれません。  コントロールの機能をカスタマイズするには、ラッパー クラスを作成する必要があります。
