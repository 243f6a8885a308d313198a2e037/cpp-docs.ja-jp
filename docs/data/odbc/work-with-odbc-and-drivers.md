# <a name="work-with-odbc-and-drivers"></a>ODBC と ODBC ドライバーの使用
ODBC では、1 つのデータベース管理システム (DBMS) に依存していないアプリケーションを作成することができます。 ユーザーは、適切な DBMS に固有の ODBC ドライバーがある複数の Dbms に操作できる 1 つのアプリケーションを記述できます。
