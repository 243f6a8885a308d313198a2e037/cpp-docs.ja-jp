# ODBC と ODBC ドライバーの使用
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ODBC を使うと、1 つの DBMS \(Database Management System\) に依存しないアプリケーションを作成できます。  つまり、複数の DBMS 上で操作できるアプリケーションを作成できます。ユーザーは、所有する ODBC ドライバーが対応する DBMS を通じて、そのアプリケーションにアクセスできます。
