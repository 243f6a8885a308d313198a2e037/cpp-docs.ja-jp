# ODBC ドライバーの一覧
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ では、次のデータベースに対応する ODBC ドライバーが用意されています。  
  
-   SQL Server  
  
-   Microsoft Access  
  
-   Microsoft Excel  
  
-   dBASE  
  
-   Paradox  
  
-   Microsoft Oracle ODBC  
  
-   テキスト ファイル  
  
 ODBC Driver Pack など、マイクロソフトおよび他社から入手できる ODBC ドライバーについては、[マイクロソフト サポート サービス](GettingHelpFromMicrosoftProductSupportServices)にご連絡ください。
