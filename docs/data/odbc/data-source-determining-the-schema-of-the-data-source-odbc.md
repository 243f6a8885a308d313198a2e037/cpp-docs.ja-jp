# データ ソース: データ ソースのスキーマの判定 (ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

このトピックの内容は、MFC ODBC クラスに該当します。  
  
 `CRecordset` オブジェクトのデータ メンバーを設定するには、接続先のデータ ソースのスキーマを特定する必要があります。  データ ソースのスキーマを判定するには、データ ソース内のテーブルのリスト、各テーブルの列のリスト、各列のデータ型、およびインデックスの有無を調べます。
