# データベース クラスを使う場面
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

多くのアプリケーションでは、データをデータベースに格納する必要があります。また、データベースを使用することによって便利になるアプリケーションもあります。  データベースは、柔軟なデータ リポジトリとして使用できます。多くの場合、複数のユーザーおよび複数のアプリケーションからアクセスできます。  データベースには大量のデータを格納でき、クエリおよび更新対象のデータに高速アクセスできます。
