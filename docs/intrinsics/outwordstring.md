# <a name="outwordstring"></a>__outwordstring
**Microsoft 固有の仕様**  
  
 生成、`rep outsw`命令で、送信`Count`で始まる単語`Buffer`出力で指定された I/O ポート`Port`です。  
  
## <a name="syntax"></a>構文  
  
```  
void __outwordstring(   
   unsigned short Port,   
   unsigned short* Buffer,   
   unsigned long Count   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Port`  
 データを送信するポート。  
  
 [入力] `Buffer`  
 指定したポートを送信するデータへのポインター。  
  
 [入力] `Count`  
 送信する文字数です。
