# <a name="writecr3"></a>__writecr3
**Microsoft 固有の仕様**  
  
 値を書き込む`Data`CR3 レジスタにします。  
  
## <a name="syntax"></a>構文  
  
```  
void writecr3(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Data`  
 CR3 レジスタに書き込む値。
