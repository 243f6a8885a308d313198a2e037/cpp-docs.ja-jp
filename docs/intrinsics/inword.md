# <a name="inword"></a>__inword
**Microsoft 固有の仕様**  
  
 使用して、指定されたポートからデータを読み取り、`in`命令します。  
  
## <a name="syntax"></a>構文  
  
```  
unsigned short __inword(  
   unsigned short Port  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Port`  
 読み取りにポートです。

 読み取られるデータの単語です。
