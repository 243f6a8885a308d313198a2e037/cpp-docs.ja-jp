# <a name="writecr0"></a>__writecr0
**Microsoft 固有の仕様**  
  
 値を書き込む`Data`CR0 レジスタにします。  
  
## <a name="syntax"></a>構文  
  
```  
void writecr0(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Data`  
 CR0 レジスタに書き込む値。
