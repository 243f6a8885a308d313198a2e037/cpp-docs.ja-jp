# <a name="writecr4"></a>__writecr4
**Microsoft 固有の仕様**  
  
 値を書き込む`Data`CR4 レジスタにします。  
  
## <a name="syntax"></a>構文  
  
```  
void writecr4(   
   unsigned __int64 Data   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Data`  
 CR4 レジスタに書き込む値。
