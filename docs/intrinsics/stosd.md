# <a name="stosd"></a>__stosd
**Microsoft 固有の仕様**  
  
 ストアの文字列の命令が生成されます (`rep stosd`)。  
  
## <a name="syntax"></a>構文  
  
```  
void __stosd(   
   unsigned long* Dest,   
   unsigned long Data,   
   size_t Count   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
*追加先*<br/>
[out]操作の転送先。  
  
*データ*<br/>
[in]格納するデータ。  
  
*カウント*<br/>
[in]書き込むダブルワードのブロックの長さ。  
  
## <a name="requirements"></a>要件  
  
|組み込み|アーキテクチャ|  
|---------------|------------------|  
|`__stosd`|x86、x64|  
  
 **ヘッダー ファイル** \<intrin.h >  
  
## <a name="remarks"></a>Remarks  
 結果はダブルワード`Data`のブロックに書き込まれる`Count`によって示されるメモリ位置にダブルワード`Dest`します。  
  
 このルーチンは、組み込みとしてのみ使用できます。  
  
## <a name="example"></a>例  
  
```  
// stosd.c  
// processor: x86, x64  
  
#include <stdio.h>  
#include <memory.h>  
#include <intrin.h>  
  
#pragma intrinsic(__stosd)  
  
int main()  
{  
    unsigned long val = 99999;  
    unsigned long a[10];  
  
    memset(a, 0, sizeof(a));  
    __stosd(a+1, val, 2);  
  
printf_s( "%u %u %u %u",  
              a[0], a[1], a[2], a[3]);   
}  
```  
  
```Output  
0 99999 99999 0  
```
