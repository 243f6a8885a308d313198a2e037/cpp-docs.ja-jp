# <a name="readcr3"></a>__readcr3
**Microsoft 固有の仕様**  
  
 CR3 レジスタを読み取り、その値を返します。  
  
## <a name="syntax"></a>構文  
  
```  
unsigned __int64 __readcr3(void);  
```  
  
## <a name="return-value"></a>戻り値  
 CR3 レジスタの値。
