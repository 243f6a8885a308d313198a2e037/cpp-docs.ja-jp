# <a name="readcr4"></a>__readcr4

**Microsoft 固有の仕様**

CR4 レジスタを読み取り、その値を返します。

## <a name="syntax"></a>構文

```
unsigned __int64 __readcr4(void);
```

## <a name="return-value"></a>戻り値

CR4 レジスタの値。

## <a name="requirements"></a>必要条件

|組み込み|アーキテクチャ|
|---------------|------------------|
|`__readcr4`|x86、x64|

**ヘッダー ファイル** \<intrin.h >

## <a name="remarks"></a>Remarks

この組み込みはカーネル モードのみで使用でき、そのルーチンは組み込みとしてのみ使用できます。
