# <a name="indwordstring"></a>__indwordstring
**Microsoft 固有の仕様**  
  
 使用して、指定されたポートからデータを読み取り、`rep insd`命令します。  
  
## <a name="syntax"></a>構文  
  
```  
void __indwordstring(  
   unsigned short Port,  
   unsigned long* Buffer,  
   unsigned long Count  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 [入力] `Port`  
 読み取りにポートです。  
  
 [出力] `Buffer`  
 ポートから読み取るデータは、ここで書き込まれます。  
  
 [入力] `Count`  
 読み取るデータのバイト数。
