# <a name="initializer-lists"></a>初期化子リスト
コンス トラクターの初期化子リストは、基底クラス コンス トラクターの前に今すぐと呼ばれます。  
  
## <a name="remarks"></a>コメント  
 Visual C 2005 では、前に、基本クラス コンス トラクターが c++ マネージ拡張でコンパイルするときに初期化子リストの前に呼び出されました。 ここでコンパイルするときに**/clr**、初期化子リストが最初に呼び出されます。
