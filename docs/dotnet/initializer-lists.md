# <a name="initializer-lists"></a>初期化子リスト

コンス トラクターの初期化子リストは基底クラスのコンス トラクターの前と呼ばれるようになりました。

## <a name="remarks"></a>Remarks

Visual C 2005 では、前に、基本クラス コンス トラクターは、c++ マネージ拡張でコンパイルするときに初期化子リストの前に呼び出されました。 ここでコンパイルするときに **/clr**、初期化子リストが最初に呼び出されます。
