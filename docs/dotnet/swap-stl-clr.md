# swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

最初のオブジェクトから 1 への 2 番目のオブジェクトのコンテンツと 2 番目のコンテンツを割り当てる 2 種類のオブジェクト間で要素の値を交換します。  
  
## 構文  
  
```  
<class _Ty> inline  
    void swap(_Ty% _Left, _Ty% _Right);  
```  
  
## 解説  
 この関数は、STL 関数 `swap`も同じように動作します。  詳細については、「[swap](../Topic/swap.md)」を参照してください。  
  
## 必要条件  
 **ヘッダー:** の \<cliext\/アルゴリズム\>  
  
 **名前空間:** の cliext
