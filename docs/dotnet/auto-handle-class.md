# auto_handle クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

マネージ型に仮想ハンドルを埋め込むために使用できる自動リソース管理。  
  
## 構文  
  
```  
template<typename _element_type>  
ref class auto_handle;  
```  
  
#### パラメーター  
 `_element_type`  
 埋め込むマネージ型。  
  
## 必要条件  
 **ヘッダー ファイル** \<msclr\\auto\_handle.h\>  
  
 **名前空間** の msclr
