# <a name="utility-stlclr"></a>utility (STL/CLR)
STL/CLR ヘッダーを含める`<cliext/utility>`テンプレート クラスを定義する`pair`といくつかのサポート テンプレート関数。  
  
## <a name="syntax"></a>構文  
  
```  
#include <utility>  
```

|クラス|説明|  
|-----------|-----------------|  
|[pair (STL/CLR)](../dotnet/pair-stl-clr.md)|要素のペアをラップします。|  
  
|演算子|説明|  
|--------------|-----------------|  
|[operator== (pair) (STL/CLR)](../dotnet/operator-equality-pair-stl-clr.md)|ペア比較します。|  
|[operator!= (pair) (STL/CLR)](../dotnet/operator-inequality-pair-stl-clr.md)|等しくない比較をペアにします。|  
|[operator< (pair) (STL/CLR)](../dotnet/operator-less-than-pair-stl-clr.md)|ペアの比較よりも小さいです。|  
|[演算子\<= (組) (STL/CLR)](../dotnet/operator-less-or-equal-pair-stl-clr.md)|以下のペアの比較できます。|  
|[operator> (pair) (STL/CLR)](../dotnet/operator-greater-than-pair-stl-clr.md)|比較よりも大きい値のペア。|  
|[operator>= (pair) (STL/CLR)](../dotnet/operator-greater-or-equal-pair-stl-clr.md)|ペアより大きいまたは等しい比較します。|
