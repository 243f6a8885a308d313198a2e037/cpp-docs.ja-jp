# タブ コントロールの使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

タブ コントロール[CTabCtrl](../Topic/CTabCtrl%20Class.md) \(\) を使用する最も簡単な方法は、ダイアログ エディターを使用してダイアログ テンプレート リソースに追加することもできます。  また、タブ コントロールを単独で使用できます。  MFC が自動的に **InitCommonControls** を呼び出します。  主要なタスクは次のとおりです。:  
  
-   [タブ コントロールの作成](../mfc/creating-the-tab-control.md)  
  
-   [タブ コントロールにタブを追加します。](../mfc/adding-tabs-to-a-tab-control.md)  
  
-   [タブ コントロールの通知メッセージの処理](../mfc/processing-tab-control-notification-messages.md)  
  
 内部タブ コントロール オブジェクトが親ビューまたはダイアログ クラスに埋め込まれている場合、コントロールは、親が破棄されると、破棄されます。
