# Making Controls the Same Width, Height, or Size
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

最も優先度の高いコントロールのサイズに基づいて、コントロール グループのサイズを変更できます。  また、[キャプション テキストの大きさに基づいてコントロールのサイズを変更](../mfc/sizing-individual-controls.md)できます。  
  
### 複数のコントロールを同じ幅、高さ、またはサイズにするには  
  
1.  サイズを変更する[コントロールを選択](../mfc/selecting-multiple-controls.md)します。  
  
     複数のコントロールを選択した場合は、最初に選択したものが最も優先度の高いコントロールになります。  グループ内のコントロールの最終的なサイズは、最も優先度の高いコントロールのサイズによって決まります。  最も優先度の高いコントロールの選択の詳細については、「[最も優先度の高いコントロールの指定](../mfc/specifying-the-dominant-control.md)」を参照してください。  
  
2.  \[書式\] メニューの \[同じサイズに揃える\] を選択し、以下のコマンドのいずれかを選択します。  
  
    -   **Both**  
  
    -   **高さ**  
  
    -   **幅**  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。マネージ プロジェクトにリソース ファイルを手動で追加する方法、リソースへのアクセス方法、静的なリソースの表示方法、およびリソース文字列をプロパティに割り当てる方法については、「[チュートリアル : Windows フォームのローカリゼーション](http://msdn.microsoft.com/ja-jp/9a96220d-a19b-4de0-9f48-01e5d82679e5)」および「[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)」を参照してください。  
  
 要件  
  
 Win32
