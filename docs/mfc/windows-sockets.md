# Windows ソケット
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ここでは、Windows ソケットの MFC 実装について説明します。  MFC は、Windows ソケット API を使用したプログラミング ネットワーク アプリケーションをサポートするための 2 種類のクラスが用意されています。  クラス [CAsyncSocket](../Topic/CAsyncSocket%20Class.md) は高度なネットワーク プログラマの最も強力で柔軟性を提供する 1 の Windows ソケット API 1 をカプセル化します。  クラス [CSocket](../mfc/reference/csocket-class.md) は [CArchive](../mfc/reference/carchive-class.md) オブジェクトとの間でデータをシリアル化するには、簡単なインターフェイスを提供します。
