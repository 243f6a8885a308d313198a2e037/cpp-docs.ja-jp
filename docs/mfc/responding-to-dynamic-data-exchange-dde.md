# ダイナミック データ エクスチェンジ (DDE) への応答
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

フレーム ウィンドウがファイル マネージャーからのファイルにダイナミック データ エクスチェンジ \(DDE\) の要求に \(ファイル拡張子をアプリケーションに登録するか、または関連付けられている場合\) 応答できます。  [シェル登録](../mfc/special-cwinapp-services.md)を参照してください。
