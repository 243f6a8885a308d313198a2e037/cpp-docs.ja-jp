# ヘッダー コントロールのヘッダー項目
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

外観の多くのコントロール、およびヘッダー コントロール \([CHeaderCtrl](../Topic/CHeaderCtrl%20Class.md)\) を構成するヘッダー項目の動作が異なります。  各ヘッダー項目に関連付けられているイメージ リストのビットマップ イメージ、文字列、イメージ、または関連付けられるアプリケーション定義の 32 ビット値を持つことができます。  文字列、ビットマップ、イメージは、ヘッダー項目に表示されます。  
  
 既存の項目を変更するか、[CHeaderCtrl::InsertItem](../Topic/CHeaderCtrl::InsertItem.md) 呼び出しを行うか、[CHeaderCtrl::GetItem](../Topic/CHeaderCtrl::GetItem.md) と [CHeaderCtrl::SetItem](../Topic/CHeaderCtrl::SetItem.md)への呼び出しとして作成した新しい項目の外観とコンテンツをカスタマイズできます。
