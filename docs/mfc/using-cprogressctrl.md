# CProgressCtrl の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

時間のかかる操作の進行状況を示すためにプログレス コントロールを使用できます。  処理の進行状況に合わせてシステムの強調表示色で塗りつぶされていくのが四角形です。  
  
 プログレス コントロールはクラス [CProgressCtrl](../mfc/reference/cprogressctrl-class.md)で MFC で表されます。  
  
 最初にプログレス コントロールを作成するときに、サイズ、位置、親ウィンドウ \(通常はダイアログ ボックス\)、および ID を指定します。  `dwStyle` パラメーターを使用して、指定の方法のために満たすまたはコントロールに関するさまざまなウィンドウ スタイルとスタイルを指定できます。
