# <a name="scroll-bar-styles"></a>スクロール バー スタイル
-   **SBS_BOTTOMALIGN**併用、 **SBS_HORZ**スタイル。 スクロール バーの下端がで指定された四角形の下端に位置合わせ、**作成**メンバー関数。 スクロール バーが、システムのスクロール バーの既定の高さ。  
  
-   **SBS_HORZ**水平スクロール バーを指定します。 どちらの場合、 **SBS_BOTTOMALIGN**も**SBS_TOPALIGN**スタイルを指定すると、スクロール バーの高さ、幅、および位置で指定されたが、**作成**メンバー関数。  
  
-   **SBS_LEFTALIGN**併用、 **SBS_VERT**スタイル。 スクロール バーの左エッジはで指定された四角形の左端に揃えられます、**作成**メンバー関数。 スクロール バーが、システムのスクロール バーの既定の幅。  
  
-   **SBS_RIGHTALIGN**併用、 **SBS_VERT**スタイル。 スクロール バーの右端がで指定された四角形の右端に揃えて配置、**作成**メンバー関数。 スクロール バーが、システムのスクロール バーの既定の幅。  
  
-   **SBS_SIZEBOX**サイズ ボックスを指定します。 どちらの場合、 **SBS_SIZEBOXBOTTOMRIGHTALIGN**も**SBS_SIZEBOXTOPLEFTALIGN**スタイルを指定すると、サイズ ボックスには、高さ、幅、および位置で指定された、**作成**メンバー関数。  
  
-   **SBS_SIZEBOXBOTTOMRIGHTALIGN**併用、 **SBS_SIZEBOX**スタイル。 [サイズ] ボックスの右下隅がで指定された四角形の右下隅に位置合わせ、**作成**メンバー関数。 サイズのボックスには、システムのサイズ ボックスの既定のサイズがあります。  
  
-   **SBS_SIZEBOXTOPLEFTALIGN**併用、 **SBS_SIZEBOX**スタイル。 [サイズ] ボックスの左上隅がで指定された四角形の左上隅に位置合わせ、**作成**メンバー関数。 サイズのボックスには、システムのサイズ ボックスの既定のサイズがあります。  
  
-   **SBS_SIZEGRIP**と同じ**SBS_SIZEBOX**、ですが、浮き出しします。  
  
-   **SBS_TOPALIGN**併用、 **SBS_HORZ**スタイル。 スクロール バーの上端がで指定された四角形の上端に位置合わせ、**作成**メンバー関数。 スクロール バーが、システムのスクロール バーの既定の高さ。  
  
-   **SBS_VERT**垂直スクロール バーを指定します。 どちらの場合、 **SBS_RIGHTALIGN**も**SBS_LEFTALIGN**スタイルを指定すると、スクロール バーの高さ、幅、および位置で指定されたが、**作成**メンバー関数。
