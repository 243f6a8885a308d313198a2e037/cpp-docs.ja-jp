# <a name="data-types-mfc"></a>データ型 (MFC)
このトピックでは、Microsoft Foundation Class ライブラリで最もよく使用されるデータ型を示します。 データ型のほとんどは、他のユーザーは MFC 固有プラットフォーム ソフトウェア開発キット (SDK) のと同じです。  
  
 Windows SDK と MFC の両方で使用されるデータ型については、次を参照してください。 [Windows Data Types](http://msdn.microsoft.com/library/windows/desktop/aa383751)です。  
  
 Microsoft Foundation Class ライブラリに固有のデータ型を以下に示します。  
  
-   **位置**値をコレクション内の要素の位置を示すために使用する以外の場合は MFC コレクション クラスで使用します。  
  
-   **LPCRECT**定数 (不変) を 32 ビット ポインター`RECT`構造体。
