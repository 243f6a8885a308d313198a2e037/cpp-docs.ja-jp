# OLE の初期化
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

アプリケーションで OLE システム サービスを使用する前に、OLE システム DLL を初期化し、DLL のバージョンが正しいかどうかを確認する必要があります。  **AfxOleInit** 関数は、OLE システム DLL を初期化します。
