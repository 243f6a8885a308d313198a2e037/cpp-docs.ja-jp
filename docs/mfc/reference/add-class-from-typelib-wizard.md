# <a name="add-class-from-typelib-wizard"></a>Typelib クラス追加ウィザード
このウィザードを使用すると、利用可能なタイプ ライブラリからの MFC クラスを追加します。 ウィザードでは、選択したタイプ ライブラリから追加する各インターフェイスのクラスを作成します。  
  
 **クラスを追加します。**  
 クラスの作成元のタイプ ライブラリの場所を指定します。

|------------|-----------------|  
|**Registry**|タイプ ライブラリは、システムに登録されます。 登録されているタイプ ライブラリは、「**使用可能なタイプ ライブラリ**です。|  
|**ファイル**|タイプ ライブラリは、システムに必ずしも登録されていませんが、ファイルに含まれます。 ファイルの場所を指定する必要があります**場所**です。|

 現在システムに登録されているタイプ ライブラリを一覧表示します。 そのインターフェイスを表示するには、この一覧から、タイプ ライブラリを選択、**インターフェイス** ボックスの一覧です。
