# CSliderCtrl の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[CSliderCtrl](../mfc/reference/csliderctrl-class.md) クラスは、トラック バーというスライダー コントロールを表します。  「スライダー コントロール」はスライダーと省略可能な目盛りを含むウィンドウです。  ユーザーが、スライダーを使用して移動するとマウスまたは方向キーは、スライダー コントロール変更を表示するには、通知メッセージを送信します。  
  
 スライダー コントロールはユーザーの範囲の連続する値を独立した値または複数選択するときに便利です。  たとえば、ユーザーが特定の目盛りにスライダーを移動すると、キーボードの繰り返し間隔を設定できるようにするために、スライダー コントロールを使用する場合があります。  
  
 スライダー コントロールでスライダーを使用して、指定のインクリメントで作成時に移動します。  スライダー コントロールは 5 の範囲が必要であることを、指定した場合、スライダーは 6 個の位置を占めるだけです。: スライダー コントロールの左端の位置と範囲の各インクリメントの 1 種類の位置。  通常、これらの位置は、識別されます。
