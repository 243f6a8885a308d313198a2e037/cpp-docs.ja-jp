# MFC テクニカル ノート
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

テクニカル ノートは、プログラマがプログラマに記述するドキュメントです。  
  
 それぞれのテクニカル ノートは MFC ドキュメントの他の範囲を超える機能または問題について説明します。  指定されたテクニカル ノートはユーザーから情報要求が反映され、MFC の開発者が上級ユーザーを予測できる特殊な情報が必要です。  
  
 テクニカル ノートを参照する方法が 2 つあります:  
  
-   [数テクニカル ノート](../mfc/technical-notes-by-number.md)  
  
-   [カテゴリ別のテクニカル ノート](../mfc/technical-notes-by-category.md)
