# <a name="mfc-technical-notes"></a>MFC テクニカル ノート
テクニカル ノートは、プログラマがプログラマ向けに書かれてドキュメントです。  
  
 各テクニカル ノートでは、問題や、他の MFC のドキュメントの範囲を超えている機能について説明します。 テクニカル ノートでは、特殊な情報を MFC には、開発者は、上級ユーザーは必要な予測と同様に、ユーザーからの情報の要求を反映します。  
  
 テクニカル ノートを参照する 2 つの方法があります。  
  
-   [番号順テクニカル ノート](../mfc/technical-notes-by-number.md)  
  
-   [カテゴリ別テクニカル ノート](../mfc/technical-notes-by-category.md)
