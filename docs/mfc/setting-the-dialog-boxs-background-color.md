# <a name="setting-the-dialog-boxs-background-color"></a>ダイアログ ボックスの背景色の設定
ダイアログ ボックスの背景色を設定するには、ダイアログ ボックス ウィンドウに WM_CTLCOLOR メッセージを処理します。 指定されたダイアログ ボックスのみを設定する色が使用されます。
