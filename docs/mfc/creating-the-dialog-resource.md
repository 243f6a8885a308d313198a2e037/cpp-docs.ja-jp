# <a name="creating-the-dialog-resource"></a>ダイアログ リソースの作成

-   表示されたときに、ダイアログ ボックスがある場所とサイズを調整します。  
  
-   コントロール パレットからさまざまな種類のコントロールをドラッグし、好きな場所 ダイアログ ボックスにドロップします。  
  
-   ツールバーの配置のボタンを備えたコントロールを配置します。  
  
-   プログラムでが動作と外観をシミュレートすることで、ダイアログ ボックスをテストします。 テスト モードでは、テキスト ボックスにテキストを入力して、プッシュ ボタンをクリックすると、ダイアログ ボックスのコントロールを操作しにできます。  
  
 完了したら、ダイアログ テンプレート リソースは、アプリケーションのリソース スクリプト ファイルに格納されます。 編集できますが後で必要な場合。 完全作成し、ダイアログ リソースを編集する方法については、次を参照してください。、[ダイアログ エディター](../windows/dialog-editor.md)トピックです。 この手法はダイアログ テンプレート リソースを作成にも使用[CFormView](../mfc/reference/cformview-class.md)と[CRecordView](../mfc/reference/crecordview-class.md)クラスです。
