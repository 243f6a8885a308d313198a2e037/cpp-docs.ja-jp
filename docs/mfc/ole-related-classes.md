# <a name="ole-related-classes"></a>OLE 関連クラス
これらのクラスは、いくつかの例外からの入力と出力をファイルに至るまで、さまざまなサービスを提供します。  
  
 [COleObjectFactory](../mfc/reference/coleobjectfactory-class.md)  
 その他のコンテナーから要求されたときに項目を作成するために使用します。 このクラスは複数の特定の種類を含む、ファクトリの基本クラスとして機能`COleTemplateServer`します。  
  
 [関数](../mfc/reference/colemessagefilter-class.md)  
 同時実行と OLE Lightweight リモート プロシージャ コール (LRPC) を管理するために使用します。  
  
 [関数](../mfc/reference/colestreamfile-class.md)  
 COM を使用して`IStream`インターフェイスを提供する`CFile`複合ファイルへのアクセス。 このクラス (から派生した`CFile`) OLE 構造化ストレージを使用する MFC シリアル化を有効にします。  
  
 [CRectTracker](../mfc/reference/crecttracker-class.md)  
 移動、サイズ変更、および埋め込み先アイテムの方向を許可するために使用します。
