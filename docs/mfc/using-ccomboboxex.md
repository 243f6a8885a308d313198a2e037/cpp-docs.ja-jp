# CComboBoxEx の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

拡張コンボ ボックスのコントロールは、項目のネイティブ イメージをサポートする標準コンボ ボックス コントロールの拡張です。  これらのイメージが現在選択および選択解除になった項目などのコンボ ボックスの項目の状態を示すために使用できます。  項目のイメージを簡単にアクセスできるようにするには、コントロールはイメージ リストをサポートします。  
  
 項目グラフィックスを描画する手動でコンボ ボックスの機能を提供するには、このコントロールを使用します。
