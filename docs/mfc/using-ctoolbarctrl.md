# CToolBarCtrl の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ボタンと省略可能な領域を含むコントロール ウィンドウを作成するには、ツール バー コントロール \([CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md)\) を使用できます。  ToolBar コントロールのウィンドウの各ボタンが親ウィンドウにユーザーが選択するようにコマンド メッセージを送信します。  通常、ツール バーのボタンは、アプリケーションのメニュー項目に対応し、追加やそのほかの直接的な方法は、アプリケーションのコマンドにアクセスするユーザーについて説明します。
