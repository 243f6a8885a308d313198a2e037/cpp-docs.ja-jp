# リスト コントロールのスクロール、整列、並べ替え、検索
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

リスト コントロール \([CListCtrl](../Topic/CListCtrl%20Class.md)\) が既定でスクロールできます。  詳細については、[!INCLUDE[winSDK](../atl/includes/winsdk_md.md)] と [スクロール](../Topic/CListCtrl::Scroll.md) のメンバー関数の [スクロール位置](http://msdn.microsoft.com/library/windows/desktop/bb774736) を参照します。  
  
 コントロールのリスト項目を配置し、項目を並べ替え、特定の項目を検索するに `CListCtrl` のメンバー関数を呼び出すことができます。  詳細については、[!INCLUDE[winSDK](../atl/includes/winsdk_md.md)] と [CListCtrl](../Topic/CListCtrl%20Class.md) のメンバー [並べ替え](../Topic/CListCtrl::Arrange.md)、[SortItems](../Topic/CListCtrl::SortItems.md)と [FindItem](../Topic/CListCtrl::FindItem.md)の [、配列、および検索並べ替えます。](http://msdn.microsoft.com/library/windows/desktop/bb774736) を参照します。
