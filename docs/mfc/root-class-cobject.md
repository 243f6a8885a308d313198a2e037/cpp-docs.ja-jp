# ルート クラス : CObject
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft Foundation Class \(MFC\) ライブラリ クラスの大部分は、クラスの階層構造のルートの単一の基本クラスから派生されます。  `CObject` は 非常に小さいオーバーヘッドがそこから派生されるすべてのクラスには、多数の便利な機能を提供します。  `CObject` と機能に関する詳細については、「[CObject の使い方](../mfc/using-cobject.md)」を参照してください。  
  
 [CObject](../Topic/CObject%20Class.md)  
 ほとんどの MFC クラスの最終的な基本クラスです。  データのシリアル化とクラスのランタイム情報を取得するサポート。  
  
 [CRuntimeClass](../Topic/CRuntimeClass%20Structure.md)  
 実行時にオブジェクトの正確なクラスを決定するために使われる構造体。
