# ダイアログ ボックスの背景色の設定
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ダイアログ ボックス ウィンドウの `WM_CTLCOLOR` メッセージの処理がダイアログ ボックスの背景色を設定できます。  設定する色は指定ダイアログ ボックスだけに使用されます。
