# リモート オートメーションのスレッド化モデル
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

リモート オートメーションは、COM シングルスレッド アパートメント スレッドとモデルの両方をサポートします。  Visual C\+\+ がサポートするすべてのページでは、32 ビットと 64 ビットのプラットフォームで、スレッド モデルは「アパートメントに設定する必要があります」。これを使用する方法と場所にの詳細についてはするに [リモート オートメーションを使用するプログラムを作成します。](../mfc/creating-programs-that-use-remote-automation.md)を参照します。
