# 印刷でのビューの役割
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ビューは、関連付けられているドキュメントの印刷の 2 種類の重要な役割を果たします。  
  
 ビュー:  
  
-   [OnDraw](../Topic/CView::OnDraw.md) 同じコードを画面の描画に関してプリンターで描画するために使用します。  
  
-   印刷時のページにドキュメントを区切る Manages。  
  
 印刷と印刷のビューの役割に関する詳細については、「[印刷と印刷プレビュー\(P\)](../mfc/printing-and-print-preview.md)」を参照してください。
