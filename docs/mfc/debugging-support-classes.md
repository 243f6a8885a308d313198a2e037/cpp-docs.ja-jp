# デバッグ サポート クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC は動的なメモリ割り当て問題のデバッグに役立つ次のクラスが用意されています。  
  
 [CDumpContext](../mfc/reference/cdumpcontext-class.md)  
 診断ダンプに対象を提供します。  
  
 [CMemoryState](../mfc/reference/cmemorystate-structure.md)  
 構成します。メモリ使用のスナップショットを提供できます。  また、高速、後のメモリ スナップショットを比較するために使用します。
