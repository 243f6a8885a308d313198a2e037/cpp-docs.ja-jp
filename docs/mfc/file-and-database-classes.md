# ファイルとデータベース クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

これらのクラスは、データベースまたはディスク ファイルに情報を格納することができます。  3 組のデータベース クラス— OLE DB、同様の機能を提供する ODBC と DAO があります。  OLE DB のグループは OLE DB を使用して実装され、OLE DB コンシューマー テンプレートを使用して、DAO のグループは Data Access Object を使用して実装され、ODBC のグループは Open Database Connectivity を使用して実装されます。  また、一連の処理の標準ファイルのクラス、アクティブなストリーム、および HTML ストリームがあります。  
  
 クラスの次のカテゴリはデータの永続性をサポートします。

-   [OLE DB クラス](../mfc/ole-db-classes.md)  
  
-   [DAO クラス](../mfc/dao-classes.md)  
  
-   [ODBC クラス](../Topic/ODBC%20Classes.md)
