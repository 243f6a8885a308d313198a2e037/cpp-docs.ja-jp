# OLE DB クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC の OLE DB サポートはクラス [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md)から現在構成されています。  `COleDBRecordView` は 直接 [CRowset](../Topic/CRowset%20Class.md) オブジェクトにアタッチされるフォーム ビューでコントロールのデータベース レコードが表示されます。  OLE DB コンシューマー テンプレートの詳細については、「[OLE DB コンシューマー テンプレートの一覧](../data/oledb/ole-db-consumer-templates-reference.md)」を参照してください。
