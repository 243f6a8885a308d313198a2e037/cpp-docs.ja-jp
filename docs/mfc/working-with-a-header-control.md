# ヘッダー コントロールの操作
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ヘッダー コントロール[CHeaderCtrl](../Topic/CHeaderCtrl%20Class.md) \(\) を使用する簡単な方法は、リスト コントロールとともに、; このトピックのファミリの [CListCtrl を使用する](../Topic/Using%20CListCtrl.md) "を参照してください。  また、ヘッダー コントロールを単独で使用できます。  MFC が自動的に **InitCommonControls** を呼び出します。  主要なタスクは次のとおりです。:  
  
-   [ヘッダー コントロールの作成](../mfc/creating-the-header-control.md)  
  
-   [ヘッダー コントロールに項目を追加します。](../mfc/adding-items-to-the-header-control.md)  
  
-   [ヘッダー項目コントロールの順序](../mfc/ordering-items-in-the-header-control.md)  
  
-   [ヘッダー コントロールの通知の処理](../mfc/processing-header-control-notifications.md)  
  
 ヘッダー コントロール オブジェクトが親ビューまたはダイアログ クラスに埋め込まれている場合、コントロールは、親が破棄されると、破棄されます。
