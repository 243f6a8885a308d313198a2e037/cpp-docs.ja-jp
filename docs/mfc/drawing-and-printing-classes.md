# 描画と印刷クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Windows では、すべてのグラフィカルな出力は、デバイス コンテキスト \(DC\) という仮想描画領域に描画されます。  MFC は DCs のさまざまな型を、ビットマップをカプセル化するクラス、ブラシ、パレット、ペンなどの Windows のペイント ツールのカプセル化を提供します。
