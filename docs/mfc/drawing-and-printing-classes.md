# <a name="drawing-and-printing-classes"></a>描画および印刷クラス
Windows では、すべてのグラフィカルな出力はデバイス コンテキスト (DC) と呼ばれる仮想描画領域に描画されます。 MFC では、さまざまな種類のビットマップ、ブラシ、パレット、およびペンなどの Windows 描画ツールのカプセル化できるだけでなく、Dc をカプセル化するクラスを提供します。
