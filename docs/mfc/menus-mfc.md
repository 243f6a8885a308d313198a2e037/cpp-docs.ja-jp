# <a name="menus-mfc"></a>メニュー (MFC)
MFC では、メニューを操作するときに 2 つの要素を指定します。  
  
-   クラス[CMenu](../mfc/reference/cmenu-class.md)実行時に、プログラムのメニューを操作するためです。 ドキュメントを使用して`CMenu`と使用方法について説明するサンプル`CMenu`効果的にします。  
  
-   メニューとツールバーのボタンを更新するメカニズム: を有効にするか、現在のプログラムの条件に合わせてその場で無効にしています。  
  
 Visual C ではまた、[メニュー エディター](../windows/menu-editor.md)を作成したり、プログラムのメニュー リソースを編集します。
