# CTabCtrl の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

"タブ コントロール" は、ノートブックの仕切りページや、ファイル キャビネットにあるラベル付きフォルダーと似ています。  タブ コントロールは、[CTabCtrl](../Topic/CTabCtrl%20Class.md) クラスで表されます。タブ コントロールを使用すると、複数のページに及ぶ情報やコントロールを一度に 1 つずつ表示できます。しかも、各ページ間の論理的な関係がわかるような形式になっています。  
  
 タブ コントロールの詳細については、[!INCLUDE[winSDK](../atl/includes/winsdk_md.md)] の「[Tab Controls](http://msdn.microsoft.com/library/windows/desktop/bb760550)」を参照してください。
