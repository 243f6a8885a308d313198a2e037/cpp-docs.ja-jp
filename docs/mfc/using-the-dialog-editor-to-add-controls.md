# ダイアログ エディターを使用したコントロールの追加
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[ダイアログ エディター](../mfc/dialog-editor.md)とダイアログ テンプレート リソースを作成すると、コントロールのパレットからコントロールをドラッグし、ダイアログ ボックスにドロップします。  つまり、ダイアログ テンプレート リソースを基にそのコントロール型の仕様を追加します。  ダイアログ オブジェクトを構築し、**作成** または `DoModal` メンバー関数を呼び出すと、フレームワークはダイアログ ウィンドウで画面の Windows コントロールと場所を作成します。  
  
 必要に応じて [コントロールを手動で作成します。](../mfc/adding-controls-by-hand.md) 代わりにできます。  これは、追加の作業です。
