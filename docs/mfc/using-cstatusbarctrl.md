# CStatusBarCtrl の使い方
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

さまざまな種類のアプリケーションに関するステータス情報を反映するコントロール ウィンドウを作成するには、ステータス バー コントロール \([CStatusBarCtrl](../mfc/reference/cstatusbarctrl-class.md)\) を使用できます。  ステータス ウィンドウは情報の種類を表示する部分に分割できます。
