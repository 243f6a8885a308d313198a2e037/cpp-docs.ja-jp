# リモート オートメーションのユーザー コンポーネント
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

各クライアント コンピューターに必要なクライアント プログラムとサポート DLL が含まれていることを確認する必要があります。  また、サーバー コンピューターで必要なサーバー アプリケーションおよびサポート DLL ことを確認する必要があります。  最後に、接続を構成するに RAC マネージャーを実行する前にサーバー プログラムが各クライアント コンピューターに登録されていることを確認する必要があります。  プログラムが自動登録の \(ほとんどとしては、\) の場合、登録するクライアント コンピューター上のプログラムを実行するだけです。  それが失敗して、提供したり、手動でレジストリを編集できます。ファイルを実行する必要があります。
