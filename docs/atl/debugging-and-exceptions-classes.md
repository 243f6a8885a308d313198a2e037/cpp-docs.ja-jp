# デバッグと例外クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

これらのクラスは、例外処理とデバッガーのサポートを提供します。  
  
-   [CAtlDebugInterfacesModule](../atl/reference/catldebuginterfacesmodule-class.md) は、インターフェイスをデバッグするには、このクラスをサポートします。  
  
-   [CAtlException](../Topic/CAtlException%20Class.md) このクラスは、ATL の例外を定義します。
