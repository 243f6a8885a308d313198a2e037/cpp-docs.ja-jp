# 永続性クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

次のクラスは、を実装するオブジェクトの永続性:  
  
-   [IPersistPropertyBagImpl](../atl/reference/ipersistpropertybagimpl-class.md) は、クライアントがプロパティ バッグにオブジェクトのプロパティを読み込み、保存することもできます。  
  
-   [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) は、クライアントがストリームにオブジェクトの永続データを読み込み、保存することもできます。  
  
-   [IPersistStorageImpl](../atl/reference/ipersiststorageimpl-class.md) は、クライアントがストレージにオブジェクトの永続データを読み込み、保存することもできます。
