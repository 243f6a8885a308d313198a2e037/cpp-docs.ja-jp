# MMC スナップイン クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

次のクラスは Microsoft 管理コンソールの \(MMC\) 用のスナップイン コンポーネントをサポートします:  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md) は、メニュー項目とツール バーを追加し、適切なハンドラー関数にスナップイン ノードのコマンドを転送などのスナップイン ノード オブジェクトを実装します。  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md) には、スナップイン プロパティ ページ オブジェクトを実装します。
