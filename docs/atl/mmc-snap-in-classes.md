# <a name="mmc-snap-in-classes"></a>MMC スナップインでクラス
次のクラスは、開発の Microsoft 管理コンソール (MMC) スナップインでコンポーネントをサポートを提供します。  
  
-   [CSnapInItemImpl](../atl/reference/csnapinitemimpl-class.md)メニュー項目やツールバーを追加して、適切なハンドラー関数にスナップイン ノード用のコマンドの転送などのスナップイン ノード オブジェクトを実装します。  
  
-   [CSnapInPropertyPageImpl](../atl/reference/csnapinpropertypageimpl-class.md)スナップイン プロパティ ページ オブジェクトを実装します。
