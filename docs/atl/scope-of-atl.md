# ATL の適用範囲
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL COM オブジェクトを簡単に、オートメーション サーバーと ActiveX コントロールを作成できるようにします。  ATL は、基本的な COM インターフェイスの多くに組み込みサポートを提供します。  
  
 ATL は、アプリケーションに含めるソース・コードとして提供されます。  コンポーネント間で共有できるコードを含む ATL は、DLL の使用可能 \(atl90.dll\) を作成します。  ただし、この DLL は必要ではありません。
