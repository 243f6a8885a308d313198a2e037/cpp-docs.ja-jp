# レジストリ データ エクスチェンジに関するマクロ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらのマクロはレジストリ データ エクスチェンジ操作を実行します。  
  
|||  
|-|-|  
|[BEGIN\_RDX\_MAP](../Topic/BEGIN_RDX_MAP.md)|レジストリ データ エクスチェンジ マップの開始位置を示します。|  
|[END\_RDX\_MAP](../Topic/END_RDX_MAP.md)|レジストリ データ エクスチェンジ マップの終了位置を示します。|  
|[RDX\_BINARY](../Topic/RDX_BINARY.md)|指定したレジストリ エントリを BYTE 型の指定したメンバー変数に関連付けます。|  
|[RDX\_CSTRING\_TEXT](../Topic/RDX_CSTRING_TEXT.md)|指定したレジストリ エントリを CString 型の指定したメンバー変数に関連付けます。|  
|[RDX\_DWORD](../Topic/RDX_DWORD.md)|指定したレジストリ エントリを DWORD 型の指定したメンバー変数に関連付けます。|  
|[RDX\_TEXT](../Topic/RDX_TEXT.md)|指定したレジストリ エントリを TCHAR 型の指定したメンバー変数に関連付けます。|
