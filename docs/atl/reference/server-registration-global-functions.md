# サーバー登録に関するグローバル関数
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらの関数はオブジェクト マップの登録および登録解除サーバー オブジェクトをサポートします。  
  
> [!IMPORTANT]
>  次の表に示す関数は [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]で実行されるアプリケーションで使用することはできません。  
  
|||  
|-|-|  
|[AtlComModuleRegisterServer](../Topic/AtlComModuleRegisterServer.md)|オブジェクト マップのオブジェクトをすべて登録します。|  
|[AtlComModuleUnregisterServer](../Topic/AtlComModuleUnregisterServer.md)|オブジェクト マップのオブジェクトの登録をすべて解除します。|  
|[AtlComModuleRegisterClassObjects](../Topic/AtlComModuleRegisterClassObjects.md)|この関数は、クラス オブジェクトを登録するために呼び出されます。|  
|[AtlComModuleRevokeClassObjects](../Topic/AtlComModuleRevokeClassObjects.md)|この関数は、COM のモジュールのクラス オブジェクトをキャンセルするために呼び出されます。|  
|[AtlComModuleGetClassObject](../Topic/AtlComModuleGetClassObject.md)|この関数は、クラス オブジェクトを取得します。|
