# ピクセルと HIMETRIC との間の変換に関するグローバル関数
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらの関数は、ピクセル、HIMETRIC 単位への変換をサポートします。  
  
> [!IMPORTANT]
>  次の表に示す関数は [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)]で実行されるアプリケーションで使用することはできません。  
  
|||  
|-|-|  
|[AtlHiMetricToPixel](../Topic/AtlHiMetricToPixel.md)|ピクセルに変換します。HIMETRIC 単位 \(各単位は 0.01 mm\)。|  
|[AtlPixelToHiMetric](../Topic/AtlPixelToHiMetric.md)|HIMETRIC 単位に変換します。ピクセル \(各単位は 0.01 mm\)。|
