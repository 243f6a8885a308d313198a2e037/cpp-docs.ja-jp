# インターフェイス、ATL コントロール ウィザード
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ウィザードのこのページでは、コントロールがサポートするインターフェイスを指定します。  既定では、サポートされるインターフェイスは通常ほとんどのコンテナーで使用されます。  
  
> [!NOTE]
>  [&#91;オプション&#93;](../../atl/reference/options-atl-control-wizard.md) タブの \[ライト コントロール\] を選択すると、既定では \[サポートあり\] ボックスにインターフェイスが表示されません。  
  
 **サポートなし**  
 現在サポートされていないが使用できる、コントロール用のインターフェイスを指定します。  
  
 **サポート状況**  
 現在サポートされている、コントロール用のインターフェイスを指定します。  
  
|転送ボタン|説明|  
|-----------|--------|  
|**\>**|\[サポートなし\] ボックスで現在選択されているインターフェイス名を \[サポートあり\] ボックスに追加します。|  
|**\>\>**|\[サポートなし\] ボックスのすべてのインターフェイス名を \[サポートあり\] ボックスに追加します。|  
|**\<**|\[サポートあり\] ボックスで現在選択されているインターフェイス名を削除します。|  
|**\<\<**|\[サポートあり\] ボックスに現在表示されているインターフェイス名をすべて削除します。|
