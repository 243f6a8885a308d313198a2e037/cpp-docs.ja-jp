# <a name="atl-wizards-and-dialog-boxes"></a>ATL ウィザードとダイアログ ボックス
Active Template Library (ATL) ウィザードでは、COM オブジェクトのさまざまな種類の定型コードを生成します。 プロジェクトのショートカット メニューを開き、ウィザードを実行する**ソリューション エクスプ ローラー**を選択して**追加**、**クラス**します。
