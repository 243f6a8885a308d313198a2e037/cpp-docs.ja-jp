##  <a name="declare_olemisc_status"></a>  DECLARE_OLEMISC_STATUS  
 ATL の ActiveX コントロールで使用すると、入りますフラグを設定できます。  
  
```

```

 *miscstatus*  
 該当するすべての入りますフラグです。
