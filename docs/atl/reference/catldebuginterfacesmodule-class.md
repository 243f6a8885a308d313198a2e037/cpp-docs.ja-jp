# CAtlDebugInterfacesModule クラス
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

このクラスは、デバッグ インターフェイスをサポートします。  
  
## 構文  
  
```  
  
class CAtlDebugInterfacesModule  
  
```  
  
## 解説  
 `CAtlDebugInterfacesModule` は、インターフェイスのデバッグに必要なサポートが用意されています。  これは [\_ATL\_DEBUG\_QI](../Topic/_ATL_DEBUG_QI.md)シンボルを定義するすべてのプロジェクトに含まれます。  
  
## 必要条件  
 **ヘッダー:** atlbase.h
