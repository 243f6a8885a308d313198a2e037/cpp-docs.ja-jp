# <a name="catldebuginterfacesmodule-class"></a>CAtlDebugInterfacesModule クラス
このクラスは、デバッグのインターフェイスのサポートを提供します。  
  
## <a name="syntax"></a>構文  
  
```
class CAtlDebugInterfacesModule
```  
  
## <a name="remarks"></a>コメント  
 `CAtlDebugInterfacesModule`デバッグのインターフェイスに必要なサポートを提供します。 シンボルを定義する他のプロジェクトに含まれている[_ATL_DEBUG_QI](http://msdn.microsoft.com/library/4f1feddf-1469-42ad-a2f7-5a816d1c0bc5)します。
