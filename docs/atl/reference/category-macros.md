# カテゴリに関するマクロ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらのマクロは、カテゴリ マップを定義します。  
  
|||  
|-|-|  
|[BEGIN\_CATEGORY\_MAP](../Topic/BEGIN_CATEGORY_MAP.md)|カテゴリ マップの開始位置を示します。|  
|[END\_CATEGORY\_MAP](../Topic/END_CATEGORY_MAP.md)|カテゴリ マップの終了位置を示します。|  
|[IMPLEMENTED\_CATEGORY](../Topic/IMPLEMENTED_CATEGORY.md)|COM オブジェクトによって実装されているカテゴリが表示されます。|  
|[REQUIRED\_CATEGORY](../Topic/REQUIRED_CATEGORY.md)|COM オブジェクトによってコンテナーの必要なカテゴリが表示されます。|
