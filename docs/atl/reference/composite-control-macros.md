# 複合コントロールに関するマクロ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらのマクロは、イベント シンク マップ エントリとを定義します。  
  
|||  
|-|-|  
|[BEGIN\_SINK\_MAP](../Topic/BEGIN_SINK_MAP.md)|複合コントロールのイベント シンク マップの開始位置を示します。|  
|[END\_SINK\_MAP](../Topic/END_SINK_MAP.md)|複合コントロールのイベント シンク マップの終了位置を示します。|  
|[SINK\_ENTRY](../Topic/SINK_ENTRY.md)|イベント シンク マップのエントリ。|  
|[SINK\_ENTRY\_EX](../Topic/SINK_ENTRY_EX.md)|追加のパラメーターのイベント シンク マップのエントリ。|  
|[SINK\_ENTRY\_INFO](../Topic/SINK_ENTRY_INFO.md)|[IDispEventSimpleImpl](../../atl/reference/idispeventsimpleimpl-class.md)で使用する手動で指定された型の情報を含むイベント シンク マップのエントリ。|
