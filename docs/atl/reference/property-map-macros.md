# プロパティ マップに関するマクロ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらのマクロは、プロパティ マップとエントリを定義します。  
  
|||  
|-|-|  
|[BEGIN\_PROP\_MAP](../Topic/BEGIN_PROP_MAP.md)|ATL のプロパティ マップの開始位置を示します。|  
|[PROP\_DATA\_ENTRY](../Topic/PROP_DATA_ENTRY.md)|ActiveX コントロールのエクステント、つまり大きさを示します。|  
|[PROP\_ENTRY\_TYPE](../Topic/PROP_ENTRY_TYPE.md)|プロパティ マップにプロパティの説明、プロパティの DISPID、およびプロパティ ページの CLSID になります。|  
|[PROP\_ENTRY\_TYPE\_EX](../Topic/PROP_ENTRY_TYPE_EX.md)|プロパティ マップにプロパティの説明、プロパティの DISPID、およびプロパティ ページの CLSID `IDispatch` IID になります。|  
|[PROP\_PAGE](../Topic/PROP_PAGE.md)|プロパティ マップにプロパティ ページの CLSID になります。|  
|[END\_PROP\_MAP](../Topic/END_PROP_MAP.md)|ATL のプロパティ マップの終了位置を示します。|
