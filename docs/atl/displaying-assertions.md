# <a name="displaying-assertions"></a>アサーションの表示
サービスに接続しているクライアントが応答を停止する場合は、サービスがアサートされるが表示されない、メッセージ ボックスが表示されますが可能性があります。 これは、コードをデバッグする Visual C のデバッガーを使用して確認できます (を参照してください[タスク マネージャーを使用して](../atl/using-task-manager.md)このセクションで前述した)。  
  
 設定することがあります、サービスで表示できない、メッセージ ボックスが表示されていると判断した場合、**デスクトップと対話をサービスに許可する**もう一度サービスを使用する前にオプションです。 このオプションは、デスクトップに表示するサービスによって表示されるすべてのメッセージ ボックスを許可する起動時のパラメーターです。 このオプションを設定するコントロール パネルの サービス アプリケーションを開くと、サービスを選択してをクリックして**スタートアップ**、クリックして、**デスクトップと対話をサービスに許可する**オプション。
