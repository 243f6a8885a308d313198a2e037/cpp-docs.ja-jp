# ATL DHTML コントロールの作成
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL コントロール ウィザードは、DHTML コントロールを作成するプロセスを自動化します。  これは、サンプル コードを含む HTML ファイルを含める必要なリソース ファイルを生成します。  
  
#### ATL DHTML コントロールを作成するには  
  
1.  [ATL プロジェクトの作成](../atl/reference/creating-an-atl-project.md)の手順に従います。  
  
2.  クラス ビューでプロジェクト ノードを右クリックし、**追加**をポイントし、ショートカット メニューの **\[クラスの追加\]** をクリックします。  **\[クラスの追加\]** のダイアログ ボックスで、[&#91;ATL コントロール ウィザード&#93;](../atl/reference/atl-control-wizard.md)をダブルクリックします。  ATL コントロール ウィザードでは、**オプション** のタブをクリックし、**\[DHTML コントロール\]**を選択します。  
  
 [既定のコントロールをテストします。](../atl/testing-the-atl-dhtml-control.md)ようになりました。
