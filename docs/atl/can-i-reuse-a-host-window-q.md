# <a name="can-i-reuse-a-host-window"></a>ホスト ウィンドウを再利用できますか。
ホストの windows を再利用することは推奨されません。 コードの堅牢性を確実には、1 つのコントロールの有効期間に、ホスト ウィンドウの有効期間を関連付ける必要があります。
