# ATL プロジェクトの修正
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

この時点で、複合コントロール プロジェクトは、複合コントロールに対して必要なオブジェクトを実装します。  次の手順では、複合コントロールが必要なイベントを処理し、コントロールを追加します。  
  
 追加の ActiveX または Windows コントロールを追加するには、新しいリソース スクリプトを追加し、ダイアログ エディターを使用します。  コントロール \(および関連のタスク\) を追加する方法については、[&#91;ダイアログ エディター&#93;](../mfc/dialog-editor.md)を参照してください。  
  
 ActiveX コントロールからの必要なイベントを処理するには、[複合コントロールへの機能の追加](../atl/adding-functionality-to-the-composite-control.md)を参照してください。
