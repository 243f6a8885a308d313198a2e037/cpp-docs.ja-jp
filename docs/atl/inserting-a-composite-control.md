# 複合コントロールの挿入
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Add Class** のダイアログ ボックスは、プロジェクトに ATL オブジェクトを挿入できるようにします。  **追加**にソリューション エクスプローラーでプロジェクト名を右クリックし、をポイントし、を **Add Class**をクリックして、このダイアログ ボックスにアクセスします。  
  
 **Add Class** のダイアログ ボックスで、**\[ATL コントロール\]**を選択します。  これは [&#91;ATL コントロール ウィザード&#93;](../atl/reference/atl-control-wizard.md)を開始します。  複合コントロールを作成するには、**\[オプション\]** のタブをクリックし、**\[複合コントロール\]** のチェック ボックスをオンにします。  
  
 既定の HTML ページには、コントロールを表示するために作成されます。
