# 埋め込み式はここでは使用できません
XML リテラル内のサポートされていない場所で埋め込み式が使用されています。 埋め込み式を使用して XML 名前空間値を指定することはできないことに注意してください。  
  
 **エラー ID:** BC31172  
  
### このエラーを解決するには  
  
-   サポートされていない場所から埋め込み式を削除します。
