# XML リテラルおよび XML 軸のプロパティは、このアセンブリでは使用されていないため、このデバッグ セッション中は使用できません
[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] の XML 機能を使用できないデバッグ セッション中に、XML リテラルまたは XML 軸のプロパティが **\[ウォッチ\]** または **\[イミディ エイト\]** ウィンドウで参照されています。 これは、[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] の XML 機能を使用しないアセンブリ、またはリリース ビルドのアセンブリの場合です。  
  
 **エラー ID:** BC31196  
  
### このエラーを解決するには  
  
-   アセンブリのデバッグ ビルドを使用して、新しいデバッグ セッションを開始します。
