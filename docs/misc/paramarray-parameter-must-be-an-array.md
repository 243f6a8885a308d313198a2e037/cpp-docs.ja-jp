# ParamArray パラメーターは配列でなければなりません
`ParamArray` 引数は、1 次元配列型として宣言する必要があります。 宣言で `ParamArray` 引数名の後にかっこが含まれていません。  
  
 **エラー ID:** BC30050  
  
### このエラーを解決するには  
  
-   `ParamArray` 引数名の後にかっこのペアを追加します。
