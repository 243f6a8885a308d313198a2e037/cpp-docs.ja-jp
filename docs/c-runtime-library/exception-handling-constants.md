# 例外処理定数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

例外が **try\-except** のステートメントの保護されたセクションの実行中に発生した場合、定数、`EXCEPTION_CONTINUE_SEARCH``EXCEPTION_CONTINUE_EXECUTION`、または `EXCEPTION_EXECUTE_HANDLER` が返されます。  戻り値は例外がどのように処理されるかを指定します。  詳細については、" *C\+\+ Language Reference*" [try\-except のステートメント](../cpp/try-except-statement.md) を参照します。
