# TMP_MAX、L_tmpnam
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 構文  
  
```  
#include <stdio.h>  
```  
  
## 解説  
 `TMP_MAX` は `tmpnam` 関数を生成できる一意のファイル名の最大数です。  `L_tmpnam` は `tmpnam`によって生成される一時ファイル名の長さです。
