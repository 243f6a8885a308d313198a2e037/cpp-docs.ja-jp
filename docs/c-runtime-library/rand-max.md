# <a name="randmax"></a>RAND_MAX
## <a name="syntax"></a>構文  
  
```  
  
#include <stdlib.h>  
  
```  
  
## <a name="remarks"></a>コメント  
 定数 `RAND_MAX` は、`rand` 関数によって返される可能性のある最大値です。 `RAND_MAX` には、値 0x7fff が定義されます。
