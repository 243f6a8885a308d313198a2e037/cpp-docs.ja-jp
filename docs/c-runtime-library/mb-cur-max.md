# MB_CUR_MAX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 構文  
  
```  
  
#include <stdlib.h>  
  
```  
  
## 解説  
 コンテキスト: ANSI のマルチバイトやワイド文字の変換関数  
  
 `MB_CUR_MAX` の値は、現在のロケールのマルチバイト文字の最大バイト数です。
