# <a name="exitsuccess-exitfailure"></a>EXIT_SUCCESS、EXIT_FAILURE
## <a name="syntax"></a>構文  
  
```  
  
#include <stdlib.h>  
```  
  
## <a name="remarks"></a>コメント  
 これらは **exit** 関数と `_exit` 関数の引数であり、`atexit` 関数と `_onexit` 関数の戻り値です。
