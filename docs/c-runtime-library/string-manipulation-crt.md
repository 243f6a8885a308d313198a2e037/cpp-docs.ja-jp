# 文字列操作 (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

これらのルーチンは、null で終わる 1 バイト文字、ワイド文字、およびマルチバイト文字の文字列に対して動作します。  null 文字で終わらない文字配列を使用するには、「[Buffer Manipulation \(バッファー操作\)](../Topic/Buffer%20Manipulation.md)」で説明するバッファー操作ルーチンを使用します。
