# <a name="cacos-cacosf-cacosl"></a>cacos、cacosf、cacosl
ブランチ カット実際の軸に沿った間隔 [-1, +1] 以外に、複素数のアーク コサインを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
_Dcomplex cacos(   
  _Dcomplex z   
);  
_Fcomplex cacos(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex cacos(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex cacosf(   
   _Fcomplex z   
);  
_Lcomplex cacosl(   
   _Lcomplex z   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `z`  
 角度をラジアンで表す複素数。  
  
## <a name="return-value"></a>戻り値  
 ラジアンで表した、`z` のアークコサイン。 結果は、虚軸方向には無制限で、実軸方向には [0, π] の区間内になります。 `z` が区間 [-1, +1] の外側にある場合は、ドメイン エラーが発生します。  
  
## <a name="remarks"></a>コメント  
 C++ ではオーバーロードが可能であるため、`cacos` および `_Fcomplex` の値を受け取って返す `_Lcomplex` のオーバーロードを呼び出すことができます。 C プログラムでは、 `cacos` は常に `_Dcomplex` 値を受け取って返します。
