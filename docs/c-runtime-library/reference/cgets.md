# <a name="cgets"></a>cgets
この POSIX 関数は使用されなくなりました。 代わりに ISO C++ 準拠の [_cgets](../../c-runtime-library/cgets-cgetws.md) またはセキュリティが強化された [_cgets_s](../../c-runtime-library/reference/cgets-s-cgetws-s.md) を使用してください。  
  
> [!IMPORTANT]
>  この API は、Windows ランタイムで実行するアプリケーションでは使用できません。 詳細については、「[HYPERLINK "http://msdn.microsoft.com/en-us/library/windows/apps/jj606124.aspx" \o "/ZW でサポートされない CRT 関数" /ZW でサポートされない CRT 関数](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx)」を参照してください。
