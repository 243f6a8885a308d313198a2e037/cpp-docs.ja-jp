# ctan、ctanf、ctanl
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

複素数のタンジェントの値を取得します。  
  
## 構文  
  
```  
_Dcomplex ctan(   
   _Dcomplex z   
);  
_Fcomplex ctan(   
   _Fcomplex z   
);  // C++ only  
_Lcomplex ctan(   
   _Lcomplex z   
);  // C++ only  
_Fcomplex ctanf(   
   _Fcomplex z   
);  
_Lcomplex ctanl(   
   _Lcomplex z   
);  
```  
  
#### パラメーター  
 `z`  
 ラジアンの角度を表す複素数。  
  
## 戻り値  
 `z` のタンジェント。  
  
|入力|SEH 例外|`_matherr` 例外|  
|--------|------------|-------------------|  
|± ∞、QNAN、IND|none|\_DOMAIN|  
|± ∞ \(`tan`, `tanf`\)|INVALID|\_DOMAIN|  
  
## 解説  
 C\+\+ ではオーバーロードが可能であるため、`ctan` および `_Fcomplex` の値を受け取って返す `_Lcomplex` のオーバーロードを呼び出すことができます。 C プログラムでは `ctan` は、 `_Dcomplex` 値。  
  
## 必要条件  
  
|ルーチン|C ヘッダー|C\+\+ ヘッダー|  
|----------|------------|----------------|  
|`ctan`、`ctanf`、`ctanl`|\<complex.h\>|\< ccomplex \>|  
  
 互換性の詳細については、「C ランタイム ライブラリ」の「[互換性](../../c-runtime-library/compatibility.md)」を参照してください。
