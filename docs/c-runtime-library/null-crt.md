# NULL (CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**NULL** は 多くのポインターの操作と関数で使用されるポインタ値です。  これは 0 と同じです。  **NULL** は 次のヘッダー ファイルに定義されています: CRTDBG.H、LOCALE.H、STDDEF.H、STDIO.H、STDLIB.H、STRING.H、TCHAR.H、TIME.H と WCHAR.H。
