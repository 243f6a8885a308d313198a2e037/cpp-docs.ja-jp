# 国際化
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft ランタイム ライブラリには、さまざまなバージョンの国際市場向けプログラムを作成する際に役立つルーチンが多数用意されています。  これには[ロケール関連のルーチン](../c-runtime-library/locale.md)、ワイド文字ルーチン、マルチバイト文字ルーチン、および汎用テキスト ルーチンが含まれます。  便宜上、ロケール関連のルーチンのほとんどが、実行する操作に従ってこのリファレンスに分類されています。  マルチバイト文字ルーチンとワイド文字ルーチンについては、対応する 1 バイト文字ルーチンと共に、このセクションとアルファベット順のリファレンスで説明します \(存在する場合\)。  
  
 また、ISO646 演算子の代替も含まれます。  
  
 NET Framework で相当する機能については、「[System::Globalization::CultureInfo Class](https://msdn.microsoft.com/en-us/library/system.globalization.cultureinfo.aspx)」を参照してください。
