# 汎用テキスト マップ
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

国際市場向けのコードの作成を容易にするためには、汎用テキスト マッピングは TCHAR.H での定義:  
  
-   [データ型](../c-runtime-library/data-type-mappings.md)  
  
-   [グローバル変数および定数](../c-runtime-library/constant-and-global-variable-mappings.md)  
  
-   [定期的なマッピング](../c-runtime-library/routine-mappings.md)  
  
 詳細については、「[汎用テキスト マッピングを使用します。](../c-runtime-library/using-generic-text-mappings.md)」を参照してください。  汎用テキスト マッピングは ANSI 互換の Microsoft 拡張機能です。
