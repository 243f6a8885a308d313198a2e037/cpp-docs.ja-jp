# FOPEN_MAX、_SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 構文  
  
```  
  
#include <stdio.h>  
  
```  
  
## 解説  
 これは同時に開くことができるファイルの最大数です。  `FOPEN_MAX` は ANSI 互換の名前です。  `_SYS_OPEN` は 既存のコードとの互換性が与えられます。
