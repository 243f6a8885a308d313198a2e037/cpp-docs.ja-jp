# グローバル変数および基本データ型
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Microsoft ランタイム ライブラリは、[グローバル変数](../c-runtime-library/global-variables.md)ライブラリ ルーチンが使用する [コントロール フラグ](../c-runtime-library/control-flags.md)と [基本データ型](../c-runtime-library/standard-types.md) の定義が含まれています。  プログラムで宣言するか、適切なヘッダー ファイルをインクルードして、これらの変数、フラグと型にアクセスします。
