# <a name="environmental-constants"></a>環境定数
## <a name="syntax"></a>構文  
  
```  
#include <stdlib.h>  
```  
  
## <a name="remarks"></a>コメント  
 この定数は、環境文字列の長さを定義します。  
  
|定数|説明|  
|--------------|-------------|  
|`_MAX_ENV`|環境文字列の最大サイズ。|
