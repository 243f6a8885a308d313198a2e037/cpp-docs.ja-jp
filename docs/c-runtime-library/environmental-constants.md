# 環境定数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 構文  
  
```  
#include <stdlib.h>  
```  
  
## 解説  
 この定数は、文字列の環境の長さを定義します。  
  
|定数|説明|  
|--------|--------|  
|`_MAX_ENV`|環境文字列の最大文字列のサイズ。|
