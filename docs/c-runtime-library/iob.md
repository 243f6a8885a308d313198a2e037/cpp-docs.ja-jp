# <a name="iob"></a>_iob

stdio 制御構造の配列。

## <a name="syntax"></a>構文

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>コメント

`IOB_ENTRIES` は stdio.h の 20 として定義されます。
