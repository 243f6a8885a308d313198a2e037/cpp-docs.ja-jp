# _iob
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

stdio の制御構造の配列。  
  
## 構文  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## 解説  
 `IOB_ENTRIES` は stdio.h の 20 として定義されます。
