# <a name="criticalsectioncs-data-member"></a>CriticalSection::cs_ データ メンバー
クリティカル セクションのデータ メンバーを宣言します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
CRITICAL_SECTION cs_;  
```  
  
## <a name="remarks"></a>Remarks  
 このデータ メンバーは保護されます。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
