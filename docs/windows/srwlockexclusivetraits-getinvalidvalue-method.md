# <a name="srwlockexclusivetraitsgetinvalidvalue-method"></a>SRWLockExclusiveTraits::GetInvalidValue メソッド
常に有効ではない SRWLockExclusiveTraits オブジェクトを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
inline static Type GetInvalidValue();  
```  
  
## <a name="return-value"></a>戻り値  
 SRWLockExclusiveTraits、空のオブジェクト。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers::HandleTraits
