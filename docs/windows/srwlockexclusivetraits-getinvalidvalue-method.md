# SRWLockExclusiveTraits::GetInvalidValue メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

常に無効である SRWLockExclusiveTraits オブジェクトを取得します。  
  
## 構文  
  
```  
inline static Type GetInvalidValue();  
```  
  
## 戻り値  
 SRWLockExclusiveTraits の空のオブジェクト。  
  
## 必要条件  
 **ヘッダー:** corewrappers.h  
  
 **名前空間:** Microsoft::WRL::Wrappers::HandleTraits
