# <a name="copying-a-device-image-image-editor-for-icons"></a>デバイス イメージのコピー (アイコン用イメージ エディター)

### <a name="to-copy-a-device-image"></a>デバイス イメージをコピーするには

1. **イメージ** メニューのをクリックして**デバイス イメージを開いて**現在のイメージ リストからイメージを選択します。 たとえば、32 × 32 の 16 色のバージョンのアイコンを選択します。

2. 現在表示されているアイコンのイメージのコピー (**Ctrl**+**C**)。

3. 別のアイコンの別のイメージを開く**イメージ エディター**ウィンドウ。 たとえば、16 × 16 の 16 色のバージョンのアイコンを開きます。

4. アイコン イメージを貼り付けます (**Ctrl**+**V**) から 1 つ**イメージ エディター**他のウィンドウ。 小さいサイズに大きなサイズを貼り付けた場合は、イメージのサイズを変更するアイコンのハンドルを使用できます。
