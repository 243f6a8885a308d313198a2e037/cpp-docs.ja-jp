# Setting the Size of the Combo Box and Its Drop-Down List
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

コンボ ボックスのサイズは、ダイアログ ボックスに追加するときに変更できます。  また、ドロップダウン リスト ボックスのサイズも指定できます。  
  
### コンボ ボックスのサイズを変更するには  
  
1.  ダイアログ ボックス内のコンボ ボックス コントロールを選択します。  
  
     最初は、左右のサイズ変更ハンドルだけがアクティブになっています。  
  
2.  サイズ変更ハンドルを使用して、コンボ ボックスの幅を設定します。  
  
 また、コンボ ボックスのドロップダウン部分の縦方向のサイズも設定できます。  
  
#### コンボ ボックスのドロップダウン リストのサイズを設定するには  
  
1.  コンボ ボックスの右側にあるドロップダウン リスト ボックスの矢印ボタンをクリックします。  
  
     ![MFC プロジェクトのコンボ ボックスの矢印](../Image/vcComboBoxArrow.gif "vcComboBoxArrow")  
  
     ドロップダウン リストの領域が拡張され、コントロールのアウトラインがコンボ ボックスのサイズを示すように変化します。  
  
2.  下側のサイズ変更ハンドルを使用して、ドロップダウン リストの領域の初期サイズを変更します。  
  
     ![MFC プロジェクトのコンボ ボックスのサイズ変更](../mfc/media/vccomboboxsizing.png "vcComboBoxSizing")  
  
3.  ドロップダウン リスト ボックスの矢印ボタンを再度クリックし、コンボ ボックスのドロップダウン リスト部分を閉じます。  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。マネージ プロジェクトにリソース ファイルを手動で追加する方法、リソースへのアクセス方法、静的なリソースの表示方法、およびリソース文字列をプロパティに割り当てる方法については、「[チュートリアル : Windows フォームのローカリゼーション](http://msdn.microsoft.com/ja-jp/9a96220d-a19b-4de0-9f48-01e5d82679e5)」および「[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)」を参照してください。
