# <a name="modulereleasenotifierrelease"></a>Module::ReleaseNotifier::Release
パラメーターを持つオブジェクトを構築した場合は、現在の module::releasenotifier オブジェクトを削除`true`です。  
  
## <a name="syntax"></a>構文  
  
```cpp  
void Release() throw();  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** module.h  
  
 **名前空間:** Microsoft::WRL
