# HANDLENullTraits::GetInvalidValue メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

無効なハンドルを表します。  
  
## 構文  
  
```  
inline static Type GetInvalidValue();  
```  
  
## 戻り値  
 常に `nullptr` を返します。  
  
## 必要条件  
 **ヘッダー:** corewrappers.h  
  
 **名前空間:** Microsoft::WRL::Wrappers::HandleTraits
