# Creating, Moving, and Editing Toolbar Buttons
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ツール バー ボタンは、簡単に作成、移動、コピー、および編集できます。  
  
 新しいボタンまたは空白のボタンは、既定ではツール バーの右端に表示されます。  このボタンを編集前に移動できます。  ボタンを新規作成するときは、編集済みボタンの右側に空白のボタンがもう 1 つ表示されます。  ツール バーを保存しても、空白のボタンは保存されません。  
  
-   [ツール バー ボタンの新規作成](../mfc/creating-a-new-toolbar-button.md)  
  
-   [ツール バー ボタンの移動](../mfc/moving-a-toolbar-button.md)  
  
-   [ツール バー リソースからのボタンのコピー](../mfc/copying-buttons-from-a-toolbar.md)  
  
-   [ツール バー ボタンの削除](../mfc/deleting-a-toolbar-button.md)  
  
-   [ツール バー リソースのボタン間の空白の挿入](../mfc/inserting-a-space-between-buttons-on-a-toolbar.md)  
  
-   [ツール バーのボタン間の空白の削除](../mfc/removing-space-between-buttons-on-a-toolbar.md)  
  
-   [ツール バーのプロパティ ページの編集](../mfc/changing-the-properties-of-a-toolbar-button.md)  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。マネージ プロジェクトにリソース ファイルを手動で追加する方法、リソースへのアクセス方法、静的なリソースの表示方法、およびリソース文字列をプロパティに割り当てる方法については、「[チュートリアル : Windows フォームのローカリゼーション](http://msdn.microsoft.com/ja-jp/9a96220d-a19b-4de0-9f48-01e5d82679e5)」および「[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)」を参照してください。
