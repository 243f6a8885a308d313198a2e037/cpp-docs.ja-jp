# <a name="propget"></a>propget
プロパティ アクセサー関数を指定します。  
  
## <a name="syntax"></a>構文  
  
```  
  
[propget]  
  
```  
  
## <a name="remarks"></a>コメント  
 **Propget** C++ 属性と同じ機能を持つ、 [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 属性。  
  
## <a name="example"></a>例  
 例を参照して[バインド可能な](../windows/bindable.md)の使用例の**propget**です。
