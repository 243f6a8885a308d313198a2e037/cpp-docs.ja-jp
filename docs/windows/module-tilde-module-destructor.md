# <a name="modulemodule-destructor"></a>Module::~Module デストラクター

## <a name="syntax"></a>構文

```cpp
virtual ~Module();
```

## <a name="requirements"></a>要件

**ヘッダー:** module.h

**名前空間:** Microsoft::WRL
