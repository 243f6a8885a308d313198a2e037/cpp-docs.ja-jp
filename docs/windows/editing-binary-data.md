# <a name="editing-binary-data"></a>バイナリ データの編集
### <a name="to-edit-a-resource-in-the-binary-editor"></a>バイナリ エディターでのリソースを編集するには  
  
1.  編集するバイトを選択します。  
  
     TAB キーでは、16 進数および ASCII のセクションでは、バイナリ エディターの間で、フォーカスを移動します。 PAGEUP、PAGEDOWN キーを使用して、一度にリソースの 1 つの画面を移動することができます。  
  
2.  新しい値を入力します。  
  
     値が 16 進数と ASCII セクションの両方ですぐに変更し、行の次の値にフォーカスを移動します。  
  
    > [!NOTE]
    >  バイナリ エディターは、エディターを閉じるときに自動的に変更を受け入れます。
