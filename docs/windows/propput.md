# <a name="propput"></a>propput
プロパティ設定関数を指定します。  
  
## <a name="syntax"></a>構文  
  
```  
  
[propput]  
  
```  
  
## <a name="remarks"></a>コメント  
 **Propput** C++ 属性と同じ機能を持つ、 [propput](http://msdn.microsoft.com/library/windows/desktop/aa367146) MIDL 属性。  
  
## <a name="example"></a>例  
 例を参照して[バインド可能な](../windows/bindable.md)の使用例の**propput**です。
