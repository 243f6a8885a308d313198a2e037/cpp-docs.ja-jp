# <a name="hstringrelease-method"></a>HString::Release メソッド
基になる文字列値を削除し、現在を初期化します。 **HString**オブジェクトから空の値。  
  
## <a name="syntax"></a>構文  
  
```  
void Release() throw()  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
