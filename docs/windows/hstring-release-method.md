# HString::Release メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

基になる文字列値を削除し、現在の HString オブジェクトを空の値に初期化します。  
  
## 構文  
  
```  
void Release() throw()  
```  
  
## 必要条件  
 **ヘッダー:** corewrappers.h  
  
 **名前空間:** Microsoft::WRL::Wrappers
