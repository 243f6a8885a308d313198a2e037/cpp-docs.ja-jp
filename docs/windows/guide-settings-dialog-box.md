# <a name="guide-settings-dialog-box"></a>[ガイドの設定] ダイアログ ボックス
## <a name="layout-guides"></a>レイアウト ガイド  
 レイアウト ガイドの設定を表示します。  
  
 **None**  
  
 レイアウト ツールを非表示にします。  
  
 **ルーラーやガイド**  
  
 有効な場合、追加ルーラー レイアウト ツールです。ガイドは、ルーラーに配置することができます。 既定ガイドは、余白は、ドラッグして移動することができます。 ガイドを配置するルーラーをクリックします。 コントロール""ガイドに合わせる場合、コントロールは、上または横に移動します。 それにアタッチしたら、コントロールは、ガイドにも移動します。 コントロールはそれぞれの側では、ガイドにアタッチされ、ガイドを移動、ときに、コントロールがサイズ変更されます。  
  
 **グリッド**  
  
 レイアウト グリッドを作成します。 新しいコントロールがグリッドに合わせて自動的に配置します。  
  
## <a name="grid-spacing"></a>グリッド間隔  
 ダイアログ ボックスの単位 (Dlu) で、グリッドの間隔の設定を表示します。  
  
 **幅: Dlu**  
  
 Dlu のレイアウト グリッドの幅を設定します。 水平 DLU は、4 で割った値 ダイアログ ボックスのフォントの幅の平均です。  
  
 **高さ: Dlu**  
  
 Dlu のレイアウト グリッドの高さを設定します。 垂直 DLU は 8 で割った値 ダイアログ ボックスのフォントの高さの平均値です。
