# コンソール アプリケーションの作成
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Win32 コンソール アプリケーションを作成する最も簡単な方法は、Win32 アプリケーション ウィザードを使用することです。  
  
### Win32 コンソール アプリケーションを作成するには  
  
1.  「[Visual C\+\+ のアプリケーション ウィザードを使用したプロジェクトの作成](../ide/creating-desktop-projects-by-using-application-wizards.md)」の手順に従います。  
  
2.  **\[新しいプロジェクト\]** ダイアログ ボックスのテンプレート ペインの **\[Win32 コンソール プロジェクト\]** を選択してウィザードを開きます。  
  
3.  [Win32 アプリケーション ウィザード](../windows/win32-application-wizard.md)を使用して[アプリケーションの設定](../Topic/Application%20Settings,%20Win%2032%20Project%20Wizard.md)を定義します。  
  
    > [!NOTE]
    >  ウィザードの既定の設定を使用する場合は、この手順を省略します。  
  
4.  **\[完了\]** をクリックしてウィザードを閉じると、新しく作成したプロジェクトが**ソリューション エクスプローラー**で開きます。
