# <a name="nil-structure"></a>Nil 構造体
WRL インフラストラクチャをサポートし、コードから直接使用するものではありません。  
  
## <a name="syntax"></a>構文  
  
```  
struct Nil;  
```  
  
## <a name="remarks"></a>コメント  
 指定されていない、省略可能なテンプレート パラメーターを示すために使用します。  
  
 Nil では、空の構造です。
