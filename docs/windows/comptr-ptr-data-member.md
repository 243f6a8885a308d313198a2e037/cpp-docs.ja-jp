# <a name="comptrptr-data-member"></a>ComPtr::ptr_ データ メンバー
使用すると、関連付けられているされ、これによって管理されるインターフェイスへのポインターを含む**ComPtr**します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
InterfaceType *ptr_;  
```  
  
## <a name="remarks"></a>Remarks  
 **ptr _** は内部の保護されたデータ メンバーです。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
