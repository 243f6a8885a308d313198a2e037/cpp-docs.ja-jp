# ComPtr::ptr_ データ メンバー
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

関連付けられ、この ComPtr オブジェクトによって管理されるインターフェイスへのポインターが格納されます。  
  
## 構文  
  
```  
InterfaceType *ptr_;  
```  
  
## 解説  
 `ptr_` は 内側のプロテクト データ メンバーです。  
  
## 必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
