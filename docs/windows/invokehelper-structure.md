# <a name="invokehelper-structure"></a>InvokeHelper 構造体

### <a name="parameters"></a>パラメーター  
 *TDelegateInterface*  
 *TCallback*  
 イベント ハンドラー関数の型。  
  
 *について*  
 引数の数、 **InvokeHelper**特殊化します。  
  
## <a name="remarks"></a>Remarks  
 実装を提供、`Invoke()`方法、指定した数値と引数の型に基づいています。
