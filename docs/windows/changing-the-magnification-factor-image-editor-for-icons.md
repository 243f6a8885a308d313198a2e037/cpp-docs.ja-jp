# <a name="changing-the-magnification-factor-image-editor-for-icons"></a>拡大率の変更 (アイコン用イメージ エディター)
既定では、イメージ エディターは、実際のサイズと実際のサイズを 6 回に右側のウィンドウで、ビューの左ペインでビューを表示します。 拡大率 (ワークスペースの下部にあるステータス バーに表示) は、イメージの実際のサイズと表示されているサイズの比率です。 既定の係数が 6 と範囲は、1 から 10。  
  
### <a name="to-change-the-magnification-factor"></a>ズームの倍率を変更するには  
  
1.  選択、**イメージ エディター**ウィンドウ拡大率を変更します。  
  
2.  [イメージ エディター ツールバー](../windows/toolbar-image-editor-for-icons.md)の右側の矢印をクリックして、[拡大ツール](../windows/toolbar-image-editor-for-icons.md)サブメニューから拡大率を選択し、: **1 X**、 **2 X**、 **6 X**、または**8 X**です。  
  
    > [!NOTE]
    >  記載されている以外の拡大率を選択する、 **Magnify**ツール、アクセラレータ キーを使用します。
