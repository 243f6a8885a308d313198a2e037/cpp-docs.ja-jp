# <a name="centering-controls-in-a-dialog-box"></a>ダイアログ ボックスのコントロールの中央揃え

### <a name="to-center-controls-in-a-dialog-box"></a>ダイアログ ボックスのコントロールの中央に

1. 配置を変更するコントロールを選択します。

2. **形式**] メニューの [選択**ダイアログの中央に**、以下の配置のいずれかを選択します。

   - `Vertical`: コントロールをダイアログ ボックスの縦方向に中央揃え。

   - `Horizontal`: コントロールをダイアログ ボックスで水平方向に中央揃え。
