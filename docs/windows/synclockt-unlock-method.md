# <a name="synclocktunlock-method"></a>SyncLockT::Unlock メソッド
WRL インフラストラクチャをサポートし、コードから直接使用するものではありません。  
  
## <a name="syntax"></a>構文  
  
```cpp  
void Unlock();  
```  
  
## <a name="remarks"></a>Remarks  
 現在保持されているリソースの制御を解放**SyncLockT**オブジェクトが存在する場合。
