# Modifying the Layout Grid
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ダイアログ ボックスにコントロールを配置したり、並べ替えたりするときに、レイアウト グリッドを使用すると正確に配置できます。  グリッドがオンになっている場合は、コントロールがグリッドの点線に磁石のように "貼り付く" ように見えます。  この "グリッドに合わせる" 機能のオンとオフを切り替えたり、レイアウト グリッド セルのサイズを変更したりできます。  
  
### レイアウト グリッドのオンとオフを切り替えるには  
  
1.  \[書式\] メニューの \[ガイドの設定\] をクリックします。  
  
2.  [&#91;ガイドの設定&#93; ダイアログ ボックス](../Topic/Guide%20Settings%20Dialog%20Box.md)の \[グリッド\] ボタンを使用してオンとオフを切り替えます。  
  
     [&#91;ダイアログ エディター&#93; ツール バー](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)の \[グリッドの切り替え\] ボタンを使用すると、各ダイアログ エディター ウィンドウのグリッドを制御できます。  
  
### レイアウト グリッドのサイズを変更するには  
  
1.  \[書式\] メニューの \[ガイドの設定\] をクリックします。  
  
2.  [&#91;ガイドの設定&#93; ダイアログ ボックス](../Topic/Guide%20Settings%20Dialog%20Box.md)で、グリッドのセルの高さと幅をダイアログ単位 \(DLU: Dialog Unit\) で入力します。  高さと幅の最小値は、4 DLU です。  DLU の詳細については、「[ダイアログ ボックスのコントロールの配置](../mfc/arrangement-of-controls-on-dialog-boxes.md)」を参照してください。  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。マネージ プロジェクトにリソース ファイルを手動で追加する方法、リソースへのアクセス方法、静的なリソースの表示方法、およびリソース文字列をプロパティに割り当てる方法については、「[チュートリアル : Windows フォームのローカリゼーション](http://msdn.microsoft.com/ja-jp/9a96220d-a19b-4de0-9f48-01e5d82679e5)」および「[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)」を参照してください。  
  
 要件  
  
 Win32
