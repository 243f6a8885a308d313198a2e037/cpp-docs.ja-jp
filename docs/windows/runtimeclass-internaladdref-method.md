# <a name="runtimeclassinternaladdref-method"></a>RuntimeClass::InternalAddRef メソッド
現在 RuntimeClass オブジェクトへの参照カウントをインクリメントします。  
  
## <a name="syntax"></a>構文  
  
```  
ULONG InternalAddRef();  
```  
  
## <a name="return-value"></a>戻り値  
 結果として得られる、参照カウントします。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** implements.h  
  
 **名前空間:** Microsoft::WRL
