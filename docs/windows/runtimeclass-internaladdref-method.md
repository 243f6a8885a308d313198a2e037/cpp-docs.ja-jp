# RuntimeClass::InternalAddRef メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

RuntimeClass で現在のオブジェクトの参照カウントをインクリメントします。  
  
## 構文  
  
```  
ULONG InternalAddRef();  
```  
  
## 戻り値  
 結果の参照カウント。  
  
## 必要条件  
 **ヘッダー:** implements.h  
  
 **名前空間:** Microsoft::WRL
