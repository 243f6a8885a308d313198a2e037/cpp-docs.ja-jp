# Attributes by Group
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+属性は、次の機能別グループに編成されます。  
  
|属性|説明|  
|--------|--------|  
|[COM 属性](../Topic/COM%20Attributes.md)|COM開発および.NET Framework共通言語ランタイムの開発の複数の領域をサポートするコードを挿入します。|  
|[IDL 属性](../windows/idl-attributes.md)|ウィザードの使用と構造体の知識とそのファイルの構文のないソース・コード ファイルの.idlファイルの変更を有効にします。|  
|[OLE DB コンシューマー属性](../windows/ole-db-consumer-attributes.md)|コマンドを実行し、データにアクセス開始テーブルのようなタスクを実行するOLE DBコンシューマーを作成するに [OLE DBコンシューマー テンプレート](../data/oledb/ole-db-consumer-templates-reference.md)でコード ベース—挿入します。|  
|[コンパイラの属性](../windows/compiler-attributes.md)|Visual C\+\+コンパイラによって指定されます。|  
  
 は、.NET Frameworkアプリケーション用のカスタム属性を定義する方法の詳細については、[User\-Defined Attributes](../windows/user-defined-attributes-cpp-component-extensions.md)を参照してください。
