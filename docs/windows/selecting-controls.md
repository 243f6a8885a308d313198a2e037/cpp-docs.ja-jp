# <a name="selecting-controls"></a>コントロールの選択
サイズにコントロールを選択、配置、移動、コピー、または削除したり、および必要な操作を実行します。 ほとんどの場合は、ツールを使用して、サイズと位置合わせの 1 つ以上のコントロールを選択する必要があります、[ダイアログ エディター ツールバー](../windows/showing-or-hiding-the-dialog-editor-toolbar.md)です。  
  
 コントロールが選択されている、網掛けされた枠線が実線 (アクティブ) であるまたは中空 (無効)「サイズ変更ハンドル、」小を 2 乗をするときに選択境界線が表示されます。 複数のコントロールを選択すると、高いコントロールには実線のサイズ変更ハンドルです。その他のすべての選択したコントロールには、中空のサイズ変更ハンドルがあります。  
  
 サイズを設定または複数のコントロールの配置は、ダイアログ エディター コントロールを使用して、"重要な"とその他のコントロールのサイズや配置の方法を決定します。 既定では、主要なコントロールは、選択されている最初のコントロールはします。
