# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

マネージ リソース ファイル \(.resx\) は XML ファイルです。  マネージ リソース ファイルを **\[新しい項目の追加\]** ダイアログ ボックスからプロジェクトに追加すると、**マネージ リソース エディター**が既定で開きます。  
  
 詳細については、「[.Resx ファイル形式のリソース](http://msdn.microsoft.com/ja-jp/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5)」を参照してください。
