# <a name="retval"></a>retval

メンバーの戻り値を受け取るパラメーターを指定します。

## <a name="syntax"></a>構文

```cpp
[retval]
```

## <a name="remarks"></a>Remarks

**Retval** C++ 属性と同じ機能を持つ、 [retval](/windows/desktop/Midl/retval) MIDL 属性。

**retval**関数の宣言の最後の引数に表示する必要があります。
