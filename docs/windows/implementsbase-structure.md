# ImplementsBase 構造体
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL のインフラストラクチャをサポートします。コードから直接使用するためのものではありません。  
  
## 構文  
  
```  
struct ImplementsBase;  
```  
  
## 解説  
 テンプレート パラメーターの検証に使用される [Implements 構造体](../Topic/Implements%20Structure.md)型を使用します。  
  
 ImplementsBase 構造体は空の仕様です。  
  
## 継承階層  
 `ImplementsBase`  
  
## 必要条件  
 **ヘッダー:** implements.h  
  
 **名前空間:** Microsoft::WRL::Details
