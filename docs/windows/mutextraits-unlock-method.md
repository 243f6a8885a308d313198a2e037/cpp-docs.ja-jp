# <a name="mutextraitsunlock-method"></a>MutexTraits::Unlock メソッド
共有リソースの排他的に制御を解放します。  
  
## <a name="syntax"></a>構文  
  
```  
inline static void Unlock(  
   _In_ Type h  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `h`  
 ミュー テックス オブジェクトへのハンドルします。
