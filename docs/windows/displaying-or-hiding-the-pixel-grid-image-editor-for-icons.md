# <a name="displaying-or-hiding-the-pixel-grid-image-editor-for-icons"></a>ピクセル グリッドの表示と非表示 (アイコン用イメージ エディター)
イメージ エディター ウィンドウの 4 つ以上の拡大率をすべて、画像内の個々 のピクセルを区切るグリッドを表示できます。  
  
> [!TIP]
>  ツール ヒントは、ツールバーのボタンの上にカーソルを置くと表示されます。 これらのヒントを使用する各ボタンの機能を特定できます。  
  
### <a name="to-display-or-hide-the-pixel-grid"></a>ピクセル グリッドの表示と非表示  
  
1.  **イメージ** メニューのをクリックして**グリッド設定**です。  
  
2.  選択、**ピクセル グリッド**グリッドを表示する、またはグリッドを非表示にする ボックスをオフにする チェック ボックスです。
