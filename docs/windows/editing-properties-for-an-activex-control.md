# Editing Properties for an ActiveX Control
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

独立系販売元提供の ActiveX コントロールは、独自のプロパティと特性を持つ場合があります。  ActiveX コントロールのプロパティは、\[プロパティ\] ウィンドウに表示されます。  また、ActiveX コントロールの作成者が作成したプロパティ ページは、\[プロパティ ページ\] ダイアログ ボックスに表示されます。特定の ActiveX コントロールのプロパティ ページを表示するには、[&#91;プロパティ&#93; ウィンドウ](../Topic/Properties%20Window.md)の \[プロパティ ページ\] ボタンをクリックします。  
  
 ActiveX コントロールのプロパティ ページには、そのコントロールに付属するプロパティ シートに応じてさまざまなタブが表示されます。  
  
> [!NOTE]
>  以下の手順は、プロパティ ページを使用して ActiveX コントロールを編集する場合に適用します。  ActiveX のプロパティを新しい \[プロパティ\] ウィンドウに表示して編集する方法もあります。  
  
### ActiveX コントロールのプロパティを編集するには  
  
1.  ActiveX コントロールを選択します。  
  
2.  \[表示\] メニューの \[プロパティ ページ\] をクリックして、プロパティを表示します。  
  
3.  プロパティ ページで必要な変更を行います。  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。マネージ プロジェクトにリソース ファイルを手動で追加する方法、リソースへのアクセス方法、静的なリソースの表示方法、およびリソース文字列をプロパティに割り当てる方法については、「[チュートリアル : Windows フォームのローカリゼーション](http://msdn.microsoft.com/ja-jp/9a96220d-a19b-4de0-9f48-01e5d82679e5)」および「[Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)」を参照してください。  
  
 要件  
  
 Win32
