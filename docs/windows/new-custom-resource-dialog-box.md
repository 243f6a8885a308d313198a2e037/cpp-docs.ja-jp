# <a name="new-custom-resource-dialog-box-c"></a>新しいカスタム リソース ダイアログ ボックス (C++)

C++ プロジェクトに新しいカスタム リソースを作成できます。

### <a name="resource-type"></a>リソースの種類

カスタム リソースの種類の名前を入力するためのテキスト ボックスを提供します。 Visual C は、名前を自動的に大文字を終了すると、**新規カスタム リソース** ダイアログ ボックス。

## <a name="remarks"></a>Remarks

リソース名または Visual Studio によって予約されている ID を指定しません。 予約済みの名前は DESIGNINFO、HWB、および TEXTINCLUDE、予約済み ID は 255 です。
