# [新規カスタム リソース] ダイアログ ボックス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

カスタム リソースを新規作成できます。  
  
 **\[リソースの種類\]**  
 表示されるテキスト ボックスに、カスタム リソースの種類の名前を入力します。  Visual C\+\+ では、**\[新規カスタム リソース\]** ダイアログ ボックスを閉じる際、名前の最初の文字が自動的に大文字になります。  
  
## 解説  
 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] で予約済みのリソース名または [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] は指定しないでください。  予約済みの名前は DESIGNINFO、HWB、および TEXTINCLUDE で、予約済みの [!INCLUDE[TLA2#tla_id](../windows/includes/tla2sharptla_id_md.md)] は 255 です。
