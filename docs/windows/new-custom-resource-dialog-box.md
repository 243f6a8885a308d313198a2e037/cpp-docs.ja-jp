# <a name="new-custom-resource-dialog-box"></a>[新規カスタム リソース] ダイアログ ボックス
新しいカスタム リソースを作成できます。  
  
### <a name="resource-type"></a>リソースの種類  
 カスタム リソースの種類の名前を入力するためのテキスト ボックスを提供します。 Visual C は、名前を自動的に大文字を終了すると、**新規カスタム リソース** ダイアログ ボックス。  
  
## <a name="remarks"></a>Remarks  
 リソース名または ID によって予約されていることを指定しない[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]します。 予約済みの名前は DESIGNINFO、HWB、および TEXTINCLUDE、予約済み ID は 255 です。
