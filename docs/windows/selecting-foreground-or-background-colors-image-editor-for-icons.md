# Selecting Foreground or Background Colors (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**\[消しゴム\]**以外の **\[イメージ エディター\]** ツール バー上のツールを使用する際、マウスの左ボタンまたは右ボタンを押すと、それぞれ現在の前景色または背景色で描画します。  
  
### 前景色を選択するには  
  
1.  マウスの左ボタンで、[色パレット](../Topic/Colors%20Window%20\(Image%20Editor%20for%20Icons\).md)にある必要な色をクリックします。  
  
### 背景色を選択するには  
  
1.  マウスの右ボタンで、**色パレット**にある必要な色をクリックします。  
  
 マネージ プロジェクトにリソースを追加する方法については、『.NET Framework 開発者ガイド』の「[アプリケーションのリソース](../Topic/Resources%20in%20Desktop%20Apps.md)」を参照してください。  
  
 必要条件  
  
 なし
