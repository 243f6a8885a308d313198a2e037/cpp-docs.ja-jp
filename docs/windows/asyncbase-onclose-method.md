# <a name="asyncbaseonclose-method"></a>AsyncBase::OnClose メソッド
派生クラスでオーバーライドされると、非同期操作を閉じます。  
  
## <a name="syntax"></a>構文  
  
```  
virtual void OnClose(  
   void  
) = 0;  
```  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** async.h  
  
 **名前空間:** Microsoft::WRL
