# <a name="asyncbaseonstart-method"></a>AsyncBase::OnStart メソッド
派生クラスでオーバーライドされると、非同期操作を開始します。  
  
## <a name="syntax"></a>構文  
  
```  
virtual void OnStart(  
   void  
) = 0;  
```
