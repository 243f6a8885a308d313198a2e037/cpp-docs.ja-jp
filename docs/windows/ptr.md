# <a name="ptr"></a>ptr

完全なポインターとしてのポインターを指定します。

## <a name="syntax"></a>構文

```cpp
[ptr]
```

## <a name="remarks"></a>Remarks

**Ptr** C++ 属性と同じ機能を持つ、 [ptr](/windows/desktop/Midl/ptr) MIDL 属性。

## <a name="example"></a>例

例をご覧ください[defaultvalue](../windows/defaultvalue.md)の使用サンプル**ptr**します。
