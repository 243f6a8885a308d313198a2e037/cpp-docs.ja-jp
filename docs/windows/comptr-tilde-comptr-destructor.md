# ComPtr::~ComPtr デストラクター
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Deinitializes ComPtr オブジェクトのインスタンス。  
  
## 構文  
  
```  
WRL_NOTHROW ~ComPtr();  
```  
  
## 必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
