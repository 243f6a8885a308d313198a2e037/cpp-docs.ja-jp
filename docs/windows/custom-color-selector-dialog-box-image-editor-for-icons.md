# <a name="custom-color-selector-dialog-box-image-editor-for-icons"></a>[色の調整] ダイアログ ボックス (アイコン用イメージ エディター)
このダイアログ ボックスでは、イメージを使用する色をカスタマイズすることができます。 詳細については、次を参照してください。[のカスタマイズまたは変更する色](../windows/customizing-or-changing-colors-image-editor-for-icons.md)です。  
  
 **グラデーションの色の表示**  
 選択した色の値を変更します。 色を変更するには、十字カーソルを置きます。 スライダーを上下に変更、明るさや色の RGB 値。  
  
 **光度バー**  
 [グラデーション] ボックスで選択した色の明るさを設定します。 をクリックし、大きい明るさのバーの白い矢印をドラッグまたは以内のダウンします。 色のボックスが選択した色の明るさを設定するの影響などが表示されるに注意してください。  
  
 **色**  
 定義した色の色合い (色相値) を一覧表示します。 値の範囲は 0 ~ 240、場所 0 は赤、60 は黄色、120 は緑、180 はシアン、200、マゼンタで 240 は青です。  
  
 **色合い**  
 定義した色の色合い (色相値) を一覧表示します。 値の範囲は 0 ~ 240、場所 0 は赤、60 は黄色、120 は緑、180 はシアン、200、マゼンタで 240 は青です。  
  
 **土曜日**  
 定義した色の鮮やかさの値を指定します。 鮮やかさは、指定された色合いの色の量です。 値の範囲は 0 ~ 240 です。  
  
 **明るさ**  
 定義した色の光度 (明るさ) の一覧を示します。 値の範囲は 0 ~ 240 です。  
  
 **赤**  
 定義した色の赤の値を指定します。 値の範囲は 0 ~ 255 です。  
  
 **緑**  
 定義した色の緑の値を指定します。 値の範囲は 0 ~ 255 です。  
  
 **青い**  
 定義した色の青の値を指定します。 値の範囲は 0 ~ 255 です。
