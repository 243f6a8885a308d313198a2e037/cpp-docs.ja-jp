# <a name="text-tool-dialog-box-image-editor-for-icons"></a>[テキスト ツール] ダイアログ ボックス (アイコン用イメージ エディター)
使用して、**テキスト ツール**テキスト カーソル、ビットマップ、またはアイコン リソースを追加する ダイアログ ボックス。  
  
 このダイアログ ボックスにアクセスするには、開く、[イメージ エディター](../windows/window-panes-image-editor-for-icons.md)です。 選択**ツール**から、**イメージ**メニューをクリックし、**テキスト ツール**コマンド。  
  
 **[フォント] ボタン**  
 開く、[テキスト ツール フォント ダイアログ ボックス](../windows/text-tool-font-dialog-box-image-editor-for-icons.md)フォント、スタイル、またはカーソルのフォントのサイズを変更できます。 変更は、テキスト領域に表示されるテキストに適用されます。  
  
 **テキスト領域**  
 リソースの一部として表示されるテキストを表示します。 最初にこの領域は空です。  
  
> [!NOTE]
>  場合**透明な背景**が設定されている、テキストのみをイメージに保存されます。 場合**不透明な背景**設定すると、格納、外接する四角形、[背景色](../windows/selecting-foreground-or-background-colors-image-editor-for-icons.md)テキストの背後に配置されます。 詳細については、次を参照してください。[選択不透明な直線および透明な背景](../windows/choosing-a-transparent-or-opaque-background-image-editor-for-icons.md)です。  
  
 右クリック、**テキスト ツール** ダイアログ ボックスは、Windows の標準コマンドの一覧を含む既定のショートカット メニューにアクセスします。
