# Arrangement of Controls on Dialog Boxes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ダイアログ エディターには、コントロールの配置とサイズ変更を自動的に行うためのレイアウト ツールが用意されています。  ほとんどのタスクに対しては、[&#91;ダイアログ エディター&#93; ツール バー](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)を使用できます。  \[ダイアログ エディター\] ツール バーのすべてのコマンドは、\[書式\] メニューでも使用できます。また、ほとんどのコマンドには対応する[ショートカット キー](../mfc/accelerator-keys-for-the-dialog-editor.md)があります。  
  
 ダイアログ ボックス用の多くのレイアウト コマンドは、複数のコントロールを選択している場合にだけ使用できます。  単一または複数のコントロールを選択できる場合に、複数のコントロールを選択すると、最初に選択したコントロールが既定で "最も優先度の高い" コントロールになります。  コントロールの選択および最も優先度の高いコントロールについては、「[コントロールの選択](../mfc/selecting-controls.md)」を参照してください。  
  
 現在のコントロールの位置、高さ、および幅は、ステータス バーの右下隅に表示されます。  ダイアログ ボックス全体を選択すると、全体の位置、高さ、および幅がステータス バーに表示されます。
