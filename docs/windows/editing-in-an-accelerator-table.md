# アクセラレータ テーブルでのエントリの編集
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### <a name="to-edit-in-an-accelerator-table"></a>アクセラレータ テーブルで編集するには  
  
1.  アクセラレータ テーブルを開くことでそのアイコンをダブルクリックして [リソース ビュー](../windows/resource-view-window.md)します。  
  
    > [!NOTE]
    >  プロジェクトに .rc ファイルがまだ含まれていない場合は、「 [リソース スクリプト ファイルの新規作成](../windows/how-to-create-a-resource-script-file.md)」を参照してください。  
  
2.  テーブル内のエントリを選択し、クリックしてインプレース編集をアクティブにします。  
  
3.  変更するには、ドロップダウン コンボ ボックスから選択するか、インプレースで入力します。  
  
    -    [ID](Id%20Property.xml), リストまたは編集するタイプを選択します。  
  
    -    [修飾子](../windows/accelerator-modifier-property.md), 、一覧から選択します。  
  
    -    [キー](../Topic/Accelerator%20Key%20Property.md), リストまたは編集するタイプを選択します。  
  
    -    [型](../windows/accelerator-type-property.md), 、ASCII または VIRTKEY を一覧から選択します。  
  
 マネージ プロジェクトにリソースを追加する方法については、『 [.NET Framework 開発者ガイド](../Topic/Resources%20in%20Desktop%20Apps.md) 』の「 *アプリケーションのリソース*  
  
 **Requirements**  
  
 Win32
