# HandleT::IsValid メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

HandleT の現在のオブジェクトがハンドルを表すかどうかを示します。  
  
## 構文  
  
```  
bool IsValid() const;  
```  
  
## 戻り値  
 HandleT がハンドルを表す`true` ; それ以外の場合は `false`。  
  
## 必要条件  
 **ヘッダー:** corewrappers.h  
  
 **名前空間:** Microsoft::WRL::Wrappers
