# <a name="safeint-library"></a>SafeInt ライブラリ

**SafeInt**ライブラリ アプリケーションは、算術演算を実行するときになる可能性がある整数のオーバーフローを防ぐために役立ちます。 このライブラリの最新バージョンは[ https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt)します。
