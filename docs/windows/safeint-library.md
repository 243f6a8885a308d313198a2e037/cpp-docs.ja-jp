# <a name="safeint-library"></a>SafeInt ライブラリ

**SafeInt**ライブラリ アプリケーションは、算術演算を実行するときになる可能性がある整数のオーバーフローを防ぐために役立ちます。

|[SafeInt 関数](../windows/safeint-functions.md)|関数を作成せずに使用できる、 **SafeInt**オブジェクト。|
