# <a name="comptrreset"></a>ComPtr::Reset

これに関連付けられているインターフェイス ポインターのすべての参照を解放**ComPtr**します。

## <a name="syntax"></a>構文

```cpp
unsigned long Reset();
```

## <a name="return-value"></a>戻り値

解放された参照の数 (存在する場合)。

## <a name="requirements"></a>要件

**ヘッダー:** client.h

**名前空間:** Microsoft::WRL
