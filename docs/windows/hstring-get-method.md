# <a name="hstringget-method"></a>HString::Get メソッド
基になる HSTRING ハンドルの値を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HSTRING Get() const throw()  
```  
  
## <a name="return-value"></a>戻り値  
 基になる HSTRING ハンドルの値  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
