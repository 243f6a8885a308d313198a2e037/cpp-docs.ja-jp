# <a name="moduledecrementobjectcount-method"></a>Module::DecrementObjectCount メソッド
モジュールによって追跡されるオブジェクトの数をデクリメントします。  
  
## <a name="syntax"></a>構文  
  
```  
virtual long DecrementObjectCount();  
```  
  
## <a name="return-value"></a>戻り値  
 デクリメント操作の前にカウントします。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** module.h  
  
 **名前空間:** Microsoft::WRL
