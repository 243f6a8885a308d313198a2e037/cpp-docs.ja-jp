# <a name="moduleterminate-method"></a>Module::Terminate メソッド
すべてのファクトリが、モジュールをシャット ダウンによってインスタンス化が発生します。  
  
## <a name="syntax"></a>構文  
  
```  
void Terminate();  
```  
  
## <a name="remarks"></a>コメント  
 キャッシュのファクトリを解放します。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** module.h  
  
 **名前空間:** Microsoft::WRL
