# <a name="load-palette-colors-dialog-box-c-image-editor-for-icons"></a>パレットの読み込み ダイアログ ボックス (C++) (アイコン用イメージ エディター)

C++ プロジェクトで使用するのに特殊なカラー パレットを読み込むには、このダイアログ ボックスを使用します。 詳細については、次を参照してください。[を保存し、さまざまなカラー パレットの読み込み](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md)します。

### <a name="look-in"></a>[検索対象]

ファイルまたはフォルダーを検索する場所を指定します。 別の場所を選択する矢印をクリックするか、レベルに移動 ツールバーでフォルダー アイコンをクリックします。

### <a name="file-name"></a>ファイル名

開くファイルの名前を入力する場所を提供します。 既に開いているファイルをすばやく検索するには、使用可能な場合、ボックスの一覧で、ファイル名をクリックします。

ファイルを検索する場合は、ワイルドカードとしてアスタリスク (*) を使用できます。 たとえば、「 \*。\*すべてファイル一覧を表示します。 ファイル、たとえば、C:\My Documents\MyColorPalette.pal の完全なパスを入力することもできます。 または\\\NetworkServer\MyFolder\MyColorPalette.pal します。

### <a name="files-of-type"></a>[ファイルの種類]

表示するファイルの種類を示します。 パレット (* *.pal) のカラー パレットの既定のファイルの種類します。
