# <a name="deleting-a-control"></a>コントロールの削除
### <a name="to-delete-a-control"></a>コントロールを削除するには  
  
1.  ダイアログ ボックスで、選択制御とキーを押して、**削除**キー。  
  
     \- または  
  
2.  **編集** メニューのをクリックして**削除**です。  
  
    > [!TIP]
    >  ダイアログ エディターをさまざまな状況で使用している場合は、マウスの右ボタンをクリックすると、頻繁に使用するコマンドのショートカット メニューを表示できます。
