# <a name="hstringhstring-destructor"></a>HString::~HString デストラクター
HString クラスの現在のインスタンスを破棄します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
~HString() throw()  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** corewrappers.h  
  
 **Namespace:** Microsoft::WRL::Wrappers
