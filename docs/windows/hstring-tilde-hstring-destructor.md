[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

HString クラスの現在のインスタンスを破棄します。  
  
## 構文  
  
```cpp  
~HString() throw()  
```  
  
## 必要条件  
 **ヘッダー:** corewrappers.h  
  
 **名前空間:** Microsoft::WRL::Wrappers
