1. **イメージ エディター**ツールバー (または**イメージ**] メニューの [**ツール**コマンド)、をクリックして、**入力**ツール。

2. 必要に応じて、描画色を選択: で、[色パレット](../windows/colors-window-image-editor-for-icons.md)前景の色を選択するマウスの左ボタンまたは背景色を選択するマウスの右ボタンをクリックします。

3. 移動、**塗りつぶし**ツールを格納する領域。

4. 前景色または背景色をそれぞれに入力を左または右マウス ボタンをクリックします。
