# <a name="modulegetmodule-method"></a>Module::GetModule メソッド
モジュールのインスタンスを作成します。  
  
## <a name="syntax"></a>構文  
  
```cpp  
static Module& GetModule();  
WRL_NOTHROW static Module& GetModule();  
```  
  
## <a name="return-value"></a>戻り値  
 モジュールへの参照。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** module.h  
  
 **名前空間:** Microsoft::WRL
