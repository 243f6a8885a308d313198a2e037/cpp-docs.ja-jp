# <a name="attribute-requirements"></a>属性の必要条件

C++ 属性に記載されている要件は、操作するには、プロジェクトの種類、コンパイラの設定、および属性に必要なその他の情報について説明します。 情報のカテゴリを以下に示します。
  
> [!NOTE]
> 属性を使用するクラスから派生したクラスで属性を使用することはサポートされていません。
  
## <a name="header"></a>Header

このフィールドでは、ヘッダー ファイル属性を使用する前に含める必要があるを一覧表示します。
  
## <a name="project"></a>プロジェクト

このフィールドには、属性を使用できるプロジェクトの種類について説明します。
  
## <a name="compiler"></a>コンパイラ

このフィールドは、この属性を使用するのに必要なコンパイラ オプションを提供します。
