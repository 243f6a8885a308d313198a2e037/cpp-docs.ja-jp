# Attribute Categories
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

属性リファレンスではVisual C\+\+ で使用できる属性のさまざまな型について説明します。  
  
## 関連項目  
 [COM 属性](../Topic/COM%20Attributes.md)  
 コードを COM 開発の多くの領域をサポートするために挿入します。  
  
 [IDL 属性](../windows/idl-attributes.md)  
 ウィザードでそのファイルの構造と構文を理解しているなしでソース・コード ファイルの関連付けられた .idl ファイルを作成または変更するコードを挿入します。  
  
 [OLE DB コンシューマー属性](../windows/ole-db-consumer-attributes.md)  
 [OLE DB コンシューマー テンプレート](../data/oledb/ole-db-consumer-templates-reference.md) に基づいてコードを開くテーブルのようなタスクを実行する OLE DB コンシューマーを作成するために挿入しコマンドおよびデータへのアクセスを実行します。  
  
 [コンパイラの属性](../windows/compiler-attributes.md)  
 さまざまな一般的なプログラミング タスクを実行するコードを挿入します。  
  
 [属性付きプログラミング](../windows/attributed-programming-concepts.md)  
 属性付きプログラミングに関する概念を説明するトピックへのリンクを示します。
