# C++ Attributes Reference
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

属性は、COM プログラミングおよび .NET Framework 共通言語ランタイム開発の簡略化を目的としてデザインされています。  ソース ファイルに属性が含まれていると、コンパイラは、プロバイダー DLL を利用して、生成されるオブジェクト ファイルにコードを挿入したり、オブジェクト ファイルのコードを変更したりします。
