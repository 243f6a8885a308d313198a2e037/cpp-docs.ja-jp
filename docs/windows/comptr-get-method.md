# ComPtr::Get メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

この ComPtr オブジェクトに関連付けられたインターフェイスへのポインターを取得します。  
  
## 構文  
  
```  
T* Get() const;  
```  
  
## 戻り値  
 この ComPtr オブジェクトに関連付けられたインターフェイスへのポインター。  
  
## 必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
