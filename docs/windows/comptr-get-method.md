# <a name="comptrget-method"></a>ComPtr::Get メソッド
この ComPtr に関連付けられたインターフェイスへのポインターを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
T* Get() const;  
```  
  
## <a name="return-value"></a>戻り値  
 この ComPtr に関連付けられているインターフェイスへのポインター。  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
