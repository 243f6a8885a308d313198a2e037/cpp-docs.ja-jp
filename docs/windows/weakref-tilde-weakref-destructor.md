# <a name="weakrefweakref-destructor"></a>WeakRef::~WeakRef デストラクター
WeakRef クラスの現在のインスタンスの初期化を解除します。  
  
## <a name="syntax"></a>構文  
  
```  
~WeakRef();  
```  
  
## <a name="requirements"></a>要件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
