### <a name="to-even-the-spacing-between-controls"></a>コントロール間の間隔を均等に  
  
1.  配置を変更するコントロールを選択します。  
  
2.  **形式**] メニューの [選択**均等スペース**、次の間隔の配置のいずれかを選択します。  
  
    -   `Across`: 等間隔の左端と右端のコントロールを選択します。  
  
    -   `Down`: 等間隔、最上位と下位のコントロールを選択します。
