# ID Property
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ID プロパティは、プログラム コード内の各アクセラレータ テーブルのエントリを参照します。  これは、ユーザーがアクセラレータ キーまたはキーの組み合わせを押したときに、プログラムが受け取るコマンド値です。  アクセラレータをメニュー項目と同じにするには、両者の \[ID\] を同じにします。ただし、これはアクセラレータ テーブルの \[ID\] がメニュー リソースの \[ID\] と同一の場合に限られます。  
  
## 要件  
 Win32
