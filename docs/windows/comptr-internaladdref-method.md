# ComPtr::InternalAddRef メソッド
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

この ComPtr オブジェクトに関連付けられたインターフェイスの参照カウントをインクリメントします。  
  
## 構文  
  
```  
void InternalAddRef() const;  
```  
  
## 解説  
 このメソッドが保護されます。  
  
## 必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
