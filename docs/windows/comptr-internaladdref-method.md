# <a name="comptrinternaladdref-method"></a>ComPtr::InternalAddRef メソッド
この ComPtr に関連付けられたインターフェイスの参照カウントをインクリメントします。  
  
## <a name="syntax"></a>構文  
  
```  
void InternalAddRef() const;  
```  
  
## <a name="remarks"></a>コメント  
 このメソッドは保護されています。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL
