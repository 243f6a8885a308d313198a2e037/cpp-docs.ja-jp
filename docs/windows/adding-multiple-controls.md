# <a name="adding-multiple-controls"></a>複数のコントロールの追加

1.  キーを押しながら、 **CTRL**キーでコントロールを選択して、[ツールボックス ウィンドウ](/visualstudio/ide/reference/toolbox)します。  
  
2.  リリース、 **CTRL**キーを何度でも、特定のコントロールを追加すると、ダイアログ ボックスをクリックします。
