# [グリッドの設定] ダイアログ ボックス (アイコン用イメージ エディター)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

イメージのグリッド設定を指定し、編集対象のイメージに重ねてグリッド線を表示できます。  グリッド線はイメージの編集に役立ちますが、イメージの一部としては保存されません。  
  
 **\[ピクセル表示\]**  
 オンにすると、イメージ エディターの各ピクセルを囲むグリッドが表示されます。  このグリッドが表示されるのは、表示倍率が 4 以上の場合だけです。  
  
 **\[タイル表示\]**  
 オンにすると、イメージ エディターにはグリッド間隔値で指定したピクセル数のブロックを囲むグリッドが表示されます。  
  
 **\[幅\]**  
 各タイル ブロックの幅を指定します。  このオプションは、一定間隔で配置される複数のイメージを含むビットマップを描画するときに便利です。  
  
 **\[高さ\]**  
 各タイル ブロックの高さを指定します。  このオプションは、一定間隔で配置される複数のイメージを含むビットマップを描画するときに便利です。
