# &lt;scoped_allocator&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

コンテナーのテンプレート クラス scoped\_allocator を定義します。  
  
## 構文  
  
```cpp  
#include <scoped_allocator>  
```  
  
### 演算子  
  
|||  
|-|-|  
|[operator\!\=](../Topic/operator!=%20Operator%20\(%3Cscoped_allocator%3E\).md)|演算子の左側の scoped\_allocator オブジェクトが右側のリスト オブジェクトと等しくないかどうかを調べます。|  
|[operator\=\=](../Topic/operator==%20Operator%20\(%3Cscoped_allocator%3E\).md)|演算子の左側の scoped\_allocator オブジェクトが右側のリスト オブジェクトと等しいかどうかを調べます。|  
  
### クラス  
  
|||  
|-|-|  
|[scoped\_allocator\_adaptor クラス](../standard-library/scoped-allocator-adaptor-class.md)|一つ以上の入れ子をカプセル化するテンプレート クラス。|
