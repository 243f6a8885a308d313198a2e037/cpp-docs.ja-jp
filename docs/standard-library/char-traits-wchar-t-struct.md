# <a name="chartraitsltwchartgt-struct"></a>char_traits&lt;wchar_t&gt; 構造体

## <a name="syntax"></a>構文

```cpp
template <>
struct char_traits<wchar_t>;
```

## <a name="remarks"></a>Remarks
