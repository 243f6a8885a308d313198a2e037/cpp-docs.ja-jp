# <a name="messagesbase-class"></a>messages_base クラス

基本クラスについて説明します、 **int**メッセージのカタログの型。

## <a name="syntax"></a>構文

```cpp
struct messages_base : locale::facet {
    typedef int catalog;
    explicit messages_base(size_t _Refs = 0)
};
```
