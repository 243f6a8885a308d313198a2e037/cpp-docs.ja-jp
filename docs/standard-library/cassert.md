# &lt;cassert&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

標準 C ライブラリ ヘッダー \<assert.h\> をインクルードし、関連する名前を `std` 名前空間に追加します。  
  
## 構文  
  
```  
  
#include <cassert>  
  
```  
  
## 解説  
 このヘッダーをインクルードすると、標準 C ライブラリ ヘッダーの外部リンケージで宣言された名前が、`std` 名前空間でも宣言されます。
