## <a name="swap"></a>  swap

2 つの deque の要素を交換します。

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>パラメーター

`left` 型のオブジェクト`deque`です。

`right` 型のオブジェクト`deque`です。
