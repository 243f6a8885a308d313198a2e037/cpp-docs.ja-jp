##  <a name="swap"></a>  swap  
 2 つの deque の要素を交換します。  
  
```
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```  
  
### <a name="parameters"></a>パラメーター  
 `left`  
 `deque` 型のオブジェクト。  
  
 `right`  
 `deque` 型のオブジェクト。
