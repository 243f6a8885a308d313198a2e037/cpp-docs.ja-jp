# &lt;forward_list&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

コンテナーのテンプレート クラス forward\_list および複数のサポートのテンプレートを定義します。  
  
## 構文  
  
```  
#include <forward_list>  
```

|[演算子\! \= \(\<forward\_list\>\)](../Topic/operator!=%20\(%3Cforward_list%3E\).md)|演算子の左側に、リスト オブジェクトが右側前方リスト オブジェクトと等しくないかどうかを調べます。|  
|[演算子\< \(\<forward\_list\>\)](../Topic/operator%3C%20\(%3Cforward_list%3E\).md)|演算子の左側に、リスト オブジェクトが右側前方リスト オブジェクトより小さいかどうかを調べます。|  
|[operator\=\< \(\<forward\_list\>\)](../Topic/operator%3C=%20\(%3Cforward_list%3E\).md)|演算子の左側に、リスト オブジェクトが右側前方リスト オブジェクト以下かどうかをテストします。|
