##  <a name="a-namegenericerrnoa--genericerrno"></a><a name="generic_errno"></a>  generic_errno  
 `<errno.h>` の Posix によって定義される、すべてのエラーコード マクロのシンボル名を提供する列挙型を表す型。  
  
```
typedef errc generic_error;
```  
  
### <a name="remarks"></a>コメント  
 この型は、[errc](../standard-library/system-error-enums.md#errc_enumeration) の同意語です。
