# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;

標準ヘッダーを含める\<typeinfo > 型識別演算子に関連付けられているいくつかの型を定義する[typeid 演算子](../cpp/typeid-operator.md)、静的および動的の両方の種類の情報を生成します。

## <a name="syntax"></a>構文

```cpp
#include <typeinfo>

```
