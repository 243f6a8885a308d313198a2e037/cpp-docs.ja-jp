# &lt;unordered_set&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

コンテナーのテンプレート クラス [unordered\_multiset](../standard-library/unordered-multiset-class.md) と [unordered\_set](../standard-library/unordered-set-class.md) およびサポートするテンプレート定義します。  
  
## 構文  
  
```  
#include <unordered_set>  
```  
  
### クラス  
  
|||  
|-|-|  
|[unordered\_multiset クラス](../standard-library/unordered-multiset-class.md)|キーのハッシュ テーブルを保存します。|  
|[unordered\_set クラス](../standard-library/unordered-set-class.md)|キーのハッシュ テーブルを保存します。|  
  
### 関数  
  
|||  
|-|-|  
|[operator\!\= \(unordered\_multiset\)](../Topic/operator!=%20\(unordered_multiset\).md)|演算子の左側の unordered\_multiset オブジェクトが右側の unordered\_multiset オブジェクトと等しくないかどうかをテストします。|  
|[operator\=\= \(unordered\_multiset\)](../Topic/operator==%20\(unordered_multiset\).md)|演算子の左側の unordered\_multiset オブジェクトが右側の unordered\_multiset オブジェクトと等しいかどうかをテストします。|  
|[swap 関数 \(unordered\_multiset\)](../Topic/swap%20Function%20\(unordered_multiset\).md)|交換 2 個のマルチセット。|  
|[operator\!\= \(unordered\_set\)](../Topic/operator!=%20\(unordered_set\).md)|演算子の左側の unordered\_set オブジェクトが右側の unordered\_set オブジェクトと等しくないかどうかをテストします。|  
|[operator\=\= \(unordered\_set\)](../Topic/operator==%20\(unordered_set\).md)|演算子の左側の unordered\_set オブジェクトが右側の unordered\_set オブジェクトと等しいかどうかをテストします。|  
|[swap 関数 \(unordered\_set\)](../Topic/swap%20Function%20\(unordered_set\).md)|交換 2 セット。|
