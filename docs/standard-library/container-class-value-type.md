# コンテナー クラス::value_type
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

> [!NOTE]
>  このトピックでは、標準 C\+\+ ライブラリで使用されるコンテナーの、機能的な例として、Visual C\+\+ のドキュメントにあります。  詳細については、「[STL コンテナー](../standard-library/stl-containers.md)」を参照してください。  
  
 テンプレート パラメーター **Ty**のシノニムを満たします。  
  
## 構文  
  
```  
  
typedef T4 value_type;  
  
```  
  
## 解説  
 これは、未指定の型 **T4** \(通常 **Alloc::value\_type**\) のシノニムとしてとして記述されています。
