# <a name="ltctimegt"></a>&lt;ctime&gt;
標準 C ライブラリ ヘッダー \<time.h> をインクルードし、関連する名前を `std` 名前空間に追加します。  
  
## <a name="syntax"></a>構文  
  
```  
#include <ctime>  
  
```  
  
## <a name="remarks"></a>コメント  
 このヘッダーをインクルードすると、標準 C ライブラリ ヘッダーの外部リンケージで宣言された名前が、`std` 名前空間でも宣言されます。
