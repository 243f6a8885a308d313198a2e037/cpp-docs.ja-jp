# [出力] ([MIDL] プロパティ ページ)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

\[MIDL\] フォルダーの \[出力\] プロパティ ページでは、以下の MIDL コンパイラ オプションを指定します。  
  
-   出力ディレクトリ \([\/out](http://msdn.microsoft.com/library/windows/desktop/aa367358)\)  
  
-   ヘダー ファイル \([\/h](http://msdn.microsoft.com/library/windows/desktop/aa367325)\)  
  
-   DLL データ ファイル \([\/dlldata](http://msdn.microsoft.com/library/windows/desktop/aa367322)\)  
  
-   IID ファイル \([\/iid](http://msdn.microsoft.com/library/windows/desktop/aa367329)\)  
  
-   プロキシ ファイル \([\/proxy](http://msdn.microsoft.com/library/windows/desktop/aa367362)\)  
  
-   タイプ ライブラリを生成 \([\/notlb](http://msdn.microsoft.com/library/windows/desktop/aa367343)\)  
  
-   タイプ ライブラリ \([\/tlb](http://msdn.microsoft.com/library/windows/desktop/aa367372)\)  
  
 \[MIDL\] フォルダーの \[出力\] プロパティ ページを表示する方法については、「[方法 : &#91;プロパティ ページ&#93; でプロジェクトのプロパティを指定する](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md)」を参照してください。  
  
 C\+\+ プロジェクト用の MIDL オプションにプログラムでアクセスする方法については、「<xref:Microsoft.VisualStudio.VCProjectEngine.VCMidlTool>」を参照してください。
