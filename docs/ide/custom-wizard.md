# カスタム ウィザード
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ カスタム ウィザードは特別な種類のウィザードで、独自のウィザード作成に役立ちます。  作成したウィザードの設定を適切に行うと、そのウィザードで、プロジェクトに必要な初期ファイルを生成できるようになります。  
  
## 概要  
 ウィザードのこのページでは、作成するカスタム ウィザード プロジェクトの現在のアプリケーション設定が示されます。  既定では、このウィザードでは、次のようなウィザード プロジェクトが作成されます。  
  
-   ウィザード プロジェクトのフレンドリ名として、**\[新しいプロジェクト\] ダイアログ ボックス**で指定されたプロジェクト名が設定されます。  
  
-   ウィザード プロジェクトにはユーザー インターフェイスが表示されます。  
  
-   ウィザード プロジェクトのユーザー インターフェイスに表示されるページは 1 つです。  
  
 これらの既定の設定は、ウィザードの [&#91;アプリケーションの設定&#93;](../Topic/Application%20Settings,%20Custom%20Wizard.md) ページで変更できます。
