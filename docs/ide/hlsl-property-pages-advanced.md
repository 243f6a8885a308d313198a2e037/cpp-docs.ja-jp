# <a name="hlsl-property-pages-advanced"></a>[詳細] ([HLSL] プロパティ ページ)
HLSL コンパイラ (fxc.exe) の次のプロパティを構成するのには、使用、**詳細**プロパティ ページ。 アクセスする方法については、**詳細**HLSL フォルダー内のプロパティ ページを参照してください[のプロジェクト プロパティの操作](../ide/working-with-project-properties.md)です。  
  
## <a name="uielement-list"></a>UIElement の一覧  
 **著作権情報を抑制します。**  
 **はい (/nologo)**スタートアップ バナーと情報メッセージの表示を抑制するそれ以外の場合、**いいえ**です。 値は、既定では、**はい (/nologo)**です。  
  
 **警告をエラーとして扱う**  
 **はい (/WX)**すべてのコンパイラ警告をエラーとして処理するそれ以外の場合、**いいえ**です。
