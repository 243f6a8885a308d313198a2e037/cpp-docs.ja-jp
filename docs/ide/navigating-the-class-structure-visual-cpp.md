# クラス各部へのジャンプ (Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[クラス ビュー](http://msdn.microsoft.com/ja-jp/8d7430a9-3e33-454c-a9e1-a85e3d2db925)または[オブジェクト ブラウザー](http://msdn.microsoft.com/ja-jp/f89acfc5-1152-413d-9f56-3dc16e3f0470)を使用して、関数、継承、およびインターフェイスを含む、プロジェクトのクラスの各部分を移動できます。  
  
 クラス ビューで関数名をダブルクリックすると、関数宣言に移動できます。  クラス ビューでクラス名をダブルクリックすると、クラス定義に移動できます。
