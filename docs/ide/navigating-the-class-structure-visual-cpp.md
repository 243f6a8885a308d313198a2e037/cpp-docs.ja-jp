# <a name="navigate-the-class-structure"></a>クラス構造を移動する

**クラス ビュー**または**オブジェクト ブラウザー**のいずれかを使用して、プロジェクトのクラス構造を移動できます。 プロジェクトのクラス構造には、関数、継承、およびインターフェイスが含まれる場合があります。 詳細については、「[コードの構造の表示](/visualstudio/ide/viewing-the-structure-of-code)」を参照してください。

関数の宣言にジャンプするには、クラス ビューで関数名をダブルクリックします。 クラスの定義にジャンプするには、クラス ビューでクラス名をダブルクリックします。
