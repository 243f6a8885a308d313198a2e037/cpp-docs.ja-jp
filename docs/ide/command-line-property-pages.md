# [コマンド ライン] プロパティ ページ
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ほとんどのプロパティ ページのフォルダーには、**\[コマンド ライン\]** プロパティ ページが含まれています。  このページには、フォルダーで設定されるプロパティが表示されます。  **\[コマンド ライン\]** プロパティ ページには **\[追加のオプション\]** ボックスもあります。このボックスでは、ツール \(フォルダーにプロパティがないツール\) に対して有効なプロパティを指定できます。  
  
 エディット ボックスに入力するコマンドは、すべてフォルダーのツールに渡されます。  入力したコマンドに対しては、検査や確認が行われません。依存関係もチェックされません。  
  
 **\[コマンド ライン\]** プロパティ ページを表示する方法については、「[方法 : &#91;プロパティ ページ&#93; でプロジェクトのプロパティを指定する](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md)」を参照してください。
