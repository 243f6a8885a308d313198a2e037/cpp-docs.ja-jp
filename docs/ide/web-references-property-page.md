# [Web 参照] プロパティ ページ
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

\[Web 参照\] プロパティ ページでは、XML Web サービスのプロキシ クラスの生成方法を指定します。  XML Web サービスのプロキシ クラスは、プロジェクトに Web 参照を追加した場合に生成されます。  詳細については、「[&#91;Web 参照の追加&#93; ダイアログ ボックス](http://msdn.microsoft.com/ja-jp/bdf05776-c591-40af-bfd7-e1e2aa1e87b5)」を参照してください。  
  
 \[Web 参照\] プロパティ ページには、以下のプロパティが含まれています。  
  
 **\[出力ファイル\]**  
 XML Web サービスのプロキシ クラスを含むファイルの名前です。  
  
 **\[著作権情報の非表示\]**  
 Web サービス記述言語ツール \(Wsdl.exe\) の情報が表示されません。  
  
 **\[名前空間\]**  
 生成される Web プロキシの名前を指定します。  
  
 **\[追加する参照\]**  
 プロキシ DLL が参照する追加の DLL を指定します。  
  
 \[Web 参照\] プロパティ ページを表示する方法については、「[方法 : &#91;プロパティ ページ&#93; でプロジェクトのプロパティを指定する](../Topic/How%20to:%20Specify%20Project%20Properties%20with%20Property%20Pages.md)」を参照してください。
