# ATL/MFC の共有クラス
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Visual C\+\+ .NET 2002 以降そのほかの MFC クラスの依存関係を減らすために、複数の既存の MFC のユーティリティ クラスを書き換えるまたは変更されました。  これらのユーティリティ クラスは、ネイティブ C\+\+ プロジェクトで使用できます。  ここでは、MFC プロジェクトに以前使用できる、共有されている `CString`の変更に関するいくつかの新しいクラスとクラスのみが含まれます。  適切なヘッダーの信頼しているネイティブ C\+\+ プロジェクトの種類で使用できる ATL クラスは含まれません。
