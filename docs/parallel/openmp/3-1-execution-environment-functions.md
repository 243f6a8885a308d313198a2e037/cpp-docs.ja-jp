- **omp_get_num_threads**関数。

- **omp_get_max_threads**関数。

- **omp_get_thread_num**関数。

- **omp_get_num_procs**関数。

- **omp_in_parallel**関数。

- **omp_set_dynamic**関数。

- **omp_get_dynamic**関数。

- **omp_set_nested**関数。

- **omp_get_nested**関数。
