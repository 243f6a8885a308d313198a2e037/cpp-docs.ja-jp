# 3.1 Execution Environment Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

このセクションで説明する関数はスレッドプロセッサ並列環境に影響を及ぼす監視します :  
  
-   **omp\_set\_num\_threads** の関数。  
  
-   **omp\_get\_num\_threads** の関数。  
  
-   **omp\_get\_max\_threads** の関数。  
  
-   **omp\_get\_thread\_num** の関数。  
  
-   **omp\_get\_num\_procs** の関数。  
  
-   **omp\_in\_parallel** の関数。  
  
-   **omp\_set\_dynamic** の関数。  
  
-   **omp\_get\_dynamic** の関数。  
  
-   **omp\_set\_nested** の関数。  
  
-   **omp\_get\_nested** の関数。
