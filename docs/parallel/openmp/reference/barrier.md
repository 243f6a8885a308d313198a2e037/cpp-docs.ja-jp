# barrier
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

チームでスレッドを同期します ; すべてのスレッドがバリアにすべてのスレッドがバリアを実行するまで一時停止します。  
  
## 構文  
  
```  
#pragma omp barrier  
```  
  
## 解説  
 `barrier` のディレクティブはOpenMP 句をサポートしていません。  
  
 詳細については、「[2.6.3 barrier Directive](../../../parallel/openmp/2-6-3-barrier-directive.md)」を参照してください。  
  
## 使用例  
 `barrier` を使用する方法の例については [master](../../../parallel/openmp/reference/master.md) を参照してください。
