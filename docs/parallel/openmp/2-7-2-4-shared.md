# <a name="2724-shared"></a>2.7.2.4 shared

この句に出現する変数を共有する、*変数一覧*チーム内のすべてのスレッド間で。 チーム内のすべてのスレッドへのアクセスに同じ記憶域**共有**変数。

構文、**共有**句を次に示します。

```
shared(variable-list)
```
