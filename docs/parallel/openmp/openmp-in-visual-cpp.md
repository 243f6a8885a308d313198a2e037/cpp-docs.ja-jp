# <a name="openmp-in-visual-c"></a>Visual C++ の OpenMP

OpenMP C および C++ アプリケーション プログラミング インターフェイスを使用して、複数のプロセッサを効果的に使用するアプリケーションを記述できます。 Visual C では、OpenMP 2.0 標準をサポートします。
