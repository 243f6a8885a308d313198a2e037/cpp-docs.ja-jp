# 2. Directives
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ディレクティブは C および C\+\+ 規格に定義されている **\#pragma** のディレクティブに基づいています。  OpenMP の C をサポートしC\+\+ API がアクティブ化 OpenMP のすべてのコンパイラ ディレクティブの解釈を含むコマンド ライン オプションをコンパイラが可能になります。
