# <a name="shortvector-structure"></a>short_vector 構造体
short_vector は short ベクターを汎用的にプログラミングに役立つメタプログラミングの定義を提供します。

```

    typename _Scalar_type,

template<>

template<>

template<>

template<>  
struct short_vector<unorm, 4>;
```  
