# <a name="ml-nonfatal-error-a2038"></a>ML の致命的でないエラー A2038
**マクロの演算子のオペランドがありません。**  
  
 アセンブラー見つかったマクロのパラメーター リストの末尾の直後に、 [!](../../assembler/masm/operator-logical-not-masm-run-time.md) または[ % ](../../assembler/masm/operator-percent.md)演算子。
