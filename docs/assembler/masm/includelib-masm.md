# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

現在のモジュールとリンクする必要が *libraryname* リンカーであることを通知します。  
  
## 構文  
  
```  
  
INCLUDELIB libraryname  
```  
  
## 解説  
 *libraryname は* 山かっこ内のシンボル単一引用符または二重引用符未満シンボル的なバックスラッシュセミコロンが含まれている囲む必要があります。
