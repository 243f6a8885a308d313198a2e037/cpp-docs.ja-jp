# <a name="comment-masm"></a>COMMENT (MASM)
すべてを扱う*テキスト*間、またはコメントとして区切り記号と同じ行にします。  
  
## <a name="syntax"></a>構文  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
