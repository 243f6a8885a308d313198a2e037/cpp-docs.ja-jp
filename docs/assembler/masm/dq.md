# <a name="dq"></a>DQ
割り当て、ごとに 8 バイトの記憶領域を必要に応じて初期化`initializer`です。 型指定子としても使用できる、型が有効な任意の場所。 `DQ` シノニムは、 [QWORD](../../assembler/masm/qword.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] DQ initializer [[, initializer]]...  
```
