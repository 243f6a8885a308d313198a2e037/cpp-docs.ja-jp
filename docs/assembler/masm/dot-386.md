# <a name="386"></a>.386
アセンブリの 80386 プロセッサの特権のない命令以降のプロセッサで導入された命令のアセンブリを無効にします。  
  
## <a name="syntax"></a>構文  
  
```  
  
.386  
  
```  
  
## <a name="remarks"></a>コメント  
 80387 指示こともできます。
