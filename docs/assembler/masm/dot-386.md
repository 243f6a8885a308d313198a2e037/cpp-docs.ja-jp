# <a name="386"></a>.386

アセンブリの 80386 プロセッサの特権命令以降のプロセッサで導入された命令のアセンブリを無効にします。

## <a name="syntax"></a>構文

> .386

## <a name="remarks"></a>Remarks

手順については 80387 こともできます。
