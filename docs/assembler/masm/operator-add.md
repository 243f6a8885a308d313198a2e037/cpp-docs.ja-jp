# operator +
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

一つ目の演算子は *expression2* を *expression1 を*  返します。  2 番目の演算子は *式*  の符号を反転させます。  
  
## 構文  
  
```  
  
      expression1 + expression2  
–expression  
```
