# <a name="macro"></a>MACRO

マークと呼ばれるマクロ ブロック*名前*と確立*パラメーター*マクロが呼び出されたときに渡される引数のプレース ホルダー。

## <a name="syntax"></a>構文

> *名前*マクロ [*パラメーター* [: REQ |: =*既定*|: VARARG]].<br/>
> *ステートメント*<br/>
> ENDM [*値*]

## <a name="remarks"></a>Remarks

マクロ、関数が返す*値*呼び出し元のステートメントにします。
