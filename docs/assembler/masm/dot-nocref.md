# <a name="nocref"></a>.NOCREF
シンボルをシンボル テーブルとブラウザー ファイルの一覧を表示しません。  
  
## <a name="syntax"></a>構文  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## <a name="remarks"></a>コメント  
 名を指定すると、指定された名前だけが抑制されます。 同じ[です。XCREF](../../assembler/masm/dot-xcref.md)です。
