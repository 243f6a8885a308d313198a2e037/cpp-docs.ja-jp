# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

シンボル テーブルとブラウザー ファイルのシンボルの一覧を表示します。  
  
## 構文  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## 解説  
 名前が指定した場合は指定した名前のみ適用されます。  [.XCREF](../../assembler/masm/dot-xcref.md) と同じです。
