# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**式の演算子がありません**  
  
 式は演算子が不足しているため評価できません。  このエラー メッセージは前のプログラム エラーの影響である場合があります。  
  
 次の行はエラーを生成します :  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
