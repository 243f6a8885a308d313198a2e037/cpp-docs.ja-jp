# <a name="ml-nonfatal-error-a2037"></a>ML の致命的でないエラー A2037
**ステートメントの構造体の定義内で使用できません。**  
  
 構造体の定義には、無効なステートメントが含まれています。  
  
 構造体は、手順、ラベル、プロシージャ、制御フローのディレクティブを含めることはできません[です。スタートアップ](../../assembler/masm/dot-startup.md)、または[です。終了](../../assembler/masm/dot-exit.md)です。
