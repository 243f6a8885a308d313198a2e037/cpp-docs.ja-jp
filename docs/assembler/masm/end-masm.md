# END (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

モジュールの末尾を指定しオプションで *アドレス*  にプログラムのエントリ ポイントを設定します。  
  
## 構文  
  
```  
  
END [[address]]  
```
