# <a name="ymmword"></a>YMMWORD

Intel Advanced Vector Extensions (AVX) 命令で 256 ビットのマルチ メディア オペランドに使用されます。

## <a name="syntax"></a>構文

> YMMWORD

## <a name="remarks"></a>Remarks

`YMMWORD` 同じ型を表すために、 `__m256` AVX 組み込みの。
