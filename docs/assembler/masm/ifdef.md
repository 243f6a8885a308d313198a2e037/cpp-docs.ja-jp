# IFDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*名前が*  前に定義されたラベル変数または記号のアセンブリができます。  
  
## 構文  
  
```  
  
IFDEF name  
```  
  
## 解説  
 完全な構文については [IF](../../assembler/masm/if-masm.md) を参照してください。
