# <a name="ifdef"></a>IFDEF
場合は、アセンブリを付与*名前*が以前に定義されたラベル、変数、またはシンボルです。  
  
## <a name="syntax"></a>構文  
  
```  
  
IFDEF name  
```  
  
## <a name="remarks"></a>コメント
