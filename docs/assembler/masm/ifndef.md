# IFNDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*名前が*  定義されていないアセンブリができます。  
  
## 構文  
  
```  
  
IFNDEF name  
```  
  
## 解説  
 完全な構文については [IF](../../assembler/masm/if-masm.md) を参照してください。
