# ML Nonfatal Error A2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**互換性のないモード CPU およびセグメントのサイズ**  
  
 とと32 ビット **USE16** セグメントで**USE32**または  **フラット**  セグメントの属性を指定した CPU との互換性開かせますではなく16 ビットの CPU に変更する。  
  
 .386 より大きいプロセッサ ディレクティブに **USE32** と  **フラット**  の属性を指定する必要があります。
