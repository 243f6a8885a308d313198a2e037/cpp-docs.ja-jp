# <a name="listmacro"></a>.LISTMACRO
コードまたはデータを生成するマクロ展開ステートメントの一覧を開始します。  
  
## <a name="syntax"></a>構文  
  
```  
  
.LISTMACRO  
  
```  
  
## <a name="remarks"></a>コメント  
 既定値です。 同じ[です。XALL](../../assembler/masm/dot-xall.md)です。
