# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コードまたはデータを生成するマクロの展開のステートメントのリストを呼び出します。  
  
## 構文  
  
```  
  
.LISTMACRO  
  
```  
  
## 解説  
 これは、既定の設定です。  [.XALL](../Topic/.XALL.md) と同じです。
