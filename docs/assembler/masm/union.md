# <a name="union"></a>UNION
1 つまたは複数のデータ型の共用体を宣言します。 *Fielddeclarations*有効なデータの定義をする必要があります。 省略、[終了](../../assembler/masm/ends-masm.md)*名前*ラベルの上に入れ子になった**共用体**定義します。  
  
## <a name="syntax"></a>構文  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
