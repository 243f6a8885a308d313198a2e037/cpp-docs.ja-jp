# <a name="union"></a>UNION

1 つまたは複数のデータ型の共用体を宣言します。 *フィールド宣言*有効なデータの定義である必要があります。 省略、[終了](../../assembler/masm/ends-masm.md)*名前*上のラベルが入れ子になった**共用体**定義します。

## <a name="syntax"></a>構文

> *名前*共用体 [*配置*] [, 一意でない]<br/>
> *フィールドの宣言*<br/>
> [*名前*] を終了
