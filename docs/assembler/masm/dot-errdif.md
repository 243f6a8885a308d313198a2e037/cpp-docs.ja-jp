# <a name="errdifi"></a>.ERRDIF[[I]]
テキスト アイテムが異なる場合は、エラーを生成します。  
  
## <a name="syntax"></a>構文  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>コメント  
 場合**すれば**が指定すると、比較では大文字小文字を区別します。
