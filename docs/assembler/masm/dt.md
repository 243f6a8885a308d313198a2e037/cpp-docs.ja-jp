# <a name="dt"></a>DT
割り当て、必要に応じて各 10 バイトのストレージを初期化`initializer`です。 型指定子として使用することも、型が有効な任意の場所。 `DT` シノニムは、 [TBYTE](../../assembler/masm/tbyte.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
