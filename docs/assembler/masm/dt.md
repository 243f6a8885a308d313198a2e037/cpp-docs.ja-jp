# DT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` の領域の 10 バイトを割り当て必要に応じて初期化します。  型指定子がどこでも可能型であるため使用できます。  `DT` は [TBYTE](../../assembler/masm/tbyte.md) のシノニムです。  
  
## 構文  
  
```  
[[name]] DT initializer [[, initializer]]...  
```
