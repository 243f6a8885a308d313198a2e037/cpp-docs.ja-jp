# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` の領域の署名されたダブル ワード \(4 バイト\) を割り当て必要に応じて初期化します。  また、型が適正である場合は、型指定子としても使用できます。  
  
## 構文  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
