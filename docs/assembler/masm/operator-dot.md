# operator .
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

一つ目の演算子は構造体や共用体の  *フィールド*  のオフセット  *を含む式*  を返します。  位置の 2 番目の演算子の戻り値は構造体または共用体の  *フィールド*  のオフセットに  *登録*  してをポイントします。  
  
## 構文  
  
```  
  
      expression  
      . field [[. field]]...  
[register]. field [[. field]]...  
```
