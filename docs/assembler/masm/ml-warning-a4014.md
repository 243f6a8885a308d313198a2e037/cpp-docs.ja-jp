# <a name="ml-warning-a4014"></a>ML の警告 A4014
手順と初期化されたデータが BSS セグメント単位でサポートされていません  
  
 BSS セクション内で初期化されたデータを定義しようとしました。  BSS セクションは、名前がある BSS クラスとして定義されます。  これには、簡略化のセグメントが含まれます`.data?`です。
