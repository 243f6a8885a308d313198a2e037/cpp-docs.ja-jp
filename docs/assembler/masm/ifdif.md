# <a name="ifdif"></a>IFDIF]
テキスト アイテムが異なる場合は、アセンブリを行います。  
  
## <a name="syntax"></a>構文  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## <a name="remarks"></a>コメント  
 場合**すれば**が指定すると、比較では大文字小文字を区別します。 参照してください[IF](../../assembler/masm/if-masm.md)の完全な構文です。
