# <a name="ml-nonfatal-error-a2107"></a>ML の致命的でないエラー A2107
**暗黙の相手側ジャンプまたはラベルの近くに呼び出すことはできません。**  
  
 遠くに移動または別のセグメントの near ラベルへの呼び出しを暗黙的に作成しようとしました。
