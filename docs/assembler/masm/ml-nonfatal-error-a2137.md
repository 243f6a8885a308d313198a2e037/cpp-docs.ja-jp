# <a name="ml-nonfatal-error-a2137"></a>ML の致命的でないエラー A2137

**INVOKE への引数が少なすぎます**

使用して渡される引数の数、 [INVOKE](../../assembler/masm/invoke.md)ディレクティブが必要なパラメーターが呼び出されるプロシージャのプロトタイプで指定された数よりも少なくなります。
