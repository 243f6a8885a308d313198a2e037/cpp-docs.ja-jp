# <a name="sword"></a>SWORD
割り当て、必要に応じて各記憶域の符号付きの単語 (2 バイト) を初期化`initializer`です。 型指定子として使用することも、型が有効な任意の場所。  
  
## <a name="syntax"></a>構文  
  
```  
  
name SWORD initializer [[, initializer]]...  
```
