# <a name="ml-fatal-error-a1009"></a>ML の致命的なエラー A1009
**行が長すぎます**  
  
 ソース ファイル内の行では、512 文字の制限を超えました。  
  
 行連結文字 (\) で複数の物理行が連結された場合、結果として得られる論理行はも、512 文字に制限します。
