# ML Fatal Error A1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**行が長すぎます。**  
  
 ソース ファイルの 512 行の文字の制限を超えています。  
  
 行連結文字を持つ複数の物理行 \(\\\) が連結すると生成された論理行は512 文字に制限されています。
