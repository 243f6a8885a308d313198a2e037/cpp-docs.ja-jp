# IF2
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**IF** ブロックはアセンブリのパスで  **オプション : SETIF2** が **True** 場合と評価されます。  
  
## 構文  
  
```  
  
IF2   
expression  
  
```  
  
## 解説  
 完全な構文については [IF](../../assembler/masm/if-masm.md) を参照してください。
