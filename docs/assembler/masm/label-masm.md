# <a name="label-masm"></a>LABEL (MASM)

現在の場所カウンター値を割り当てることで新しいラベルを作成しますと、指定された*型*に*名前*。

## <a name="syntax"></a>構文

> *名前*ラベル*型*
>
> *名前*ラベル\[NEAR |ここまで |PROC] PTR \[*型*]
