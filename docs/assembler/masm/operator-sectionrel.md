# <a name="operator-sectionrel"></a>SECTIONREL 演算子
最終的な実行可能ファイルのターゲットがあるセクションの基準とした式のセクションの相対オフセットを返します。  
  
## <a name="syntax"></a>構文  
  
```  
SECTIONREL expression  
```  
  
## <a name="remarks"></a>コメント  
 SECTIONREL は、COFF オブジェクトの出力でのみ使用できます。
