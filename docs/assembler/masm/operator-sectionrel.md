# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

最終的な実行可能ファイルからターゲットを含むセクションに対する式のセクションの相対的なオフセットを返します。  
  
## 構文  
  
```  
SECTIONREL expression  
```  
  
## 解説  
 SECTIONREL はCOFF オブジェクトの出力でのみ使用できます。
