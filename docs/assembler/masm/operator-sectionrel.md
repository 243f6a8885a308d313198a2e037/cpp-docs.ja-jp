# <a name="operator-sectionrel"></a>SECTIONREL 演算子

最終的な実行可能ファイルでターゲットを格納しているセクションの基準とした式のセクションの相対オフセットを返します。

## <a name="syntax"></a>構文

> SECTIONREL 式

## <a name="remarks"></a>Remarks

SECTIONREL が COFF オブジェクトの出力でのみ使用できます。
