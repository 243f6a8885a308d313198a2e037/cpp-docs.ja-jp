# <a name="dd"></a>DD
割り当て、必要に応じて 1 ダブルワード (4 バイト) を各記憶域の初期化`initializer`です。 `DD` シノニムは、 [DWORD](../../assembler/masm/dword.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>コメント  
 型指定子として使用することも、型が有効な任意の場所。
