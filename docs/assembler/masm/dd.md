# DD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` の領域のダブル ワード \(4 バイト\) を割り当て必要に応じて初期化します。  `DD` は [ダブルワード](../../assembler/masm/dword.md) のシノニムです。  
  
## 構文  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## 解説  
 型指定子がどこでも可能型であるため使用できます。
