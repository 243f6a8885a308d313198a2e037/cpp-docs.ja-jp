# ML Nonfatal Error A2119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**言語を指定する必要があります**  
  
 プロシージャの定義またはプロトタイプは言語を指定しました。  
  
 言語は各プロシージャの定義またはプロトタイプで既定の言語の型を指定する宣言する必要があります。  既定の言語の型は [.MODEL](../../assembler/masm/dot-model.md) のディレクティブ **オプション LANG**または ML のコマンド ライン オプション **\/Gc** または **\/Gd** を使用して設定します。
