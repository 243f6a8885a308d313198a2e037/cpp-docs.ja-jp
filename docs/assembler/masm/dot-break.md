# .BREAK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*条件が* true のコードを [.WHILE](../../assembler/masm/dot-while.md) または [.REPEAT](../../assembler/masm/dot-repeat.md) ブロックを終了する場合に生成されます。  
  
## 構文  
  
```  
  
.BREAK [[.IF condition]]   
```
