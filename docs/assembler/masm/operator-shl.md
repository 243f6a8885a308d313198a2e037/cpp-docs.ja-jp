# operator SHL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ビットの  *式に保持*  される  *数*  のビットをシフトした結果を返します。  
  
## 構文  
  
```  
  
expression SHL count  
```
