# <a name="erridn"></a>.ERRIDN]
テキスト アイテムが同一の場合は、エラーを生成します。  
  
## <a name="syntax"></a>構文  
  
```  
  
.ERRIDN[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## <a name="remarks"></a>コメント  
 場合**すれば**が指定すると、比較では大文字小文字を区別します。
