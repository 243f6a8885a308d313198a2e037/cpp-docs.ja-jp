# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*名前*  に *textitem を*  割り当てます。   *textitem は* リテラル文字列`%` を指定する定数またはマクロ関数によって返される文字列のいずれかになります。  
  
## 構文  
  
```  
  
name TEXTEQU [[textitem]]  
```
