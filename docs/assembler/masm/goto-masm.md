# GOTO (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**:** の行でマークされた *macrolabel* にアセンブリをコピーします。  
  
## 構文  
  
```  
  
GOTO   
macrolabel  
  
```  
  
## 解説  
 **GOTO** は [マクロ](../Topic/MACRO.md)[FOR](../../assembler/masm/for-masm.md)[FORC](../Topic/FORC.md)[繰り返し](../../assembler/masm/repeat.md) と  **と**  ブロック内でのみ使用できます。  ラベルは行のディレクティブであり先頭の後に指定する必要があります。
