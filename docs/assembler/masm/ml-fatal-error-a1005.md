# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**アセンブラーの制限 : 完全マクロ テーブルのパラメーター名**  
  
 規定より多くのパラメーターローカルまたはマクロはマクロ ラベルに定義されています。  マクロ名のテーブルにこれ以上のスペースがありません。  
  
 以下の短い名前を定義したり不要なマクロを削除します。
