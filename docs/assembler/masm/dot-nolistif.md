# .NOLISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

条件が false に評価する条件ブロックのリストを制限します \(0\)。  
  
## 構文  
  
```  
  
.NOLISTIF  
  
```  
  
## 解説  
 これは、既定の設定です。  [.SFCOND](../Topic/.SFCOND.md) と同じです。
