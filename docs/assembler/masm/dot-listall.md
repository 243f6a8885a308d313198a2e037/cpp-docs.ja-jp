# .LISTALL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

すべてのステートメントのリストを呼び出します。  
  
## 構文  
  
```  
  
.LISTALL  
  
```  
  
## 解説  
 [.LIST](../Topic/.LIST.md) [.LISTIF](../../assembler/masm/dot-listif.md) と [.LISTMACROALL](../../assembler/masm/dot-listmacroall.md) の組み合わせと同じです。
