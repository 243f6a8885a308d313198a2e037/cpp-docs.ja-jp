# <a name="operator-mask"></a>MASK 演算子
返しますが、ビット マスク内のビット*recordfieldname*または*レコード*が設定され、その他のすべてのビットが消去します。  
  
## <a name="syntax"></a>構文  
  
```  
  
MASK {recordfieldname | record}  
```
