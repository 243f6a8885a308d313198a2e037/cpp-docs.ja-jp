# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

最初の演算子は *式を*  強制します。指定 `type` を持つとして処理します。  2 番目の演算子は `type` へのポインターを指定します。  
  
## 構文  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
