# ML Fatal Error A1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**開いているファイルはできません : fileName**  
  
 アセンブラーはなくソースを開きまたは出力ファイルが含まれます。  
  
 次のいずれかが原因である可能性があります :  
  
-   ファイルが存在しません。  
  
-   ファイルが別のプロセスによって使用されています。  
  
-   ファイル名が無効です。  
  
-   出力ファイル名の読み取り専用ファイルは既に存在します。  
  
-   現在のドライブがいっぱいになります。  
  
-   現在のディレクトリがルートで完全です。  
  
-   デバイスには使用できません。  
  
-   ドライブの準備が完了していません。
