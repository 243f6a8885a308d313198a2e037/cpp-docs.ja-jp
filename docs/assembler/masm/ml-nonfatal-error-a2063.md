# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**2 の累乗にのみ配置できます : 式**  
  
 [整列](../../assembler/masm/align-masm.md) のディレクティブに指定した式が無効です。  
  
 **整列**  の式は2 ~ 2562 の累乗で現在のセグメント構造体または共用体の配置の値以下である必要があります。
