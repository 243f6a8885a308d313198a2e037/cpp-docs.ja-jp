# <a name="ml-nonfatal-error-a2063"></a>ML の致命的でないエラー A2063
**2 の累乗にのみ配置ことができます式。**  
  
 指定した式、 [ALIGN](../../assembler/masm/align-masm.md)ディレクティブが無効でした。  
  
 **ALIGN**式は 2 ~ 256 の間の 2 の累乗にする必要があり、現在のセグメント、構造体、または共用体のアラインメント未満にする必要があります。
