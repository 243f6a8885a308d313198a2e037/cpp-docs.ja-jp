# <a name="fword"></a>FWORD
割り当て、ごとに 6 バイトの記憶領域を必要に応じて初期化`initializer`です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] FWORD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>コメント  
 型指定子としても使用できる、型が有効な任意の場所。
