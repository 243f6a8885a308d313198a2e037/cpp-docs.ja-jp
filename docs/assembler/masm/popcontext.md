# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

現在の `context` 復元の一部またはすべて \([PUSHCONTEXT](../../assembler/masm/pushcontext.md) のディレクティブが格納されます\)。  `context` は  **想定しています** `RADIX` **一覧 CPU**または  **すべて**  のいずれかになります。  
  
## 構文  
  
```  
  
POPCONTEXT context  
```
