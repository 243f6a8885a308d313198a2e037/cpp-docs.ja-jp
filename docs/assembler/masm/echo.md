# <a name="echo"></a>ECHO
表示*メッセージ*を標準出力デバイス (既定では、画面)。  
  
## <a name="syntax"></a>構文  
  
```  
  
ECHO   
message  
  
```  
  
## <a name="remarks"></a>コメント  
 同じ[%out](../../assembler/masm/percent-out.md)です。
