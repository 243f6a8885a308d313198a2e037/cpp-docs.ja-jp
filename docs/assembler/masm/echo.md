# ECHO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

標準出力デバイス \(既定ではは画面\) に  *メッセージを*  表示します。  
  
## 構文  
  
```  
  
ECHO   
message  
  
```  
  
## 解説  
 [%OUT](../../assembler/masm/percent-out.md) と同じです。
