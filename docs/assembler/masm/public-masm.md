# <a name="public-masm"></a>PUBLIC (MASM)
各変数、ラベル、またはとして指定されている絶対記号は、*名前*プログラムの他のすべてのモジュールを使用できます。  
  
## <a name="syntax"></a>構文  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
