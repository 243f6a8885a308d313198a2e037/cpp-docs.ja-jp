# <a name="386p"></a>.386P
アセンブリの 80386 プロセッサのすべての手順 (特権も含む)以降のプロセッサで導入された命令のアセンブリを無効にします。  
  
## <a name="syntax"></a>構文  
  
```  
  
.386P  
  
```  
  
## <a name="remarks"></a>コメント  
 80387 指示こともできます。
