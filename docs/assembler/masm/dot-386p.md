# .386P
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

80386 プロセッサのすべての命令のアセンブリ \(を含む\) にアクセスします ; 低速なプロセッサによって導入される命令のアセンブリを無効にします。  
  
## 構文  
  
```  
  
.386P  
  
```  
  
## 解説  
 80387 または命令を有効にします。
