# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

short にラベルを設定します。   *ラベルされる*  すべてのジャンプは短くなければ必要があります \(スコープ –  *ラベル*  にジャンプ命令の 128 から \+127 バイト内で\)。  
  
## 構文  
  
```  
  
SHORT label  
```
