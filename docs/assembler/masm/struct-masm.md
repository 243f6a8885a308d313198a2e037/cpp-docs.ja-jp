# <a name="struct-masm"></a>STRUCT (MASM)
指定したを持つ構造型を宣言*fielddeclarations*です。 各フィールドは、有効なデータ定義である必要があります。 同じ[STRUC](../../assembler/masm/struc.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
