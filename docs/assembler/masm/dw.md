# DW
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` の領域の単語 \(2 バイト\) を割り当て必要に応じて初期化します。  型指定子がどこでも可能型であるため使用できます。  `DW` は [word](../../assembler/masm/word.md) のシノニムです。  
  
## 構文  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
