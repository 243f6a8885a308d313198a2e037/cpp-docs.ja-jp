# <a name="dw"></a>DW
割り当て、必要に応じて各記憶域の単語 (2 バイト) を初期化`initializer`です。 型指定子として使用することも、型が有効な任意の場所。 `DW` シノニムは、 [WORD](../../assembler/masm/word.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] DW initializer [[, initializer]]...  
```
