# <a name="operator-imagerel"></a>IMAGEREL 演算子

イメージの相対オフセットを返します`expression`します。

## <a name="syntax"></a>構文

> IMAGEREL 式

## <a name="remarks"></a>Remarks

結果の値は、RVA または相対仮想アドレスとも呼ばれます。

IMAGEREL が COFF オブジェクトの出力でのみ使用できます。
