# operator IMAGEREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`expression` イメージの相対的なオフセットを返します。  
  
## 構文  
  
```  
IMAGEREL expression  
```  
  
## 解説  
 RVA または相対仮想アドレスとして結果値が参照される場合があります。  
  
 IMAGEREL はCOFF オブジェクトの出力でのみ使用できます。
