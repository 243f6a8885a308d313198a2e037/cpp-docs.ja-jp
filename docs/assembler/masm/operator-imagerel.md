# <a name="operator-imagerel"></a>IMAGEREL 演算子
イメージの相対オフセットを返します`expression`です。  
  
## <a name="syntax"></a>構文  
  
```  
IMAGEREL expression  
```  
  
## <a name="remarks"></a>コメント  
 結果の値は、RVA または相対仮想アドレスと呼ばれます。  
  
 IMAGEREL は、COFF オブジェクトの出力でのみ使用できます。
