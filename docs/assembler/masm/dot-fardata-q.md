# .FARDATA?
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

遠いデータ セグメントの開始初期化されていないデータ セグメント名 FAR\_BSS \(または  *名前* \) に [.MODEL](../../assembler/masm/dot-model.md) に使用された場合。  
  
## 構文  
  
```  
  
.FARDATA? [[name]]  
```
