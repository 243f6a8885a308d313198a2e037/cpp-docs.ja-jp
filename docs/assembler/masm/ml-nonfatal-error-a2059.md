# ML Nonfatal Error A2059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

 条件付きディレクティブの [.ERRIDN](../../assembler/masm/dot-erridn.md) またはエラーが **.ERRIDNI** このエラーを生成するために使用されています。
