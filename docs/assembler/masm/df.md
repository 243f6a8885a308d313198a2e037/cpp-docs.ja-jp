# <a name="df"></a>DF
割り当て、ごとに 6 バイトの記憶領域を必要に応じて初期化`initializer`です。 `DF` シノニムは、 [FWORD](../../assembler/masm/fword.md)です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] DF initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>コメント  
 型指定子としても使用できる、型が有効な任意の場所。
