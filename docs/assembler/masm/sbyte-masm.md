# SBYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` の領域の符号付きバイトを割り当て必要に応じて初期化します。  型指定子がどこでも可能型であるため使用できます。  
  
## 構文  
  
```  
  
name SBYTE initializer [[, initializer]]...  
```
