# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**空の文字列 \(空白\)**  
  
 文字列は引用符や区切り文字のペアで構成ありません。  
  
 有効な文字列に 1\-255 の文字を含める必要があります。
