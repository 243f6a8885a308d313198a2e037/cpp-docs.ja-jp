# <a name="ml-nonfatal-error-a2047"></a>ML の致命的でないエラー A2047
**空 (null) の文字列**  
  
 文字列を行いました区切りと 1 組の引用符文字がありません。  
  
 有効にして、文字列の 1 ~ 255 文字以内でなければなりません。
