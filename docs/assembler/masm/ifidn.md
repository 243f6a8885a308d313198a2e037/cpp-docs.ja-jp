# IFIDN]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

テキストの項目が同じでアセンブリを使用します。  
  
## 構文  
  
```  
  
IFIDN[[I]] textitem1, textitem2  
```  
  
## 解説  
 **i** が指定されている場合比較では大文字と小文字が区別されません。  完全な構文については [IF](../../assembler/masm/if-masm.md) を参照してください。
