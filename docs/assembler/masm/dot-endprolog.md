# <a name="endprolog"></a>.ENDPROLOG
プロローグの宣言の終了を通知します。  
  
## <a name="syntax"></a>構文  
  
```  
.ENDPROLOG  
```  
  
## <a name="remarks"></a>コメント  
 プロローグの宣言のいずれかの間の領域の外部で使用すると、エラーは[PROC](../../assembler/masm/proc.md)フレームとします。ENDPROLOG です。
