# <a name="exitm"></a>EXITM

現在の繰り返しまたはマクロ ブロックの拡張を終了し、次のステートメント ブロックの外側のアセンブリを開始します。

## <a name="syntax"></a>構文

> EXITM [テキスト アイテム]

## <a name="remarks"></a>Remarks

関数では、マクロ、*テキスト アイテム*返される値。
