# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

現在の繰り返しブロックまたはマクロの展開を終了しブロックの外側に次のアセンブリを開始します。  
  
## 構文  
  
```  
  
EXITM [[textitem]]   
```  
  
## 解説  
 マクロでは関数  *は textitem* 返される値です。
