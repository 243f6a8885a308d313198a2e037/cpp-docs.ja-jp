# <a name="ml-nonfatal-error-a2078"></a>ML の致命的でないエラー A2078

**命令は FAR を許可しない間接的なアドレス指定**

条件付きのジャンプまたはループは、メモリ オペランドを使用できません。 相対アドレスまたはラベルを指定します。
