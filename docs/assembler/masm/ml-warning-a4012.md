# ML Warning A4012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**CODE のないクラスのセグメントの行番号情報**  
  
 「と」終了コード。クラス名がないセグメントに命令が発生しましたアセンブラーは次の手順の CodeView 情報は生成されませんでした。  
  
 コードは CodeView 「と」終了クラス名を使ってセグメントのコードのモジュールを処理できません
