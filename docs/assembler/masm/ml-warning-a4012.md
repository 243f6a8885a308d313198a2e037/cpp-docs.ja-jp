# <a name="ml-warning-a4012"></a>ML の警告 A4012

**クラス 'CODE' を指定せず、セグメントの行番号情報**

「コードです」で終わるクラス名がないセグメントの指示がありました アセンブラーは、次の手順の CodeView 情報を生成できませんでした。

CodeView「コードです」で終わらないクラス名を持つセグメント内のコードでのモジュールを処理できません。
