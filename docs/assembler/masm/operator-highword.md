# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*式*  の下位 32 ビットの上位 16 ビットを返します。  MASM 式は 64 ビット値です。  
  
## 構文  
  
```  
  
HIGHWORD expression  
```
