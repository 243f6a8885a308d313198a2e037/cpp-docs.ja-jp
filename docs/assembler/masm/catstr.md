# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

テキストの項目を連結します。  
  
## 構文  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## 解説  
 テキストの各項目はリテラル文字列`%` を指定する定数またはマクロ関数によって返される文字列のいずれかになります。
