# <a name="catstr"></a>CATSTR

項目のテキストを連結します。

## <a name="syntax"></a>構文

> CATSTR という名前を [textitem1 [、textitem2]...]

## <a name="remarks"></a>Remarks

各テキスト項目が前に、定数、リテラル文字列を指定できます、 `%`、またはマクロの関数によって返される文字列。
