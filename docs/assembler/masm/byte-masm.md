# <a name="byte-masm"></a>BYTE (MASM)
割り当て、必要に応じて各記憶域のバイトを初期化`initializer`です。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] BYTEinitializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>コメント  
 型指定子として使用することも、型が有効な任意の場所。
