# operator LROFFSET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*式*  のオフセットを返します。   **オフセット**  と同様にWindows がコード セグメントを再配置させるローダーによって解決されるオフセットを生成します。  
  
## 構文  
  
```  
  
LROFFSET expression  
```
