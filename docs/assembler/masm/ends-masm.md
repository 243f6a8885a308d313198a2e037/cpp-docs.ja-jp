# <a name="ends-masm"></a>ENDS (MASM)
セグメント、構造体、または共用体の末尾をマーク*名前*で以前に開始された[セグメント](../../assembler/masm/segment.md)、[構造体](../../assembler/masm/struct-masm.md)、[共用体](../../assembler/masm/union.md)、または簡略化されたセグメントディレクティブです。  
  
## <a name="syntax"></a>構文  
  
```  
  
name ENDS  
```
