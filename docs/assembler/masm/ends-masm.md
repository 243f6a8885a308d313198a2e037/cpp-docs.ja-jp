# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

セグメント構造体または [セグメント](../../assembler/masm/segment.md) 前に[構造体](../Topic/STRUCT%20\(MASM\).md)[共用体](../../assembler/masm/union.md)または簡略化セグメントのディレクティブで始まる共用体の  *名前*  の終了を示します。  
  
## 構文  
  
```  
  
name ENDS  
```
