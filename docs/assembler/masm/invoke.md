# <a name="invoke"></a>INVOKE
によって指定されたアドレスでプロシージャが呼び出される*式*、スタックまたは言語の種類の標準的な呼び出し規約に従ってレジスタで引数を渡すことです。  
  
## <a name="syntax"></a>構文  
  
```  
  
INVOKE   
expression [[, arguments]]  
```  
  
## <a name="remarks"></a>コメント  
 プロシージャに渡された各引数式、レジスタのペア、またはアドレス式が適用される場合があります (式に続く`ADDR`)。
