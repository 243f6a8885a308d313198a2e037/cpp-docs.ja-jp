# FOR (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

`argument` に一度だけ記述される各繰り返しの `parameter` を削除し現在の `argument` がブロックを示します。  
  
## 構文  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```  
  
## 解説  
 [IRP](../../assembler/masm/irp.md) と同じです。
