# <a name="for-masm"></a>FOR (MASM)
ごとに 1 回繰り返されるブロックをマーク`argument`、現在の`argument`交換`parameter`繰り返されるたびにします。  
  
## <a name="syntax"></a>構文  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```
