# Intel&#39;s MMX Instruction Set
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

## Microsoft 固有の仕様 →  
 Visual C\+\+ コンパイラにはインライン アセンブラーでIntel MMX \(マルチメディア拡張機能\) 命令セットを使用できるようになります。  MMX 命令はデバッガーの構成でサポートされます。  関数が MMX 命令が含まれているが生成したマルチメディアの状態を作る EMMS 命令は含まれません警告メッセージが表示されます。  詳細についてはIntel の Web サイトを参照してください。  
  
 **終了 Microsoft 固有の仕様→**
