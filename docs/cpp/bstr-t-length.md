# <a name="bstrtlength"></a>_bstr_t::length
**Microsoft 固有の仕様**  
  
 カプセル化された `_bstr_t` の `BSTR` の文字数を返します (終端の null は含まれません)。  
  
## <a name="syntax"></a>構文  
  
```  
  
unsigned int length ( ) const throw( );  
  
```
