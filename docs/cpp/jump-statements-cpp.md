# ジャンプ ステートメント (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C \+\+ のジャンプ ステートメントは、コントロールをローカルで即座に移動します。  
  
## 構文  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
