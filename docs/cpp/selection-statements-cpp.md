# 選択ステートメント (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ の選択ステートメントである [if](../cpp/if-else-statement-cpp.md) および [switch](../cpp/switch-statement-cpp.md) は、コード セクションを条件付きで実行する手段を提供します。  
  
 [\_\_if\_exists](../cpp/if-exists-statement.md) および [\_\_if\_not\_exists](../cpp/if-not-exists-statement.md) ステートメントを使用すると、シンボルの有無に応じて条件付きでコードをインクルードできます。  
  
 各ステートメントの構文については、個々のトピックを参照してください。
