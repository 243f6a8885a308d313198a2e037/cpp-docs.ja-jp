# <a name="selection-statements-c"></a>選択ステートメント (C++)

 [_ _If_exists](../cpp/if-exists-statement.md)と[_ _if_not_exists](../cpp/if-not-exists-statement.md)ステートメントを使用すると、条件付きで、シンボルの有無によってコードが含まれます。  
  
 各ステートメントの構文については、個々のトピックを参照してください。
