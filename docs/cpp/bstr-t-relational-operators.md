# <a name="bstrt-relational-operators"></a>_bstr_t 関係演算子
**Microsoft 固有の仕様**  
  
 2 つの `_bstr_t` オブジェクトを比較します。  
  
## <a name="syntax"></a>構文  
  
```  
  
      bool operator!( ) const throw( );   
bool operator==(  
   const _bstr_t& str   
) const throw( );  
bool operator!=(  
   const _bstr_t& str   
) const throw( );  
bool operator<(  
   const _bstr_t& str   
) const throw( );  
bool operator>(  
   const _bstr_t& str   
) const throw( );  
bool operator<=(  
   const _bstr_t& str   
) const throw( );  
bool operator>=(  
   const _bstr_t& str   
) const throw( );  
```  
  
## <a name="remarks"></a>コメント  
 これらの演算子は 2 つの `_bstr_t` オブジェクトを辞書式に比較します。 演算子を返します**true**それ以外の場合を返す場合は、比較が保持、 **false**です。
