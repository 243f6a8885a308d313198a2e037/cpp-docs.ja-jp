# 式の型
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ の式は、複数のカテゴリに分けられます。  
  
-   [1 次式](../cpp/primary-expressions.md)。  これは、その他すべての式を構築する基本構成単位です。  
  
-   [後置式](../cpp/postfix-expressions.md)。  これは、演算子が続く 1 次式です \(配列の添字や後置インクリメント演算子など\)。  
  
-   [単項演算子で記述される式](../Topic/Expressions%20with%20Unary%20Operators.md)。  単項演算子は、式の中で 1 つのオペランドに対してのみ作用します。  
  
-   [二項演算子で記述される式](../cpp/expressions-with-binary-operators.md)。  二項演算子は、式の中で 2 つのオペランドに作用します。  
  
-   [条件演算子を含む式](../cpp/conditional-operator-q.md)。  条件演算子は、3 つのオペランドを受け取る、C\+\+ 言語で唯一の三項演算子です。  
  
-   [定数式](../cpp/cpp-constant-expressions.md)。  定数式は、定数データのみで構成されます。  
  
-   [明示的な型変換を含む式](http://msdn.microsoft.com/ja-jp/060ad6b4-9592-4f3e-8509-a20ac84a85ae)。  明示的な型変換、つまり "キャスト" を式で使用できます。  
  
-   [メンバーへのポインター演算子を含む式](../cpp/pointer-to-member-operators-dot-star-and-star.md)。  
  
-   [キャスト](../cpp/casting.md)。  式でタイプ セーフな "キャスト" を使用できます。  
  
-   [ランタイム型情報](../Topic/Run-Time%20Type%20Information.md)。  プログラムの実行中にオブジェクトの型を決定します。
