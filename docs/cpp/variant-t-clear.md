# _variant_t::Clear
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 固有の仕様 →**  
  
 カプセル化された **VARIANT** オブジェクトをクリアします。  
  
## 構文  
  
```  
  
void Clear( );  
  
```  
  
## 解説  
 カプセル化された **VARIANT** オブジェクトで **VariantClear** を呼び出します。
