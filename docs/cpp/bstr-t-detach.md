# _bstr_t::Detach
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 固有の仕様 →**  
  
 `_bstr_t` でラップされた `BSTR` を返し、`_bstr_t` から `BSTR` をデタッチします。  
  
## 構文  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## 戻り値  
 `_bstr_t` によってラップされる `BSTR`。  
  
## 使用例  
 **Detach** の使用例については「[\_bstr\_t::Assign](../cpp/bstr-t-assign.md)」を参照してください。
