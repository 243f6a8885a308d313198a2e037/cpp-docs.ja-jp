# <a name="bstrtdetach"></a>_bstr_t::Detach
**Microsoft 固有の仕様**  
  
 `BSTR` でラップされた `_bstr_t` を返し、`BSTR` から `_bstr_t` をデタッチします。  
  
## <a name="syntax"></a>構文  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## <a name="return-value"></a>戻り値  
 `BSTR` によってラップされる `_bstr_t`。  
  
## <a name="example"></a>例  
 参照してください[_bstr_t::assign](../cpp/bstr-t-assign.md)使用例については**デタッチ**です。
