# 配列型の間接参照
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*n* 次元の配列型に対して間接演算子 \(**\***\) を使用すると、*n*–1 次元の配列が生成されます。  *n* が 1 の場合、スカラー \(または配列要素\) が生成されます。
