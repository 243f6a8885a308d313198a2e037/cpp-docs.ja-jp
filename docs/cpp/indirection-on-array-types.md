# <a name="indirection-on-array-types"></a>配列型の間接参照
間接演算子の使用 (**\***) で、 *n*-次元の配列型の生成、 *n*-1 次元配列。 場合*n*は 1 です。 スカラー (または配列の要素) が返されます。
