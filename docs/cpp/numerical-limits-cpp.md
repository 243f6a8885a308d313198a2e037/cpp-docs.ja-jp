# <a name="numerical-limits-c"></a>数値制限 (C++)

2 つの標準のインクルード ファイル、 \<limits.h > と\<float.h >、数値の制限、または指定した型の変数が保持できる最小値と最大の値を定義します。 これらの最小値と最大値は ANSI C と同じデータ表現を使用する任意の C++ コンパイラへの移植性を保証します。\<Limits.h > を含めるファイルを定義、[整数型の数値の制限](../cpp/integer-limits.md)と\<float.h > 定義、[浮動小数点型の数値の制限](../cpp/floating-limits.md)します。
