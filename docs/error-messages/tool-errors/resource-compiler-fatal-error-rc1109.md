# <a name="resource-compiler-fatal-error-rc1109"></a>リソース コンパイラの致命的なエラー RC1109
リソース名を作成中にエラー  
  
 指定した .res ファイルを作成できませんでした。 読み取り専用のドライブに作成されていないことを確認します。 /V オプションを使用して、ファイルが作成されているかどうかを確認します。
