# <a name="resource-compiler-fatal-error-rw1009"></a>リソース コンパイラの致命的なエラー RW1009
リソース名を作成中にエラー  
  
 指定された .res ファイルを作成できませんでした。 読み取り専用のドライブに作成されていないことを確認します。 /V を使用すると、ファイルが作成されているかどうかを確認します。
