# 式エバリュエーター エラー CXX0045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

関数ではありません  
  
 関数名ではないシンボルに対して引数リストが指定されました。  
  
## 使用例  
  
```  
queue( alpha, beta )  
```  
  
 この式で `queue` が関数でない場合、エラーが発生します。  
  
 このエラーは CAN0045 と同じものです。
