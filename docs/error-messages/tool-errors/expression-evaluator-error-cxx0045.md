# <a name="expression-evaluator-error-cxx0045"></a>式エバリュエーター エラー CXX0045
関数ではありません。  
  
 引数リストは、関数の名前ではないプログラム内のシンボルに指定されました。  
  
## <a name="example"></a>例  
  
```  
queue( alpha, beta )  
```  
  
 ときに`queue`関数ではありません。  
  
 このエラーは、can0045 と同じものと同じです。
