# <a name="expression-evaluator-error-cxx0045"></a>式エバリュエーター エラー CXX0045

関数ではありません。

関数の名前のないプログラム内のシンボルの引数リストが指定されました。

## <a name="example"></a>例

```
queue( alpha, beta )
```

ときに`queue`関数ではありません。

このエラーは、can0045 と同じものと同じです。
