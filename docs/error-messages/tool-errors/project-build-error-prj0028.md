# プロジェクト ビルド エラー PRJ0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

一時ファイル 'file' はユーザーの ANSI コード ページに変換できない Unicode コンテンツを含んでいます。  
  
 リンカー オプション [\/MIDL \(MIDL コマンド ライン オプションの指定\)](../../build/reference/midl-specify-midl-command-line-options.md) に、システム コード ページで解決できない値が指定されています。  
  
 MIDL コマンド \(入力コード ページ\) の指定時には、システム コード ページと同じコード ページを使用する必要があります。
