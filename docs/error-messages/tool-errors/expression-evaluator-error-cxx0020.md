# <a name="expression-evaluator-error-cxx0020"></a>式エバリュエーター エラー CXX0020
この操作に対して無効なオペランドです。  
  
 演算子は、そのオペレーターの無効な型の式に適用されました。  
  
 たとえば、登録または浮動小数点式を持つ配列の添字のアドレスを取得する有効なは。  
  
 このエラーは、can0020 と同じものと同じです。
