# 式エバリュエーター エラー CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

関数入れ子の深さを越えています  
  
 式中の関数の入れ子の深さが制限を超えています。  式を書き直して、入れ子の深さを減らしてください。  
  
 このエラーは CAN0061 と同じものです。
