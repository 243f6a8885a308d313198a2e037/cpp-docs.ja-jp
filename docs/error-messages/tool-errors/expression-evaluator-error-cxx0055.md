# 式エバリュエーター エラー CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンテキスト演算子にあいまいなシンボルがあります  
  
 コンテキスト演算子 \(**{}**\) 内のシンボルがプログラム内の複数のシンボルを参照しました。  
  
 スコープ演算子 \(`::`\) を使うと、より正確に指定できます。  
  
 このエラーは CAN0055 と同じものです。
