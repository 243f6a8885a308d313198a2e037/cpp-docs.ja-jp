# プロジェクト ビルド エラー PRJ0026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

応答ファイル 'file' にユーザーの ANSI コード ページに変換できない Unicode のコンテンツが含まれています。  
  
 ***UNICODE contents of file***  
  
 応答ファイルで、ユーザーの現在の ANSI コード ページに正しく変換できない Unicode の内容が見つかりました。  
  
 このエラーを解決するには、ANSI を使用するように応答ファイルの内容を更新するか、コンピューターにコード ページをインストールしてシステムを既定の設定にしてください。
