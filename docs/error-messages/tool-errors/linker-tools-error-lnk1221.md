# リンカ ツール エラー LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

サブシステムは推論できません。定義されている必要があります。  
  
 情報不足のため、リンカーはターゲット サブシステムを推論できません。  
  
 このエラーを修正するには、[\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md) オプションを指定します。
