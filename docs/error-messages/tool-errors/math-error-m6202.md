# <a name="math-error-m6202"></a>数値演算エラー M6202

'function': エラー

指定された関数の引数は、この関数の値が不適切でした。 関数は、その引数が定義されていません。

このエラーが、`_matherr`関数、関数名、引数、およびエラーの種類を使用します。 書き直すことができます、`_matherr`特定の実行時の浮動小数点数値演算エラーの処理をカスタマイズする関数。
