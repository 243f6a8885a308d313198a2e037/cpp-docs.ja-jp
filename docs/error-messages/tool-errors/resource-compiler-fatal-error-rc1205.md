# リソース コンパイラの致命的なエラー RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

invalid code page  
  
 \[NLS 変換で使うコード ページの指定\] オプション \(\/c オプション\) の後ろにある、コード ページの指定が有効なものではありません。  有効なコード ページの詳細については、「ランタイム ライブラリ リファレンス」の「[コード ページ](../../c-runtime-library/code-pages.md)」を参照してください。
