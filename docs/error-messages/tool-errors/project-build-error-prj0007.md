# プロジェクト ビルド エラー PRJ0007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

出力ディレクトリ 'directory' を作成できませんでした。  
  
 Visual C\+\+ は出力ディレクトリを作成できませんでした。  以下の原因が考えられます。  
  
-   出力ディレクトリの作成先ディレクトリが読み取り専用である。  
  
-   コンピューターの空き容量が不足している。  
  
-   指定したディレクトリ パスが無効である。  
  
-   ユーザーのアクセス許可が適切でないため、ディレクトリを作成できない。
