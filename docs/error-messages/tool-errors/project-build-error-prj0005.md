# プロジェクト ビルド エラー PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ディレクトリ 'directory' に一時ファイルを作成できません。  
  
 一時的なファイルを作成するための呼び出しが失敗しました。  このエラーには以下の原因が考えられます。  
  
-   一時的なファイル名を使い果たした。  
  
-   一時ディレクトリが読み取り専用である。  
  
-   一時ディレクトリまたは TMP 環境変数が指定されていない。  
  
-   コンピューターの空き容量が不足している。
