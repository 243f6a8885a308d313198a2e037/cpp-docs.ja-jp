# <a name="project-build-error-prj0005"></a>プロジェクト ビルド エラー PRJ0005
ディレクトリ 'directory' に一時ファイルを作成できません。  
  
 一時ファイルを作成するための呼び出しが失敗しました。 失敗の原因は次のとおりです。  
  
-   一時ファイル名が不足しました。  
  
-   一時ディレクトリは読み取り専用です。  
  
-   一時ディレクトリまたは TMP 環境変数はありません。  
  
-   コンピューターが空きディスク領域が不足しています。
