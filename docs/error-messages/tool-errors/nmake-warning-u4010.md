# <a name="nmake-warning-u4010"></a>NMAKE の警告 U4010
'target': ビルドに失敗しました。/K 指定すると、操作を続行しています.  
  
 指定されたターゲットのコマンドのブロックのコマンドでは、0 以外の終了コードが返されます。 /K オプション、nmake (nmake の) セッションが終了した場合は、終了コード 1 を発行して、ビルドの関連付けられていない部分の処理を続行します。  
  
 NMAKE が警告を発行する場合は、指定されたターゲット自身が別の対象の依存[U4011](../../error-messages/tool-errors/nmake-warning-u4011.md)この警告の後にします。
