# リソース コンパイラの致命的なエラー RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

I\/O error reading file  
  
 ファイルを読み取ることができません。  ファイルを保存しているドライブが使用可能かどうか、またファイルが有効かどうかを確認してください。
