# リソース コンパイラの警告 RC4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

マクロ '識別子' に指定された実引数の数が多すぎます。  
  
 メッセージに示された識別子に指定された実パラメーターの数が、この識別子のマクロ定義で指定されている仮パラメーターの数より多くなっています。  
  
 余分な実パラメーターは渡されますが、マクロの展開時には無視されます。
