# リンカ ツール エラー LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

COFF シンボル テーブルが壊れているか、または無効です。未定義のスタティック シンボルまたはラベル シンボルです。  
  
 指定されたオブジェクトのセクションに関してコンパイラで生成された内容が破損しています。  
  
 このエラーは、コンパイラのエラーを示している場合があります。
