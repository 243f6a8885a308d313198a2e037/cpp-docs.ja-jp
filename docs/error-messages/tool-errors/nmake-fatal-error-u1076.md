# NMAKE の致命的なエラー U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

名前が長すぎます。  
  
 文字列が次のいずれかの制限を超えました。  
  
-   1024 文字 \(マクロ名の場合\)  
  
-   256 文字 \(ターゲット パス名の場合\)  
  
-   2048 文字 \(コマンドの場合\)
