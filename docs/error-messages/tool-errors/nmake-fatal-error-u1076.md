# <a name="nmake-fatal-error-u1076"></a>NMAKE の致命的なエラー U1076

名が長すぎます

文字列では、次の制限の 1 つを超えています。

- マクロ名を 1024 文字です。

- ターゲット パス名は、256 文字です。

- コマンドの 2048 文字です。
