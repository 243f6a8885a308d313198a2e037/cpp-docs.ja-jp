# リソース コンパイラ エラー RC2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expected numerical dialog constant  
  
 **DIALOG** ステートメントの *x*、*y*、*width*、*height* の各フィールドには整数値の指定が必須です。  これらの値が **DIALOG** キーワードの後ろに置かれているか、また負の値になっていないかどうかを確認してください。
