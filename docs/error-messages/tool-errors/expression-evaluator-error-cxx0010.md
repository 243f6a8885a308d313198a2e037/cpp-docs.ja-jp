# <a name="expression-evaluator-error-cxx0010"></a>式エバリュエーター エラー CXX0010
左角かっこがありません。  
  
 式には、不均衡の角かっこが含まれています。 左右の角かっこで式を再入力します。  
  
 このエラーは、can0010 と同じものと同じです。
