# リンカー ツールの警告 LNK4065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' は指定できません。無視されます。  
  
 指定された関数は、パッケージ化された関数としてコンパイルされていません。  [\/Gy](../../build/reference/gy-enable-function-level-linking.md) オプションを指定して再コンパイルしてください。
