# 式エバリュエーター エラー CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

文字定数の後に ' がありません  
  
 文字定数の最後に単一引用符 \('\) がありません。  式を書き直して、文字定数の最後に単一引用符を追加してください。  
  
 このエラーは CAN0009 と同じものです。
