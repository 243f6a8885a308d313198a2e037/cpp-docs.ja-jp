# <a name="nmake-fatal-error-u1007"></a>NMAKE の致命的なエラー U1007
二重引用符文字が名前では使用できません。  
  
 指定されたターゲットの名前またはファイル名に二重引用符が含まれている (**"**)。  
  
 二重引用符は、ファイル名を囲むことができますが、それに含まれることはできません。
