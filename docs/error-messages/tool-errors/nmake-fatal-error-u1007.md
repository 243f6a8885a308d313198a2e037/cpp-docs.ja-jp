# NMAKE の致命的なエラー U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

二重引用符は名前の中には使えません。  
  
 ターゲット名あるいはファイル名に、二重引用符 \(**"**\) が入っています。  
  
 二重引用符でファイル名を囲むことはできますが、ファイル名の中に使用することはできません。
