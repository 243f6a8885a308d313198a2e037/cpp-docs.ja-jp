# コマンド ライン エラー D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\/option' には複数のソースファイルを指定できません。  
  
 次のコンパイラ オプションでは、複数のソース ファイルは指定できません。  
  
-   アセンブリ リスティング ファイルの名前を指定するためのオプション \(\/Fa オプション\)  
  
-   オブジェクト ファイル名を変更するためのオプション \(\/Fo オプション\)  
  
-   ローカル変数を含まないソース ブラウザー情報ファイルを作成するためのオプション \(\/Fr オプション\)  
  
-   ローカル変数も含むソース ブラウザー情報ファイルを作成するためのオプション \(\/FR オプション\)
