# 式エバリュエーター エラー CXX0006
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左かっこがありません  
  
 式中の左右のかっこ \(\) の数が一致しません。  式を書き直して、左右のかっこの数を一致させてください。  
  
 このエラーは CAN0006 と同じものです。
