# 式エバリュエーター エラー CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**型は演算子と互換性がありません**  
  
 演算子のオペランドの型が不適切です。  
  
 たとえば、ポインターを被乗数として使用することはできません。  
  
 オペランドとして使用する値を演算子が許す型へキャストできる場合もあります。
