# リンカ ツール エラー LNK1189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

LIBTOOMANYMEMBERS:: オブジェクト number のライブラリ制限を超えています。  
  
 オブジェクトまたはライブラリ内のメンバーの制限である 65,535 を超えています。  
  
 この問題を解決するには、ライブラリ を 2 つ以上に分割し、各ライブラリのオブジェクトまたはメンバーの数を少なくします。
