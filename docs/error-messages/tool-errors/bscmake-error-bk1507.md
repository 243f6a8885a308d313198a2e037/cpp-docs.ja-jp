# BSCMAKE エラー BK1507
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

一時ファイル 'ファイル名' を開けません \[: 理由\]  
  
 一時的なファイルの 1 つを開けません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  環境変数 **TMP** に、有効でないドライブまたはフォルダーが設定されています。  
  
2.  ディスクの空き領域が不足しています。
