# <a name="linker-tools-warning-lnk4076"></a>リンカー ツールの警告 LNK4076
インクリメンタル ステータス ファイル 'filename';インクリメンタル リンク  
  
 リンクは、インクリメンタル ステータス (.ilk) ファイルに書き込むことはできません。 いずれか`filename`が壊れているか、インクリメンタル リンクのデータベースではありません。 ファイルを削除して再リンクしてください。
