# リソース コンパイラ エラー RC2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

END expected in dialog  
  
 **DIALOG** ステートメントの最後には、**END** キーワードが必要です。  これより前のステートメントで、引用符の対応関係が正しいかどうかを確認してください。
