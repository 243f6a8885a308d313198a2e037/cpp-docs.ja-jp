# <a name="expression-evaluator-error-cxx0001"></a>式エバリュエーター エラー CXX0001
実行中のユーザー関数を例外  
  
 実行対象のコードには、一般保護違反が発生しました。 このエラーは、can0001 と同じものと同じです。
