# 式エバリュエーター エラー CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

変数にはスタック フレームが必要です  
  
 変数がスタック フレームに現れないので、式エバリュエーターは変数を評価できません。  これは、変数がインライン関数の一部として宣言されている場合に起こります。
