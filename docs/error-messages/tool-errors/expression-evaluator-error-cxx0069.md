# <a name="expression-evaluator-error-cxx0069"></a>式エバリュエーター エラー CXX0069

変数には、スタック フレームが必要です。

スタック フレームが発生しないために、式エバリュエーターは、変数を評価できません。 これは、インライン関数の一部として宣言された変数によって発生する可能性があります。
