# <a name="expression-evaluator-error-cxx0051"></a>式エバリュエーター エラー CXX0051

ブレークポイントで指定された 1 つ以上のオーバー ロードされたシンボル

式エバリュエーターでは、ブレークポイントとして使用する 1 つ以上のオーバー ロードされた記号のうち特定できませんでした。

このエラーは、can0051 と同じものと同じです。
