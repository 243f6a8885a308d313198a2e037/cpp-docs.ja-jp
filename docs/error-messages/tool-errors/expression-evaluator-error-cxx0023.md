# 式エバリュエーター エラー CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

不正な基数です  
  
 C の式エバリュエーターが、指定された基数を認識しません。  使用できる基数は 10 進数、16 進数、8 進数だけです。  
  
 このエラーは CAN0023 と同じものです。
