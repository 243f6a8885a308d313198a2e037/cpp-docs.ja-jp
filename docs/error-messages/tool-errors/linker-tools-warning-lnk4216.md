# <a name="linker-tools-warning-lnk4216"></a>リンカー ツールの警告 LNK4216
エクスポートされたエントリ ポイントのエントリ  
  
 エントリ ポイントは、DLL からエクスポートされました。 DLL のエントリ ポイントは、エクスポートする必要はありません。
