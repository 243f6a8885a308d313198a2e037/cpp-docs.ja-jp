# <a name="resource-compiler-error-rc2152"></a>リソース コンパイラ エラー RC2152
無効な制御文字  
  
 **ACCELERATORS** ステートメントの制御文字は無効です。 有効な制御文字は、(のみ)、キャレットを次の&1; つの文字で構成されています (**^**)。
