# 式エバリュエーター エラー CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

レジスタは使用できません  
  
 使用できないレジスタが式中で指定されています。  このエラーは、プログラムを実行しているコンピューターに存在しないレジスタをアクセスしようとしたときに発生します。  
  
 このエラーは CAN0060 と同じものです。
