# <a name="expression-evaluator-error-cxx0060"></a>式エバリュエーター エラー CXX0060
レジスタは使用できません。  
  
 式は、使用できないレジスタを指定します。 このエラーを実行しているコンピューターに存在しないレジスタにアクセスしようとして発生することができます。  
  
 このエラーは、can0060 と同じものと同じです。
