# 式エバリュエーター エラー CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ブレークポイントのバインディング中に関数以外のシンボルが一致しました  
  
 関数ではないシンボルに対してブレークポイントを設定しています。  データ メンバーをブレークポイントとして指定すると、このエラーが発生する場合があります。  
  
 このエラーは CAN0053 と同じものです。
