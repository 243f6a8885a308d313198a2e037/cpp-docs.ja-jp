# <a name="expression-evaluator-error-cxx0053"></a>式エバリュエーター エラー CXX0053

以外のシンボルにブレークポイントのバインド中に一致します。

ブレークポイントとして使用されるシンボルは、関数ではないです。 ブレークポイントとしてデータ メンバーを指定すると、このエラーが発生することができます。

このエラーは、can0053 と同じものと同じです。
