# リソース コンパイラの警告 RW4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.EXE 処理オプション \(\/L \/M \/P \/T \/K \/E \/31、または \/30\)  
  
 .EXE 処理オプションが指定されていますが、処理対象の実行可能ファイルがありません。  .res ファイルに対しては、このオプションを使用しないでください。
