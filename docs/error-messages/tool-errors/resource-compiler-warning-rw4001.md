# <a name="resource-compiler-warning-rw4001"></a>リソース コンパイラの警告 RW4001
.EXE 処理オプション (/L/M/P/T/K/E/31 または/30)  
  
 処理オプション、exe ファイルを指定したが、処理する実行可能ファイルがありませんでした。 .Res ファイルを使用してこれらのオプションを使用しないでください。
