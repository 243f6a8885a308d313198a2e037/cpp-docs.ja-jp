# <a name="resource-compiler-error-rc2103"></a>リソース コンパイラ エラー RC2103

予期しないファイルの終わり文字列リテラル

文字列の末尾の前に、ファイルの終わりが見つかりました。 文字列は、終わりの二重引用符不足している可能性があります (**"**)。
