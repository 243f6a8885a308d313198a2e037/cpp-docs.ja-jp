# <a name="resource-compiler-error-rc2103"></a>リソース コンパイラ エラー RC2103
予期しない終了途中でファイルの文字列リテラル  
  
 文字列の終了前にファイルの終わりが見つかりました。 文字列は終わりの二重引用符がない可能性があります (**"**)。
