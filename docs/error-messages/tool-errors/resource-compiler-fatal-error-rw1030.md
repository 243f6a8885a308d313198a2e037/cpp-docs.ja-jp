# リソース コンパイラの致命的なエラー RW1030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Output Error  
  
 このエラーは、リソース コンパイラが対象のファイルを作成できなかった場合に発生します。  ディスク容量に十分余裕があるかどうか、およびボリュームへの書き込み許可があるかどうかを確認してください。
