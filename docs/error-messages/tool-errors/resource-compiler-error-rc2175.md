# リソース コンパイラ エラー RC2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

resource file filename is not in 3.00 format  
  
 このリソースには、Version 3.00 より前の形式が使われています。  このリソース ファイルは変換するか、Version 3.00 以降の形式を使って作り直す必要があります。
