# <a name="linker-tools-error-lnk1164"></a>リンカ ツール エラー LNK1164
セクション セクション配置 (数)/ALIGN 値より大きい  
  
 オブジェクト ファイルで指定されたセクションの配置のサイズで指定した値を超えています、 [配置/](../../build/reference/align-section-alignment.md)オプション。 **配置/**値は 2 のべき乗である必要があり、等しいか、またはオブジェクト ファイル内のセクションの配置を超えます。  
  
 小さなセクションの配置または増加でいずれかの再コンパイル、 **配置/**値。
