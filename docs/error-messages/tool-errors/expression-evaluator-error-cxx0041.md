# 式エバリュエーター エラー CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

クラスの要素は静的メンバーかメンバー関数である必要があります  
  
 クラスのインスタンスを指定せずに静的でないクラス メンバー \(あるいは構造体メンバー、共用体メンバー\) が使用されました。  
  
 インスタンスを指定せずに使用できるのは静的データ メンバーと静的メンバー関数だけです。  
  
 このエラーは CAN0041 と同じものです。
