# プロジェクト ビルド エラー PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

NULL デバイスがシステム内にありません。ビルドを実行できません。  
  
 このエラーは、システム リソースが少ないときに発生します。  一部のアプリケーションを終了するか、コンピューターを再起動してください。  
  
 コンピューターの NULL デバイスに対する適切な特権がないためにこのエラーが発生することもあります。
