# <a name="project-build-error-prj0015"></a>プロジェクト ビルド エラー PRJ0015

NULL デバイスは、システム内にありません。 ビルドの起動、できます。

このエラーは、システム リソースの不足によって発生することができます。 一部のアプリケーションを閉じるか、再起動します。

コンピューター上の NULL デバイスに十分な特権がいない場合にも発生することができます。
