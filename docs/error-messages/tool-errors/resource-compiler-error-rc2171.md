# <a name="resource-compiler-error-rc2171"></a>リソース コンパイラ エラー RC2171

DIB ヘッダー形式が不明です。

ビットマップのヘッダーが、 **BITMAPCOREHEADER**または**BITMAPINFOHEADER**構造体。
