# リソース コンパイラ エラー RC2171
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unknown DIB header format  
  
 このビットマップのヘッダーは、**BITMAPCOREHEADER** 構造体でも **BITMAPINFOHEADER** 構造体でもありません。
