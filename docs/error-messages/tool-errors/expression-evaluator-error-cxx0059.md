# <a name="expression-evaluator-error-cxx0059"></a>式エバリュエーター エラー CXX0059
左側のオペランドは関数名ではないクラスです。  
  
 関数呼び出しの左オペランドはクラス名、関数呼び出しを解決できませんでした。 式の中でメンバー関数の名前を省略すると、このエラーが発生することができます。  
  
 このエラーは、can0059 と同じものと同じです。
