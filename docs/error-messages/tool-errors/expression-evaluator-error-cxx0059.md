# 式エバリュエーター エラー CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左のオペランドは関数名でないクラスです  
  
 関数呼び出しの左辺として指定されているクラス名に対する関数呼び出しができません。  式の中のメンバー関数の名前を省略すると、このエラーが発生する場合があります。  
  
 このエラーは CAN0059 と同じものです。
