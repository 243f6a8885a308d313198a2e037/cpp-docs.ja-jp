# 式エバリュエーター エラー CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

オーバーロードされた演算子はありません  
  
 式の左辺として指定されているクラスのオーバーロード演算子が定義されていません。  
  
 このエラーは CAN0058 と同じものです。
