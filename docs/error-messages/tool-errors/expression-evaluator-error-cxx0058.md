# <a name="expression-evaluator-error-cxx0058"></a>式エバリュエーター エラー CXX0058
オーバー ロードされた演算子が見つかりません  
  
 クラス型では、式の左辺オペランドとして指定されましたが、クラスのオーバー ロードされた演算子が定義されていません。  
  
 このエラーは、can0058 と同じものと同じです。
