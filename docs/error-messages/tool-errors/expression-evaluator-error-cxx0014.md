# 式エバリュエーター エラー CXX0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

オペランドがありません  
  
 演算子に必要なオペランドが指定されていません。  式の構文を確認します。  
  
 このエラーは CAN0014 と同じものです。
