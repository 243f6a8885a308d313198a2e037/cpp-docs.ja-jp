# <a name="expression-evaluator-error-cxx0014"></a>式エバリュエーター エラー CXX0014
オペランドがありません。  
  
 必要なオペランドに演算子が指定されました。 式の構文を確認します。  
  
 このエラーは、can0014 と同じものと同じです。
