# 式エバリュエーター エラー CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

展開できない式です  
  
 式エバリュエーターの内部エラーが発生しました。  
  
 正しく評価されるような構造の、等価な式を作成してください。  
  
 このエラーは CAN0031 と同じものです。
