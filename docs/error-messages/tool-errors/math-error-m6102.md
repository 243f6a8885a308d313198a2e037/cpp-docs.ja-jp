# 数値演算エラー M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

denormal  
  
 演算によって、小さい浮動小数点数が生成されました。有効桁数が失われたので無効です。  通常、これらの浮動小数点値の例外はマスクされるため、トラップされて演算が行われます。  
  
 プログラムは終了コード 130 で終了します。
