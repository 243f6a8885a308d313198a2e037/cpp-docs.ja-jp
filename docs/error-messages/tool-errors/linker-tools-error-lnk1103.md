# リンカ ツール エラー LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

デバッグ情報が壊れています。モジュールを再コンパイルしてください。  
  
 このエラーは、有効なオブジェクト ファイルが作成される前にコンパイルが終了したために発生する場合があります。  
  
 オブジェクト ファイルを再コンパイルしてください。  再コンパイルしても問題が解決しない場合は、エラーの状況を書き留め、できる限り問題点を特定し、エラーが再現するようなテスト条件を見つけてください。  そのうえで、サポート サービスにご連絡ください。サポート サービスについては、パッケージに同梱されているマニュアルの製品サポート情報を参照してください。
