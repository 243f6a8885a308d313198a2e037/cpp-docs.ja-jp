# 式エバリュエーター エラー CXX0011
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

右角かっこがありません  
  
 式中の左右の角かっこ \[\] の数が一致しません。  式を書き直して、左右の角かっこの数を一致させてください。  
  
 このエラーは CAN0011 と同じものです。
