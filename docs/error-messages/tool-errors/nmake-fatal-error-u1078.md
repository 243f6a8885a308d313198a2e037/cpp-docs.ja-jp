'expression' で定数のオーバーフロー

指定された式には-2,147, 483,648 を 2,147, 483,647 の範囲を超えています。 定数が含まれています。 次の状況のいずれかで、定数が表示されました。

- プリプロセッサ ディレクティブで指定された式

- 使用する、指定したエラー レベル (**-**) コマンド修飾子
