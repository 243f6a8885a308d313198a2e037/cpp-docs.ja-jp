# <a name="resource-compiler-fatal-error-rw1004"></a>リソース コンパイラの致命的なエラー RW1004
予期しないファイルの終わり  
  
 このエラーは、ライン フィード文字およびテキスト ファイルの最後の行にキャリッジ リターン文字がないために発生することができます。
