# 式エバリュエーター エラー CXX0050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

:: の左側のタイプはクラス\/構造体\/共用体である必要があります  
  
 スコープ演算子 \(`::`\) の左辺にあるシンボルがクラス、構造体、共用体のいずれでもありません。  
  
 このエラーは CAN0050 と同じものです。
