# <a name="expression-evaluator-error-cxx0050"></a>式エバリュエーター エラー CXX0050
左側にある:: クラス/構造体/共用体にする必要があります  
  
 スコープ解決演算子の左側にある記号 (`::`) クラス、構造体、または共用体でした。  
  
 このエラーは、can0050 と同じものと同じです。
