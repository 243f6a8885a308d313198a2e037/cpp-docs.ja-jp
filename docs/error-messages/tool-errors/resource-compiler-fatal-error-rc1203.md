# <a name="resource-compiler-fatal-error-rc1203"></a>リソース コンパイラの致命的なエラー RC1203
指定された無効な&16; 進数の既定言語の ID。  
  
 既定の言語の指定 (/l) オプションが無効な&16; 進数の言語 ID に続けて 参照してください[言語および国/地域識別文字列](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)で、*ランタイム ライブラリ リファレンス*の有効な言語 Id の一覧です。
