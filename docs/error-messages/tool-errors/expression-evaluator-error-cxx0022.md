# <a name="expression-evaluator-error-cxx0022"></a>式エバリュエーター エラー CXX0022

(_m) 前に、の関数呼び出し

C の式エバリュエーターは、デバッガーが関数を入力する前に関数を評価できません **_main**します。 まで、プログラムが正しく初期化されていない **_main**が呼び出されました。

このエラーは、can0022 と同じものと同じです。
