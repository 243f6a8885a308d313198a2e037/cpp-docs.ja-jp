# <a name="expression-evaluator-error-cxx0022"></a>式エバリュエーター エラー CXX0022
_main の前に関数呼び出し  
  
 C の式エバリュエーターは、デバッガーが関数を入力する前に関数を評価できません**_main**です。 までプログラムが正しく初期化されていない**_main**が呼び出されています。  
  
 このエラーは、can0022 と同じものと同じです。
