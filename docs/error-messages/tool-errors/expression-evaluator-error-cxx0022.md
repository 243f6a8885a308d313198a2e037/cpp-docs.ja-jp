# <a name="expression-evaluator-error-cxx0022"></a>式エバリュエーター エラー CXX0022
_main の前に関数呼び出しがあります  
  
 デバッガーが関数を入力する前に、C の式エバリュエーターは、関数を評価できません**_main**します。 まで、プログラムが正しく初期化されていない**_main**が呼び出されています。  
  
 このエラーは、can0022 と同じものと同じです。
