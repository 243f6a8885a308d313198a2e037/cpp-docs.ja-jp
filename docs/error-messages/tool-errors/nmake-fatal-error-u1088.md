# NMAKE の致命的なエラー U1088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

推論規則に無効なセパレーター '::' があります。  
  
 推論規則の最後には、単独のコロン \(**:**\) を使用してください。
