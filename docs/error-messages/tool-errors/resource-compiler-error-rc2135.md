# リソース コンパイラ エラー RC2135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

file not found: filename  
  
 リソース コンパイラのコマンド ラインで指定されたファイルが見つかりません。  ファイルが別のフォルダーに移動されていないか、またファイル名やパスが正しく入力されているかどうかを確認してください。  
  
 ファイルの検索は、環境変数 **INCLUDE** や Visual C\+\+ の **INCLUDE** の設定を使って行われます。
