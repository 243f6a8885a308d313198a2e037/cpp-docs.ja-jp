# リンカ ツール エラー LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンピューターの種類 type が無効です。  
  
 オブジェクト ヘッダーに指定されたコンピューターの種類が無効です。  
  
 このエラーは、ファイルが破損していると発生する場合があります。  オブジェクトを再度ビルドしてください。
