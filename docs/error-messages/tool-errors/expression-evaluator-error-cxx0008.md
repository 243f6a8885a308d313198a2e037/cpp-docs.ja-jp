# 式エバリュエーター エラー CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**文字列の終わりに " がありません**  
  
 リテラル文字列の最後に二重引用符 \("\) がありません。  式を書き直して、リテラル文字列の最後に二重引用符を追加してください。  
  
 このエラーは CAN0008 と同じものです。
