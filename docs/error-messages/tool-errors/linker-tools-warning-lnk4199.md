# リンカー ツールの警告 LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:dllname は無視されます。dllname にインポートがありません。  
  
 `dllname` からのエクスポート関数は必要ないため、リンカーは `dllname` を無視します。
