# リソース コンパイラ エラー RC2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expected control class name  
  
 **DIALOG** ステートメント内の **CONTROL** ステートメントの `class` フィールドは BUTTON、COMBOBOX、EDIT、LISTBOX、SCROLLBAR、STATIC、ユーザー定義のいずれかである必要があります。  クラスのスペルが正しいかどうかを確認してください。
