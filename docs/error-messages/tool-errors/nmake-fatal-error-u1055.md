# <a name="nmake-fatal-error-u1055"></a>NMAKE の致命的なエラー U1055
環境の領域不足  
  
 オペレーティング システムは、環境変数の領域不足になりました。  
  
 環境の領域を増やすか、以下の環境変数を設定します。
