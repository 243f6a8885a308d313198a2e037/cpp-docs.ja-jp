# <a name="nmake-fatal-error-u1087"></a>NMAKE の致命的なエラー U1087
できません: と:: 同じターゲットの依存関係  
  
 両方を 1 つのコロンでターゲットを指定することはできません (**:**) とダブル コロン (`::`) 依存関係。  
  
 複数の記述ブロックのターゲットを指定するには、使用`::`各依存関係行にします。
