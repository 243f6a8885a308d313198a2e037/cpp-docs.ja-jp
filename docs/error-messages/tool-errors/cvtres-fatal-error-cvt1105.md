# CVTRES の致命的なエラー CVT1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイルの中を検索できません。  
  
 CVTRES でファイル内を移動できません。  
  
 このエラーは、ファイルの破損が原因で発生する場合があります。
