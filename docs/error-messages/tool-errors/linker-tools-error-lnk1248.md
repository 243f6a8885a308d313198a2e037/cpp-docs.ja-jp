# リンカ ツール エラー LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

イメージ サイズ \('size'\) が最大許容サイズ \(80000000\) を超えています。  
  
 リンカーは、出力ファイルのサイズがプログラム イメージの最大許容サイズを超えていると判断しました。  プログラムを複数の DLL にする必要があります。  実行可能プログラム \(PE\) の詳細については、記事「Peering Inside the PE "を参照します: Win32 の移植可能な実行可能ファイル ファイル形式の概要」 [MSDN ライブラリ](http://go.microsoft.com/fwlink/?linkid=556)。
