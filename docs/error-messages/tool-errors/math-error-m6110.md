# 数値演算エラー M6110
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

スタック オーバーフロー  
  
 浮動小数点の式によって、浮動小数点スタック オーバーフローが発生しました。  
  
 浮動小数点数スタック オーバーフローの例外は、8087\/287\/387 コプロセッサがサポートする 8 レベルのトラップに加え、さらに 7 レベルのトラップが可能です。  
  
 プログラムは終了コード 138 で終了します。
