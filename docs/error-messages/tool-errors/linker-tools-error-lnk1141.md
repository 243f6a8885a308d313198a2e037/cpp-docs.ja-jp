# リンカ ツール エラー LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

エクスポート ファイルのビルド中に障害が起こりました。  
  
 [LINK](../../build/reference/linker-command-line-syntax.md) はエクスポート ファイル \(.exp\) をビルドできませんでした。  
  
 このエラーの原因として、メモリ不足などのシステムの問題、およびオプション指定やモジュール定義ステートメントでの構文エラーが考えられます。  
  
 このエラーの前に出力されるエラーには、参考となる情報が含まれています。
