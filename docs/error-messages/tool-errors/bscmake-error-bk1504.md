# BSCMAKE エラー BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイル 'filename' 中の特定の位置に移動できません。\[: reason\]  
  
 ファイル内の特定位置への移動ができません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  ディスクの空き領域が不足しています。  
  
2.  ハードウェア エラーです。  
  
3.  ディスク容量の不足、または .sbr ファイルの作成中に割り込みが発生したことによって、ファイルが切り捨てられています。
