# <a name="project-build-error-prj0017"></a>プロジェクト ビルド エラー PRJ0017
現在の作業ディレクトリが正しくありません。  
  
 現在の作業ディレクトリへのパスは、_MAXPATH を超えています。 このエラーを解決するには、プロジェクトを作成しない、このような深いレベルでします。
