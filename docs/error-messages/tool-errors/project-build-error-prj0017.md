# プロジェクト ビルド エラー PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

現在の作業ディレクトリが無効です。  
  
 現在の作業ディレクトリのパスが \_MAXPATH より深くなっています。  このエラーを解決するには、現在より浅いレベルでプロジェクトを作成してください。
