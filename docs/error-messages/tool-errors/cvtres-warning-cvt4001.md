# CVTRES の警告 CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**コンピューターの種類が指定されていません。**   
 ***type***  
  
 CVTRES ではコンピューターの仕様に関する情報を検出できませんでした。  メッセージに示された種類のコンピューターであると想定されます。  表示されている種類が正しくない場合は、\/MACHINE オプションを指定し、CVTRES を再実行してください。
