# <a name="linker-tools-warning-lnk4205"></a>リンカー ツールの警告 LNK4205
'filename' には、参照するモジュールの現在のデバッグ情報がありません。オブジェクトをリンク デバッグ情報がありません。  
  
 .Pdb ファイルは、古い情報です。 リンカーでは、引き続きデバッグ情報なしのオブジェクトをリンクします。 オブジェクト ファイルを使用して、再コンパイルすることも、 [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md)オプション。
