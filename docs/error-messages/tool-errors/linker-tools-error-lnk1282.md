# リンカ ツール エラー LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/REBASE 'file' はできません ; 署名されています。  
  
 署名されたアセンブリのベース アドレスを、[editbin](../Topic/EDITBIN%20Reference.md) の \/REBASE オプションで変更しようとしました。  この処理を行うには、最初にベース アドレスを変更してから、アセンブリに署名してください。
