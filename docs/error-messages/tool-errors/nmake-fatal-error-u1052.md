# NMAKE の致命的なエラー U1052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイル 'filename' が見つかりません。  
  
 次のいずれかの方法で指定されたファイルが見つかりません。  
  
-   **\/F** オプション  
  
-   **\!INCLUDE** プリプロセス ディレクティブ  
  
-   応答ファイル指定のためのアット マーク指定子 \(**@**\)  
  
 このファイルが存在しているかどうか、またファイル名が間違っていないかどうかを確認してください。
