# プロジェクト ビルド エラー PRJ0036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Web 配布ツールの \[追加ファイル\] プロパティには無効な入力項目項目が含まれています。  
  
 \[Web デプロイメント\] プロパティ ページの \[追加ファイル\] プロパティで、マクロ評価の問題が原因と思われるエラーが見つかりました。  このエラーは、ファイル パスとして不正な文字または文字の組み合わせが含まれているためにパスの書式が正しくない場合にも発生します。  
  
 このエラーを解決するには、マクロを修正するか、有効なパスを指定してください。  パスには、プロジェクト ディレクトリからの絶対パスを指定する必要があります。  
  
 参照されるファイルのいずれかが存在しないことを意味する場合もあります。
