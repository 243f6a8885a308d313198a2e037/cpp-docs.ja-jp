# <a name="expression-evaluator-error-cxx0021"></a>式エバリュエーター エラー CXX0021
構造体または共用体がスカラーとして使用  
  
 構造体または共用体は、式で使用されましたが、要素が指定されていません。  
  
 構造体または共用体変数を操作するには、変数の名前可能性がありますフィールド修飾子を持たない、単独で表示されます。 構造体または共用体は式の中で使用する場合は、必要な特定の要素で修飾する必要があります。  
  
 式で使用するのには、値を持つ要素を指定します。  
  
 このエラーは、can0021 と同じものと同じです。
