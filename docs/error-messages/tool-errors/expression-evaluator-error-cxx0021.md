# <a name="expression-evaluator-error-cxx0021"></a>式エバリュエーター エラー CXX0021
構造体または共用体がスカラーとして使われています  
  
 構造体または共用体は、式では、使用されたが、要素が指定されていません。  
  
 構造体または共用体変数を操作するには、変数の名前がフィールド修飾子を持たない自体は、表示されます。 式では、構造体または共用体が使用する場合は、必要な特定の要素で修飾する必要があります。  
  
 値が、式で使用する要素を指定します。  
  
 このエラーは、can0021 と同じものと同じです。
