# リソース コンパイラ エラー RC2116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

expecting number for ID  
  
 **DIALOG** ステートメント内のコントロール ステートメントの `id` フィールドには、数字を指定する必要があります。  コントロール ID に対して、数値または `#define` ステートメントを指定しているかどうかを確認してください。
