# 式エバリュエーター エラー CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ユーザー メモリへのアクセスに失敗しました  
  
 式の中で、プログラムに割り当てられていないメモリを参照しようとしました。  
  
 このエラーは CAN0002 と同じものです。
