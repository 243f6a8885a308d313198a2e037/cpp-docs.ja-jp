# 式エバリュエーター エラー CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構文エラーです  
  
 式の構文が間違っています。  
  
 正しい構文に合うように式を修正してください。  
  
 このエラーは CAN0004 と同じものです。
