# <a name="linker-tools-error-lnk1136"></a>リンカ ツール エラー LNK1136
ファイルが無効か破損しています  
  
 入力ファイルが破損しているヘッダーか、サイズまたは異常小規模のゼロです。  
  
 ファイルが正しく指定されていない場合、追加の依存関係として、LNK1136 を発生もあります。
