# リンカ ツール エラー LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイルが無効であるか、または壊れています。  
  
 入力ファイルのヘッダーが破損しているか、またはサイズが 0 あるいはごく小さいファイルが指定されています。  
  
 LNK1136 は、ファイルが追加の依存関係として誤って指定された場合にも発生することがあります。
