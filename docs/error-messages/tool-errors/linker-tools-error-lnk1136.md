# <a name="linker-tools-error-lnk1136"></a>リンカ ツール エラー LNK1136

無効または壊れたファイル

入力ファイルは、ヘッダーが壊れているかが 0 のサイズまたはごく小さいです。

LNK1136 は、追加の依存関係として、ファイルが正しく指定されている場合にも発生する可能性があります。
