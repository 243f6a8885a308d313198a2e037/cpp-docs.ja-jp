# 式エバリュエーター エラー CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

呼び出し規約がサポートされていません  
  
 式中で使用されている関数呼び出しの中に、C の式エバリュエーターでは処理できない部分があります。  この関数呼び出しの形式は、ウォッチ ウィンドウでは使用できません。  
  
 このエラーは CAN0048 と同じものです。
