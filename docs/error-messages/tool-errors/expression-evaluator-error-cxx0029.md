# <a name="expression-evaluator-error-cxx0029"></a>式エバリュエーター エラー CXX0029
構造体ポインターではありません。  
  
 メンバー選択演算子 (**->**) 構造体へのポインターではない式に適用されました。  
  
 式全体が、正しく、かっこで囲まれた型、アドレス式を適切な構造体ポインター型をキャストすることを確認します。  
  
 このエラーは、can0029 と同じものと同じです。
