# <a name="expression-evaluator-error-cxx0029"></a>式エバリュエーター エラー CXX0029

構造体ポインターではありません。

メンバー選択演算子 (**->**) 構造体へのポインターでない式に適用されました。

式全体が正しく、かっこで囲まれたが型を適切な構造体のポインター型に、アドレス式をキャストすることを確認します。

このエラーは、can0029 と同じものと同じです。
