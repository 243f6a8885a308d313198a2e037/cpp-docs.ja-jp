# 式エバリュエーター エラー CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構造体ヘのポインターではありません  
  
 メンバー選択演算子 \(**\-\>**\) 構造体へのポインターではない式に適用されました。  
  
 式中のかっこが適切かどうか、またアドレス式に対する型キャストが適切な構造体ポインター型を生成しているかどうかを確認してください。  
  
 このエラーは CAN0029 と同じものです。
