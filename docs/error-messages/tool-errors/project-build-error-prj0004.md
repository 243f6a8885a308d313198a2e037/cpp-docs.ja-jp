# プロジェクト ビルド エラー PRJ0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'tool' ツールのコマンド ラインを生成できませんでした。  
  
 1 つ以上のプロパティの指定で、***tool*** の呼び出しの構文が無効です。  
  
-   指定したマクロの書式が間違っているか、不明なマクロを指定した可能性があります。  
  
-   コンピューターの空き容量が少なくなっている可能性があります。  
  
 [&#91;コマンド ライン&#93; プロパティ ページ](../../ide/command-line-property-pages.md)を参照し、ツールの設定を確認してください。
