# <a name="resource-compiler-error-rc2151"></a>リソース コンパイラ エラー RC2151
文字列定数を再利用することはできません。  
  
 2 回の同じ値を使用している、 **STRINGTABLE**ステートメントです。 10 進数と 16 進数の値の重複を混在させてされていないことを確認してください。  
  
 内の各 ID、 **STRINGTABLE**一意である必要があります。 効率が最大使用 16 の倍数を開始する連続した定数。
