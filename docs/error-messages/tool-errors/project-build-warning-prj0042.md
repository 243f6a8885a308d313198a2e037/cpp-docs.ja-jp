# プロジェクト ビルドの警告 PRJ0042
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイル 'file' のカスタム ビルド ステップに関する \[出力ファイル\] プロパティが設定されていません。カスタム ビルド ステップはスキップされます。  
  
 出力が指定されていないため、カスタム ビルド ステップが実行されませんでした。  
  
 このエラーを解決するには、以下のいずれかの操作を実行します。  
  
-   ビルドからカスタム ビルド ステップを除外する。  
  
-   出力を追加する。  
  
-   カスタム ビルド ステップのコマンドの内容を削除する。
