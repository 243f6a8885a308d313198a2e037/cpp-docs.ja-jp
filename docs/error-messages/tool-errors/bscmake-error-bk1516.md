# <a name="bscmake-error-bk1516"></a>BSCMAKE エラー BK1516
bscfile が壊れています。インクリメンタル ビルドができません。  
  
 .Bsc ファイルが破損している、可能性のあるビルド中にシステム エラーのためです。 .Bsc ファイルを削除し、すべての .sbr ファイルを再構築、.bsc ファイルを再構築します。
