# BSCMAKE エラー BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'bscfile'  不正 : インクリメンタル ビルドができません。  
  
 .bsc ファイルが破損しています。ビルド中に発生したシステム障害が原因である可能性があります。  この .bsc ファイルを削除し、すべての .sbr ファイルをビルドし直してから、再度 .bsc ファイルをビルドします。
