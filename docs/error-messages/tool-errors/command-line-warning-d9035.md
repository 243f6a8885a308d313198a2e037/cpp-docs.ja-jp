# コマンド ラインの警告 D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

オプション 'option' の使用は現在推奨されていません。今後のバージョンからは削除されます。  
  
 Visual C\+\+ コンパイラの将来のリリースで削除される予定のコンパイラ オプションを指定しました。  この警告は [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) の警告に *option*の提案された置換がある続きます。  
  
 指定したオプションは引き続き動作しますが、ビルド構成を今すぐ更新する必要があります。  その結果、コンパイラをアップグレードしたときにプロジェクトのビルドが続行する可能性が高くなります。
