# BSCMAKE エラー BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ファイル 'filename' から読み込めません。\[: reason\]  
  
 ファイルから読み込めません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  ファイルが破損しています。  
  
2.  ディスク容量の不足、または .sbr ファイルの作成中に割り込みが発生したことによって、ファイルが切り捨てられています。
