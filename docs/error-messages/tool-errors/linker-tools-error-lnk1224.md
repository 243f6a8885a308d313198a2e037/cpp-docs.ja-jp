# <a name="linker-tools-error-lnk1224"></a>リンカ ツール エラー LNK1224

無効なイメージのベース アドレス

イメージのベース アドレスが無効です指定しました。 ベース アドレスは 64 KB に配置する必要があります (最後の 4 つの 16 進数字は 0 である必要があります) と基本イメージは、32 ビット符号付きまたは符号なし値内に収まる必要があります。
