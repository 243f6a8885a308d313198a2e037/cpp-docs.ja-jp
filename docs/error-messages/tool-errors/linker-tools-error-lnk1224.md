# リンカ ツール エラー LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無効なイメージ ベース address です。  
  
 イメージに無効なベース アドレスが指定されています。  ベース アドレスは、64 KB に配置する必要があります \(16 進数の最後の 4 桁をゼロにします\)。また、イメージ ベースは 32 ビットの符号付きまたは符号なしの値である必要があります。
