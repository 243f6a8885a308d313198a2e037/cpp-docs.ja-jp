# リンカー ツールの警告 LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/SECTION オプションで指定されたセクション 'name' は存在しません。  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) または [EDITBIN](../Topic/EDITBIN%20Reference.md) は、入力ファイルにセクション `name` を見つけることができません。  
  
 ファイルに含まれるセクションを見るには、[\/HEADERS](../../build/reference/headers.md) オプションを指定して DUMPBIN を実行してください。
