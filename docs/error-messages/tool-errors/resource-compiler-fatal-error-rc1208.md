# リソース コンパイラの致命的なエラー RC1208
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

input file has .RES extension  
  
 リソース コンパイラの入力ファイルの拡張子に .RES が使われています。  リソース コンパイラのスクリプト ファイルの拡張子は .RC にする必要があります。
