# ガイド付き最適化のプロファイルの警告 PG1039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sweep を実行する待ち時間に失敗しました。  
  
 アプリケーションでは、[pgosweep](../../build/reference/pgosweep.md) の実行が許可されませんでした。可能な場合は、アプリケーションがアイドル状態のときに pgosweep を実行します。
