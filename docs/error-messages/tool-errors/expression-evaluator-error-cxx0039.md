# 式エバリュエーター エラー CXX0039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

シンボルがあいまいです  
  
 式中のシンボルが示す対象が確定できません。  継承関係の中でシンボルが 2 回以上使用されています。  
  
 式中でスコープ演算子 \(`::`\) を使用して、シンボルの意味を明確にしてください。  
  
 このエラーは CAN0039 と同じものです。
