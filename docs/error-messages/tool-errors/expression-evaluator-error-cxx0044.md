# 式エバリュエーター エラー CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_based \(void なポインター\) を使用する必要があります:\> オペレータ  
  
 `void` 型をベースにしたポインターを直接使用することはできません。  **:\>** の演算子を使用して、完全なポインターを形成する必要があります。  
  
 このエラーは CAN0044 と同じものです。
