# 式エバリュエーター エラー CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Error: 型情報がないか認識できません  
  
 .pch ファイルがリンクされていないか、\/Zi オプションによってコンパイルされないモジュールにある型への参照がコードに含まれています。  
  
 このエラーは CAN0067 と同じものです。
