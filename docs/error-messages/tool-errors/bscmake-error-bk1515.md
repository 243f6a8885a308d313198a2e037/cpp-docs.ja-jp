# BSCMAKE エラー BK1515
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'bscfile': バージョンの不適合 : インクリメンタル ビルドができません。  
  
 .bsc ファイルが、異なるバージョンの BSCMAKE で作成されています。  .bsc ファイルを作成し直してください。
