# プロジェクト ビルドの警告 PRJ0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

以下の環境変数が見つかりませんでした。  
  
 環境変数が定義されていません。  このエラーでは、定義されていない環境変数の一覧が表示されます。  
  
 ビルド処理のマクロについては、「[ビルドのコマンドとプロパティのマクロ](../../ide/common-macros-for-build-commands-and-properties.md)」を参照してください。
