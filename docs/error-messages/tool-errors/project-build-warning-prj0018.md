# <a name="project-build-warning-prj0018"></a>プロジェクト ビルドの警告 PRJ0018
次の環境変数が見つかりませんでした。  
  
 環境変数が定義されていません。 このエラーは、定義されていない環境変数を一覧表示します。  
  
 参照してください[カスタム ビルド コマンドのマクロ](../../ide/common-macros-for-build-commands-and-properties.md)については、ビルド処理でマクロです。
