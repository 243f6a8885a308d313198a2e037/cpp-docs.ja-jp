# <a name="resource-compiler-fatal-error-rc1116"></a>リソース コンパイラの致命的なエラー RC1116

RC のプリプロセッサ エラーの後に終了しています

リソース コンパイラは、他のエラーによって停止されました。 その他のエラーを修正して再コンパイルします。
