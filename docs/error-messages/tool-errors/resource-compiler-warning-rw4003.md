# リソース コンパイラの警告 RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

SHIFT or CONTROL used without VIRTKEY  
  
 アクセラレータ テーブル リソース内では、**Shift** キーまたは **Ctrl** キーには VIRTKEY を指定する必要があります。  **Shift** キーと **Ctrl** キーは VIRTKEY 型のアクセラレータの中でフラグ ビットとして示されるため、VIRTKEY なしで存在することはできません。
