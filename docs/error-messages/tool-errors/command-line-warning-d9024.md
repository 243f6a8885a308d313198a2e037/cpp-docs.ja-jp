# コマンド ラインの警告 D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ソースファイルの種類 'filename' は認識できませんでした。オブジェクト ファイルと仮定します。  
  
 メッセージに示されているファイルの拡張子を認識できません。  このファイルはオブジェクト ファイルであると解釈され、リンカーに渡されました。  
  
 認識される拡張子は、次のとおりです。  
  
-   .c \(C のソース ファイル\)  
  
-   .cxx \(C\+\+ のソース ファイル\)  
  
-   .cpp \(C\+\+ のソース ファイル\)  
  
-   .obj \(オブジェクト ファイル\)  
  
-   .lib \(ライブラリ ファイル\)  
  
-   .def \(モジュール定義ファイル\)  
  
-   .exp \(LINK \/LIB で生成されたリンカーのエクスポート ファイル\)
