# C ランタイム エラー R6027
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

lowio の初期化のためのディスク領域が不足しています  
  
 このエラーは、下位入出力のサポートの初期化に使用できる十分な空きメモリがない場合に発生します。
