# 式エバリュエーター エラー CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

行番号の位置にコードはありません  
  
 指定された行番号に対応するコードが生成されていません。  この部分に対してブレークポイントは指定できません。  
  
 このエラーは CAN0057 と同じものです。
