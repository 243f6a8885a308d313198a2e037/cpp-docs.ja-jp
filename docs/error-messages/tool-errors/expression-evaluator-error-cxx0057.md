# <a name="expression-evaluator-error-cxx0057"></a>式エバリュエーター エラー CXX0057
行番号にコードがありません。  
  
 指定した行番号のコードが生成されません。 ブレークポイントとして使用することはできません。  
  
 このエラーは、can0057 と同じものと同じです。
