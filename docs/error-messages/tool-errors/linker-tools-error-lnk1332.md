# リンカ ツール エラー LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

1 個の \<モジュール\> でインポートおよび別のモジュールで定義されている検出されたカウント ウィンドウのランタイム型  
  
 これが現在のターゲットを生成した場合、リンカーは \<\> それぞれ 1 個のモジュールでインポートおよび別のモジュールで定義されている`count` の Windows ランタイム型を検出しました。  
  
### このエラーを解決するには  
  
-   エラー メッセージの推奨事項に従ってビルドの LNK2039 エラーを一つずつ解決します。
