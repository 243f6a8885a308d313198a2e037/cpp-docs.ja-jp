# リソース コンパイラの致命的なエラー RW1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**I\/O error writing file**  
  
 ファイルに書き込むことができません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  ディスクの空き領域が足りません。  必要な空き領域は、最低でも、生成される実行可能ファイルのサイズの 2 倍です。  
  
2.  ボリュームが読み取り専用です。  
  
3.  セクターが不良です。  
  
4.  共有違反です。
