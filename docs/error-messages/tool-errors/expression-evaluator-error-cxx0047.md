# 式エバリュエーター エラー CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

引数リストが関数に適合しません  
  
 関数呼び出しで指定された実パラメーター リストが、同じ名前のどの関数の仮パラメーター リストとも一致しません。  
  
 オーバーロード関数はパラメーターが完全に一致する場合か、オブジェクトを生成しなくても一致する場合にのみ呼び出されます。  
  
 このエラーは CAN0047 と同じものです。
