# <a name="linker-tools-warning-lnk4200"></a>リンカー ツールの警告 LNK4200
オブジェクト ファイルで破損している行番号の情報無視されます。  
  
 オブジェクト ファイル内の行番号情報が壊れています。 再ビルドします。
