# <a name="math-error-m6203"></a>数値演算エラー M6203
'function': _OVERFLOW エラー  
  
 指定された関数の結果が大きすぎてを表現できます。  
  
 このエラーが、`_matherr`関数名、その引数、およびエラーの種類を持つ関数です。 書き直すことができます、`_matherr`特定の浮動小数点演算の実行時エラーの処理をカスタマイズする関数。
