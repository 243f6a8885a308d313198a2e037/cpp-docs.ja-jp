# 式エバリュエーター エラー CXX0043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

このポインターはメンバー関数の外で使われています  
  
 **this** ポインターは静的でないメンバー関数に対してのみ使用できます。  
  
 このエラーは CAN0043 と同じものです。
