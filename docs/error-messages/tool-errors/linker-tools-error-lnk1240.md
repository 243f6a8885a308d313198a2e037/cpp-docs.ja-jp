# リンカ ツール エラー LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

IDL コンテントのコンパイルに失敗しました。  
  
 リンカーは埋め込み IDL をコンパイルするために MIDL を起動しましたが、問題がありました。  MIDL で指定されたエラーをチェックしてください。
