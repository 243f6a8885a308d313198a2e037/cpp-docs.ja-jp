# <a name="linker-tools-error-lnk1240"></a>リンカ ツール エラー LNK1240
IDL コンテントのコンパイルに失敗しました  
  
 リンカー MIDL で埋め込み IDL のコンパイルを生成するが、問題が発生しました。 MIDL によって指定されたエラーを確認します。
