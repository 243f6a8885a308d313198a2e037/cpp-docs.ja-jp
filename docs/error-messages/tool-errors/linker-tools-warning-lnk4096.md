# リンカー ツールの警告 LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Windows 95、Windows 98 では \/BASE 値 "number" は無効です。イメージは動作しない可能性があります。  
  
 指定したベース アドレスは無効です。  Windows 95 と Windows 98 の実行可能ファイルのベース アドレスは 0x400000 以上であることが必要です。  ベース アドレスの詳細については、[\/BASE](../../build/reference/base-base-address.md) リンカー オプションのトピックを参照してください。
