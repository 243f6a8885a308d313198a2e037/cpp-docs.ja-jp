# <a name="linker-tools-error-lnk1287"></a>リンカ ツール エラー LNK1287
無効なマネージ エントリ ポイント関数  
  
 イメージのマネージ エントリ ポイントが正しくありません。 マネージ エントリ ポイント関数の戻り値の型は、必ず`void`または`int`です。 マネージ エントリ ポイント関数のパラメーターの型は、必ず`void`または`String []`です。
