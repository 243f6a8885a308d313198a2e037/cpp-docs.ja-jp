# CVTRES の致命的なエラー CVT1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' を読み取れません。  
  
 メッセージに示されているファイルを CVTRES で読み取ろうとしたときにエラーが発生しました。  
  
 このエラーは、ファイルの破損が原因で発生する場合があります。
