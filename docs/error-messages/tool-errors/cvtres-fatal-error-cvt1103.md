# <a name="cvtres-fatal-error-cvt1103"></a>CVTRES の致命的なエラー CVT1103
ファイル名を読み取ることができません。  
  
 CVTRES を指定されたファイルを読み取るしようとすると、回復不能なエラーが発生しました。  
  
 このエラーは、ファイルの破損によって発生することができます。
