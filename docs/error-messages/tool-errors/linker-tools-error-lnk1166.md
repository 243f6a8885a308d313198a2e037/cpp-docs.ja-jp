# リンカ ツール エラー LNK1166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

オフセット \=offset、va\=value にコードを調整できません。  
  
 LINK は、指定どおりにコードを埋め込むことができません。  
  
 プロセッサによっては、特定の命令でページ境界をまたぐことができない場合があります。  LINK は埋め込みを追加して問題の解決を試みます。  このエラーが出力された場合、LINK は問題を解決できませんでした。
