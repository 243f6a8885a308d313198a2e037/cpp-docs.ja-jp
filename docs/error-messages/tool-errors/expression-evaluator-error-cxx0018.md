# 式エバリュエーター エラー CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

レジスタ名が無効です  
  
 指定されたレジスタが存在しないか、あるいは表示できません。  
  
 ウォッチ ウィンドウでは次のレジスタが表示できます。  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 このエラーは CAN0018 と同じものです。
