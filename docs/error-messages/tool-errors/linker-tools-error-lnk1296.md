# <a name="linker-tools-error-lnk1296"></a>リンカ ツール エラー LNK1296
ファイル名を読み込めません。  
  
 指定された DLL は使用できませんでした。  
  
 DLL の可能な場所は、現在のディレクトリ、システム ディレクトリ、Windows ディレクトリおよび PATH 環境変数で指定したディレクトリです。
