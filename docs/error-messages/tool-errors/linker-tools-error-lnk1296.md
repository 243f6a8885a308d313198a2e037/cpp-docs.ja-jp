# リンカ ツール エラー LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename を読み込めません。  
  
 指定された DLL は使用できませんでした。  
  
 DLL は、現在のフォルダー、システム フォルダー、Windows フォルダー、または環境変数 **PATH** で指定されたフォルダーのいずれかにある可能性があります。
