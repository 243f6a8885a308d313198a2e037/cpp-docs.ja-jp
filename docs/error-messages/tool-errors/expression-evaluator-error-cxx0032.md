# 式エバリュエーター エラー CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0 で除算が行われました。  
  
 式中に 0 による除算が含まれています。  数字の 0、あるいは 0 として評価される式が除数として使用されています。  
  
 このエラーは CAN0032 と同じものです。
