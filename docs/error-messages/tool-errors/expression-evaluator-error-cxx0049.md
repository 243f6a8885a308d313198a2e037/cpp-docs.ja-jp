# 式エバリュエーター エラー CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

互換性維持のための OMF です \- プログラムと再リンクしてください  
  
 旧形式のオブジェクト モジュール形式 \(OMF: Object Module Format\) が使用されています。  
  
 現在のバージョンのリンカーを使用して、プログラムを再リンクしてください。  
  
 このエラーは CAN0049 と同じものです。
