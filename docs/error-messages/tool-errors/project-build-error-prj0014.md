# <a name="project-build-error-prj0014"></a>プロジェクト ビルド エラー PRJ0014

生成されたプロセスを制御するために使用するジョブ オブジェクトが失敗しました。  ビルドを続行できません。

開発環境でエラーが発生しました。

このエラーを解決するのには、閉じてから、プロジェクトの再読み込み。 必要に応じて、終了し、Visual Studio を再起動します。
