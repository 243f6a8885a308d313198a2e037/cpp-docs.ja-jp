# <a name="project-build-error-prj0014"></a>プロジェクト ビルド エラー PRJ0014
起動されたプロセスの制御に使用するジョブ オブジェクトが失敗しました。  ビルドを続行できません。  
  
 開発環境でエラーが発生しました。  
  
 このエラーを解決するには、閉じてプロジェクトの再読み込みします。 必要に応じて、終了して、Visual Studio を再起動します。
