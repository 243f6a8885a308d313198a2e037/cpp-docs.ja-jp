# <a name="bscmake-error-bk1508"></a>BSCMAKE エラー BK1508
一時ファイル 'filename' を削除できません [: 理由]  
  
 BSCMAKE は、いずれかの一時ファイルを削除できません。  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>次のような原因をチェックして問題を解決するには  
  
1.  別のプロセスが、ファイルを開きます。  
  
2.  ハードウェア エラーです。
