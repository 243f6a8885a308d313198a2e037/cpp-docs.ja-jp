# NMAKE の致命的なエラー U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ターゲット 'targetname' の依存ツリーに循環定義があります。  
  
 指定されたターゲットの依存関係のツリーに循環依存関係があります。  つまり、このターゲットがこのターゲットの依存ファイルのいずれかの依存ファイルになっています。  循環依存関係は無効です。
