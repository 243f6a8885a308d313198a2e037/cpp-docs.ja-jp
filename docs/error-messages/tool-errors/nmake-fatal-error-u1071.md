# <a name="nmake-fatal-error-u1071"></a>NMAKE の致命的なエラー U1071
ターゲット 'targetname' の依存関係ツリーに順番に表示します。  
  
 特定のターゲットの依存関係ツリーに循環する依存関係が存在します。 指定されたターゲットは、指定されたターゲットの依存オブジェクトのいずれかの依存関係です。 循環依存関係が無効です。
