# リソース コンパイラ エラー RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

invalid control type  
  
 **DIALOG** ステートメント内の各コントロール ステートメントは、3STATE、AUTO3、AUTOCHECK、AUTORADIO、BEDIT、CHECKBOX、COMBOBOX、CONTROL、CTEXT、DEFPUSHBUTTON、EDITTEXT、GROUPBOX、HEDIT、ICON、IEDIT、LISTBOX、LTEXT、PUSHBOX、PUSHBUTTON、RADIOBUTTON、RTEXT、SCROLLBAR、USERBUTTON のいずれかにする必要があります。  
  
 これらのコントロール ステートメントのスペルが間違っていないかどうかを確認してください。
