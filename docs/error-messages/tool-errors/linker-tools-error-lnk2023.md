# <a name="linker-tools-error-lnk2023"></a>リンカ ツール エラー LNK2023
dll またはエントリ ポイント\<dll またはエントリ ポイント >  
  
 リンカーは、msobj90.dll の正しくないバージョンを読み込んでいます。 Link.exe とパスに msobj90.dll に同じバージョンがあることを確認します。  
  
 Msobj90.dll の依存関係が存在していない可能性があります。 Msobj90.dll の依存関係の一覧を示します。  
  
-   Msvcr90.dll  
  
-   Kernel32.dll  
  
 期限切れ msobj90.dll のコピーが他のコンピューターを確認します。
