# <a name="project-build-error-prj0002"></a>プロジェクト ビルド エラー PRJ0002
'コマンド ライン' から返されたエラーの結果。  
  
 コマンド、***コマンドライン***、におけるユーザー入力から構成されている、**プロパティ ページ**ダイアログ ボックスで、エラー コードがない情報が返されますが、出力ウィンドウに表示されます。  
  
 このエラーの解決方法は、エラーを生成したツールによって異なります。 Midl、どの/o (出力のリダイレクト) が定義されている場合に発生したかのものが表示されます。  
  
 カスタム ビルド ステップまたはエラー状態に関する情報を含まない、ビルド イベントなど、バッチ ファイルでは、このエラーの理由があってもかまいません。
