# C ランタイム エラー R6026
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

stdio の初期化のためのディスク領域が不足しています  
  
 このエラーは、標準の入出力のサポートの初期化に十分な空きメモリを使用できない場合に発生します。
