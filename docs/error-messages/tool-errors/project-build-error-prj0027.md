# <a name="project-build-error-prj0027"></a>プロジェクト ビルド エラー PRJ0027
Unicode のログ メッセージが '内容' には、ユーザーの ANSI コード ページに変換できませんでしたコンテンツが含まれています。  
  
 通常この警告は、バッチまたは応答ファイルの作成エラー組み合わせてをのみ表示されます。  
  
 このエラーの解決方法では、ANSI を使用するかをコード ページをコンピューターにインストールし、システムの既定値として設定、ビルド ログの内容を更新します。
