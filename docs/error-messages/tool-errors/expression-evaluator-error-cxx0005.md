# <a name="expression-evaluator-error-cxx0005"></a>式エバリュエーター エラー CXX0005
演算子はサポートされていません  
  
 C 演算子がサポートされていませんが、式で指定されました。 サポートされている C 演算子を使用して同等の式を記述します。  
  
 このエラーは、can0005 と同じものと同じです。
