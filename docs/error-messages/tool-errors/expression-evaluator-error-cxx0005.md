# 式エバリュエーター エラー CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

サポートされていない演算子です  
  
 C ではサポートされていない演算子が式で使われています。  同じ内容の式を C でサポートされている演算子を使って記述してください。  
  
 このエラーは CAN0005 と同じものです。
