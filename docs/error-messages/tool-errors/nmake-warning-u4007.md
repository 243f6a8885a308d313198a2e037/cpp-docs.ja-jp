# <a name="nmake-warning-u4007"></a>NMAKE の警告 U4007
ファイル名 'filename' が長すぎます。8.3 の切り捨てください。  
  
 指定されたファイルのベース名が&8; 個を超える文字または、拡張機能には、3 つ以上の文字が含まれています。 NMAKE では、8 文字ベースで、3 文字の拡張機能に名前が切り捨てられます。  
  
 場合は、ファイル システムでは、長いファイル名がサポートされている、名前は二重引用符で囲みます (**"**)。
