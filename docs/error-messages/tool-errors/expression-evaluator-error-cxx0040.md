# <a name="expression-evaluator-error-cxx0040"></a>式エバリュエーター エラー CXX0040
関数には、暗黙的な変換が必要です。  
  
 C の式エバリュエーターは、コンス トラクター呼び出しを使用する暗黙的な変換をサポートしていません。  
  
 このエラーは、can0040 と同じものと同じです。
