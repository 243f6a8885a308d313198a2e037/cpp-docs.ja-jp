# <a name="project-build-error-prj0047"></a>プロジェクト ビルド エラー PRJ0047
中断されたプロセスを再開できませんでした。  ビルドが失敗しました。  
  
 開発環境でエラーが発生しました。  
  
 このエラーを解決するには、閉じてプロジェクトの再読み込みします。 必要に応じて、終了して、Visual Studio を再起動します。
