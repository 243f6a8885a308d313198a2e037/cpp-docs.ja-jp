# リソース コンパイラ エラー RC2164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unexpected value in RCDATA  
  
 **RCDATA** ステートメントの *raw\-data* 値には、いくつかの整数値または文字列をコンマで区切って指定する必要があります。  文字列を囲むための二重引用符やコンマを忘れていないかどうかを確認してください。
