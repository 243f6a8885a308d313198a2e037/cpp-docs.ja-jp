# <a name="resource-compiler-error-rc2164"></a>リソース コンパイラ エラー RC2164

RCDATA に予期しない値

*RCDATA* ステートメントの **生データ** 値は整数または文字列で、それぞれをコンマで区切る必要があります。 コンマや、文字列を囲む二重引用符が不足していないことを確認してください。
