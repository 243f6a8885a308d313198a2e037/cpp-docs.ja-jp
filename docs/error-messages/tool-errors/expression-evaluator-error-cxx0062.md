# <a name="expression-evaluator-error-cxx0062"></a>式エバリュエーター エラー CXX0062
コンストラクター呼び出しはサポートされていません  
  
 式は、コンス トラクターへの呼び出しを実行します。 式は、コンス トラクターを明示的に呼び出すか、コンス トラクターの呼び出しを必要とする変換を行うことはできません。  
  
 このエラーは、can0062 と同じものと同じです。
