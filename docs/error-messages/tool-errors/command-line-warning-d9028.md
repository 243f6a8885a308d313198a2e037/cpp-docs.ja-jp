# コマンド ラインの警告 D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

簡易リビルド時のエラ\-です、通常のビルドに戻します。  
  
 プロジェクトの .idb ファイルが破損しています。  このファイルを削除して、リビルドしてください。  
  
 簡易リビルドと .idb ファイルの詳細については、「[\/Gm \(簡易リビルドの有効化\)](../../build/reference/gm-enable-minimal-rebuild.md)」を参照してください。
