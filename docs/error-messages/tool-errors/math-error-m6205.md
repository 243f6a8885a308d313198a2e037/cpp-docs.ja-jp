# <a name="math-error-m6205"></a>数値演算エラー M6205
'function': _TLOSS エラー  
  
 全体の有効桁数 (精度) 機能の損失が発生しました。  
  
 このエラーは cos、sin のオペランドとして非常に大きい数値を付けることで発生した可能性がまたはオペランドが 0 ~ 2 の数値に限定する必要がありますので tan * pi します。
