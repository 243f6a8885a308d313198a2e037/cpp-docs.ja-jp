# <a name="nmake-fatal-error-u1097"></a>NMAKE の致命的なエラー U1097
ファイル名分割構文が依存している必要があります。  
  
 現在の依存関係には、明示的な依存または暗黙の依存ファイルのいずれかがありません。 割合を使用するファイル名分割構文 (`%`) 指定子の現在のターゲットの最初の依存コンポーネントを表します。
