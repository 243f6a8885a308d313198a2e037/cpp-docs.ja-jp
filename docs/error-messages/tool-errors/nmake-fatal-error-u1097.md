# <a name="nmake-fatal-error-u1097"></a>NMAKE の致命的なエラー U1097
ファイル名の構成要素には依存ファイルが必要です。  
  
 明示的な依存ファイルまたは暗黙の依存ファイルのいずれか、現在の依存関係はありません。 割合を使用するファイル名分割構文 (`%`) 指定子の現在のターゲットの最初の依存コンポーネントを表します。
