# <a name="expression-evaluator-error-cxx0016"></a>式エバリュエーター エラー CXX0016
定数が大きすぎます。  
  
 C の式エバリュエーターには、符号なし整数の定数 4,294,967,295 (0FFFFFFFF 16 進数) より大きいまたは絶対値が最大値よりも大きい浮動小数点定数を受け入れることはできません。  
  
 このエラーは、can0016 と同じものと同じです。
