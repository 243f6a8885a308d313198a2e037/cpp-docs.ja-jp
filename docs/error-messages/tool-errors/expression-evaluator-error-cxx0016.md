# 式エバリュエーター エラー CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

定数が大きすぎます。  
  
 C の式エバリュエーターが扱える符号なし整数定数の最大値は 4,294,967,295 \(0x0FFFFFFFF\) です。また浮動小数点定数の最大値は、およそ 1.8E\+308 です。  
  
 このエラーは CAN0016 と同じものです。
