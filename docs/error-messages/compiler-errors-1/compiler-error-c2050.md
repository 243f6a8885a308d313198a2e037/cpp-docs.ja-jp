# <a name="compiler-error-c2050"></a>コンパイラ エラー C2050
switch 式のない整数  
  
 `switch`式は整数でない値に評価します。 エラーを解決するには、switch ステートメントで整数値だけを使用します。  
  
 次の例では、C2050 が生成されます。  
  
```  
// C2050.cpp  
int main() {  
   int a = 1;  
   switch ("a") {   // C2050  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
   }  
}  
```  
  
 考えられる解決方法:  
  
```  
// C2050b.cpp  
int main() {  
   int a = 1;  
   switch (a) {  
      case 1:  
         a = 0;  
      default:  
         a = 2;  
   }  
}  
```
