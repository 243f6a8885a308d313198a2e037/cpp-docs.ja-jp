# <a name="fatal-error-c1057"></a>致命的なエラー C1057

マクロ展開中に予期せぬ EOF を検出しました。

コンパイラでは、マクロの呼び出しにかっこが不足している可能性があります、マクロ呼び出しの引数を収集中にソース ファイルの末尾に達しました。
