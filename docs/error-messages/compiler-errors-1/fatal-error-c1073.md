# 致命的なエラー C1073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

インクリメンタル コンパイルで内部エラーが発生しました \(コンパイラ ファイル 'filename', 行番号 number\)。  
  
 インクリメンタル コンパイルを使用せずにファイルを再コンパイルしてください。
