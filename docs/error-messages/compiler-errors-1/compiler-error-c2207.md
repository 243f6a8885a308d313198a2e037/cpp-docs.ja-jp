# コンパイラ エラー C2207
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member': クラス テンプレートのメンバーは関数型を取得できません  
  
 クラス テンプレートの `member` は、非静的データ メンバーとして以前に解析されました。  メンバー関数として再定義することはできません。
