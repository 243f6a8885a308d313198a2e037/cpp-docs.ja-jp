# コンパイラ エラー C2168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 組み込み関数の実引数が少なすぎます。  
  
 `intrinsic` 関数への参照のパラメーターが少なすぎます。
