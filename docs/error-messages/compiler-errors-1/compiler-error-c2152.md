# <a name="compiler-error-c2152"></a>コンパイラ エラー C2152

'identifier': 異なる属性を持つ関数へのポインター

1 つの呼び出し規約を持つ関数へのポインター (`__cdecl`、 `__stdcall`、または`__fastcall`) が別の呼び出し規約を持つ関数へのポインターに割り当てられます。
