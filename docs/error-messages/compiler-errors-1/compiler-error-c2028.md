# <a name="compiler-error-c2028"></a>コンパイラ エラー C2028
構造体や共用体のメンバーは、構造体か共用体の中になければなりません。  
  
 構造体または共用体のメンバーは、構造体または共用体の内側で宣言する必要があります。
