# コンパイラ エラー C2028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構造体や共用体のメンバーは、構造体か共用体の中になければなりません。  
  
 構造体または共用体のメンバーは、構造体または共用体の内側で宣言する必要があります。
