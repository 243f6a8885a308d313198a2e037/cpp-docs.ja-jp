# <a name="fatal-error-c1038"></a>致命的なエラー C1038
コンパイラの制限: function: 制御フローが複雑すぎます。関数を単純化してください  
  
 関数の制御フローの数が、コンパイラで処理できる量を超えています。 制御フローを簡単にするか、関数を小さな関数に分割してください。
