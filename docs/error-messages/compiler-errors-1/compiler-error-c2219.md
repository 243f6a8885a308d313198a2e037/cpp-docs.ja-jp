# コンパイラ エラー C2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構文エラー : 型修飾子は '\*' の後にのみ指定できます。  
  
 型修飾子 \(`const` または `volatile`\) が指定された場所が正しくありません。
