# 致命的なエラー C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

CryptoAPI への呼び出しに失敗しました  
  
 コンパイラが CryptoAPI への呼び出しを行いましたが、その呼び出しが失敗しました。  Visual Studio を再インストールしてください。また、場合によってはオペレーティング システムを再インストールしてください。
