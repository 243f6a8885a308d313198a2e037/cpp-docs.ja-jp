# <a name="compiler-error-c2318"></a>コンパイラ エラー C2318
この catch ハンドラーと関連付けられた try ブロックはありません。  
  
 `catch` ハンドラーが定義されていますが、その前に `try` ブロックがありません。  
  
 次の例では C2318 が生成されます。  
  
```  
// C2318.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   // no try block  
   catch( int ) {}   // C2318  
}  
```  
  
 考えられる解決策:  
  
```  
// C2318b.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try{}  
   catch( int ) {}  
}  
```
