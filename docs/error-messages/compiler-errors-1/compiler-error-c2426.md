# <a name="compiler-error-c2426"></a>コンパイラ エラー C2426
'トークン': 'コンテキスト' 内の無効な演算子  
  
 トークンは、このコンテキストでは演算子として使用できません。 インデックス演算子では、たとえば、入れ子にできません。
