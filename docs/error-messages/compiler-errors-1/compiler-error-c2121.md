# コンパイラ エラー C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#' : 無効な文字です : マクロ展開が解決できません。  
  
 無効なシャープ記号 \(\#\) は、文字列化演算子 \(\#\) の代わりにトークン連結演算子 \(\#\#\) を使用する、正しくないマクロによって挿入されたと思われます。
