# <a name="compiler-error-c2469"></a>コンパイラ エラー C2469
'operator': 'type' オブジェクトを割り当てることができません  
  
 演算子に無効な型が渡されました。  
  
 次の例では C2469 が生成されます。  
  
```  
// C2469.cpp  
int main() {  
   int *i = new void;   // C2469  
   int *i = new int;   // OK  
}  
```
