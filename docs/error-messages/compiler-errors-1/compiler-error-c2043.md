# <a name="compiler-error-c2043"></a>コンパイラ エラー C2043
'break' が正しくありません。  
  
 A [break](../../cpp/break-statement-cpp.md)は内でのみ有効な`do`を`for`を`while`をまたは`switch`ステートメント。
