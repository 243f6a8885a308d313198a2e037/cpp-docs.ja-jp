# 致命的なエラー C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プログラム データベース ファイル 'pdbfile' は旧形式です。このファイルを削除して再コンパイルしてください。  
  
 バージョン番号が古いプログラム データベース ファイルは更新できません。  ファイルを削除し、**\/Zi** または **\/ZI** を指定してプログラムを再コンパイルしてください。  詳細については、「[\/Z7、\/Zi、\/ZI \(デバッグ情報の形式\)](../Topic/-Z7,%20-Zi,%20-ZI%20\(Debug%20Information%20Format\).md)」を参照してください。
