# <a name="fatal-error-c1051"></a>致命的なエラー C1051
プログラム データベース ファイルは、'pdbfile' は旧形式、それを削除し、再コンパイル  
  
 コンパイラは、以前のバージョン番号のあるプログラム データベース ファイルを更新できません。 ファイルを削除し、使用してプログラムを再コンパイル**/Zi**または**/ZI**です。 詳細については、次を参照してください[/Z7、/Zi、/ZI (デバッグ情報の形式)。](../../build/reference/z7-zi-zi-debug-information-format.md)
