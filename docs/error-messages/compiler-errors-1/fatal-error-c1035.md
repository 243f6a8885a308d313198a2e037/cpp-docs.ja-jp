# 致命的なエラー C1035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

式が複雑すぎます。式を単純化してください。  
  
 コンパイラは、複雑な式のコードを生成できませんでした。 式をより簡単な式に分割し、再コンパイルします。
