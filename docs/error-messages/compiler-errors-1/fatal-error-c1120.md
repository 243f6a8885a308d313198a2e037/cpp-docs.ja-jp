# 致命的なエラー C1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' の GetProcAddress の呼び出しに失敗しました。  
  
 このエラーは、Visual C\+\+ を再インストールする必要があることを示します。
