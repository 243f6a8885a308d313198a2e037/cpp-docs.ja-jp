# コンパイラ エラー C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ブラウザー ファイル filename を削除できません。  
  
 ブラウザー ファイルを削除できません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  ファイルが読み取り専用です。  
  
2.  別のプロセスでファイルが使用されています。
