# <a name="compiler-error-c2499"></a>コンパイラ エラー C2499
'class': クラスは基底クラスをすることはできません  
  
 基底クラスとして定義するクラスを指定しようとしました。  
  
 次の例では、C2499 が生成されます。  
  
```  
// C2499.cpp  
// compile with: /c  
class CMyClass : public CMyClass {};   // C2499  
class CMyClass{};   // OK  
```
