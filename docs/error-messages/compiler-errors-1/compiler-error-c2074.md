# <a name="compiler-error-c2074"></a>コンパイラ エラー C2074
'identifier' : 'class-key' の初期化には中かっこ ({}) が必要です  
  
 指定したクラス、構造体、共用体の初期化子の周囲に中かっこがありませんでした。
