# <a name="compiler-error-c2212"></a>コンパイラ エラー C2212
'identifier': _ _based 指定は関数へのポインターの使用  
  
 関数へのポインターを宣言できません。`__based`です。 コード ベースのデータを必要がある場合、`__declspec`キーワードまたは`data_seg`プラグマ。
