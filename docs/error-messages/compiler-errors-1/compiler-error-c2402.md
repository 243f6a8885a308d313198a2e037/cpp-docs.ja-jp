# コンパイラ エラー C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': レジスタは 'context' のインデックス レジスタでなければなりません  
  
 間接メモリ オペランドで使うレジスタは、このコンテキストではインデックス レジスタである必要があります。
