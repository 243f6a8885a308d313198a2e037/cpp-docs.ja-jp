# コンパイラ エラー C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?': 左のオペランドが無効です。数値またはポインター型でなければなりません。  
  
 `?` の左側の式は 0 と比較できません。  0 と比較できるように、数値式かポインター式を使ってください。
