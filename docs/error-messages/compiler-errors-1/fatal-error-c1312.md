# <a name="fatal-error-c1312"></a>致命的なエラー C1312
関数内に条件分岐が多すぎます。 ソース コードを簡略化するか、リファクターを使用してください。  
  
 コードが複雑すぎるため、コンパイラが処理中にスタック メモリ不足を起こします。  コードを簡略化します。
