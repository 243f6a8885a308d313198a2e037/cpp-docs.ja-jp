# <a name="compiler-error-c2151"></a>コンパイラ エラー C2151

1 つ以上の言語 属性

関数が 1 つ以上のキーワード ( `__cdecl`、 `__stdcall`、または`__fastcall`) 呼び出し規約を指定します。
