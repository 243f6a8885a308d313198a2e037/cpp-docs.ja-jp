# コンパイラ エラー C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

複数の言語属性が存在します。  
  
 呼び出し規約を指定する複数のキーワード \(`__cdecl`、`__stdcall`、または `__fastcall`\) が関数に指定されています。
