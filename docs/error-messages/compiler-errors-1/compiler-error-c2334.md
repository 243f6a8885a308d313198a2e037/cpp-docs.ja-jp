# <a name="compiler-error-c2334"></a>コンパイラ エラー C2334
上記の予期しないトークン ': または {'; 関数の本体をスキップしています  
  
 次の例では、C2334 を生成します。 このエラーは、エラー C2059 の後に発生します。  
  
```  
// C2334.cpp  
// compile with: /c  
// C2059 expected  
struct s1 {  
   s1   {}   // C2334  
   s1() {}   // OK  
};  
```
