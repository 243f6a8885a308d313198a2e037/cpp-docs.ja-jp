# 致命的なエラー C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ブロックが大きすぎて、コンパイルできません。  
  
 ブロックが大きすぎてコンパイルできません。  
  
### 次のような原因をチェックして問題を解決するには  
  
1.  中かっこ \({}\) が一致していません。  
  
2.  関数やクラスが大きすぎます。
