# <a name="fatal-error-c1109"></a>致命的なエラー C1109
DLL ファイル 'dll' に 'entry point' が見つかりません  
  
 コンパイラによって要求された [遅延読み込み DLL](../../build/reference/linker-support-for-delay-loaded-dlls.md) のエントリ ポイントが見つかりませんでした。
