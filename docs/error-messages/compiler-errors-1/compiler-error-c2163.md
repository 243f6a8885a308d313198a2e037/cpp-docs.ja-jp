# <a name="compiler-error-c2163"></a>コンパイラ エラー C2163

'function': 組み込み関数として使用できません。

`intrinsic`または`function`プラグマには、組み込み形式では使用できない関数が一覧表示されます。 たとえば、/clr プログラミングを使用するプログラムをコンパイルするときに特定の組み込み関数は使用できません。
