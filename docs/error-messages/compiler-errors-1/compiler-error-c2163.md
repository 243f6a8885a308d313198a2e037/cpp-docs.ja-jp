# コンパイラ エラー C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 組み込み関数として使用できません。  
  
 `intrinsic` プラグマまたは `function` プラグマの関数リストに、組み込み形式で使用できない関数が含まれています。  たとえば、\/clr プログラミングを使用するプログラムのコンパイルに利用できない組み込み形式があります。
