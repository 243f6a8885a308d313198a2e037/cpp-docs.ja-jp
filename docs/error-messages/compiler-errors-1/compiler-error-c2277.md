# <a name="compiler-error-c2277"></a>コンパイラ エラー C2277
'identifier': このメンバー関数のアドレスは取得できません  
  
 メンバー関数のアドレスを取得できません。  
  
 次の例では、C2277 が生成されます。  
  
```  
// C2277.cpp  
class A {  
public:  
   A();  
};  
(*pctor)() = &A::A;   // C2277   
```
