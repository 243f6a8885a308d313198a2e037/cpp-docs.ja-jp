# <a name="compiler-error-c2010"></a>コンパイラ エラー C2010
'character': マクロの仮パラメーター リストに予期しません。  
  
 マクロ定義の仮パラメーター リストの中で、文字が正しくない方法で使用されています。 エラーを解決するのには、文字を削除します。  
  
 次の例では、C2010 が生成されます。  
  
```  
// C2010.cpp  
// compile with: /c  
#define mymacro(a|) (2*a)   // C2010  
#define mymacro(a) (2*a)   // OK  
```
