# <a name="fatal-error-c1046"></a>致命的なエラー C1046
コンパイラの制限: 構造体のネストが深すぎます  
  
 構造体、共用体、またはクラスは、15 レベルの入れ子の制限を超えました。 入れ子のレベルを削減するように定義を書き直してください。 構造体、共用体、またはクラスを使用して、2 つ以上の部分に分割`typedef`を 1 つ以上の入れ子になった構造体を定義します。
