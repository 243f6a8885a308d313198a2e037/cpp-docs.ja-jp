# <a name="compiler-error-c2465"></a>コンパイラ エラー C2465
名前のない構造体、共用体、または、列挙型がかっこの式内で定義されました。  
  
 匿名構造体、共用体、または列挙型は、かっこで囲まれた式の内部で定義されます。 この定義は関数スコープにおいて意味がないため、これは C++ では無効です。
