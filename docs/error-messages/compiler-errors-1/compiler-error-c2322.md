# コンパイラ エラー C2322
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : dllimport 'dllimport' のアドレスは静的ではありません  
  
 `dllimport` で宣言された関数のアドレスとして、静的でない値が指定されます。
