# 致命的なエラー C1108
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

次の DLL ファイルが見つかりません: 'dll'  
  
 指定された DLL \(***dll***\) がパスに見つかりませんでした。  
  
 このエラーを解決するには、Visual C\+\+ を再インストールするか、インストール CD からコンピューターに適切な .dll ファイルをコピーします。
