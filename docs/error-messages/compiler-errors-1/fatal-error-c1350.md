# 致命的なエラー C1350
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

DLL 'dll' の読み込み時のエラーです: DLL が見つかりませんでした。  
  
 操作に必要な DLL が見つかりませんでした。  
  
 このエラーは、インストールに問題があることを示します。製品の CD から DLL を再インストールする必要があります。
