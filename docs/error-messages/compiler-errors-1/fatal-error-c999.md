# <a name="fatal-error-c999"></a>致命的なエラー C999
不明なメッセージ、Visual C ヘルプ メニューのサポート情報コマンドを選択してください詳細については、テクニカル サポート ヘルプ ファイルを開く  
  
 通常、このエラーは、異なるバージョンのコンパイラのファイルが混在しているか、コンパイラのインストールが破損していることを示します。 使用して、**プログラムと機能**修復または製品を再インストールするには、コントロール パネルのアプレットします。
