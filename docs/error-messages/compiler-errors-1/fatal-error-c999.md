# <a name="fatal-error-c999"></a>致命的なエラー C999

Visual C ヘルプ メニューに、サポート情報コマンドを選択して不明なメッセージくださいまたは、詳細については、サポート情報ヘルプ ファイルを開く

通常、このエラーは、異なるバージョンのコンパイラのファイルが混在しているか、コンパイラのインストールが破損していることを示します。 使用して、**プログラムと機能**修復または製品を再インストールするには、コントロール パネルのアプレットです。
