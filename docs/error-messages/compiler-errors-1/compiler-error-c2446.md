# コンパイラ エラー C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'演算子' : 'type1' 型から 'type2' 型への変換ができません。  
  
 `type1` を `type2` に変換できません。  C\/C\+\+ 言語上意味をなさない、またはその意味を失わせるような変換は定義されていません。
