# 致命的なエラー C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

API の呼び出し 'function name' は 'location of call' に失敗しました : 'text from run\-time'  
  
 .NET 関数の呼び出しが失敗しました。  COM ランタイムによって ***text from run\-time*** が提供される場合とされない場合があります。  
  
 システム エラー メッセージの詳細については、winerror.h システム ファイルおよび「[FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351)」を参照してください。
