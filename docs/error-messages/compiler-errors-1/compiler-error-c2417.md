# コンパイラ エラー C2417
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'context' 内で使われた除算演算子 '\/' の 2 番目の引数が 0 です  
  
 このコンテキストでは、除算演算子の右側のパラメーターが 0 です。
