# 致命的なエラー C1904
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プロバイダーの動作が正しくありません: 'file'  
  
 このエラーは、属性プロバイダーのエラーを示します。  
  
 エラーの状況を書きとめ、問題の分離を試行し、再現可能なテストケースを作成したうえで、[ご意見](../Topic/Talk%20to%20Us.md) にお問い合わせください。
