# <a name="compiler-error-c2175"></a>コンパイラ エラー C2175
'ロケール': 無効なロケール  
  
 指定されたロケールが正しくありません。 参照してください[言語および国/地域識別文字列](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)で、*ランタイム ライブラリ リファレンス*のロケールをサポートします。
