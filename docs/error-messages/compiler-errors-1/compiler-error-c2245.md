# <a name="compiler-error-c2245"></a>コンパイラ エラー C2245
friend と指定された存在しないメンバー関数 'function' (メンバー関数署名がオーバーロードと一致しません)  
  
 フレンドとして指定された関数がコンパイラによって検出されませんでした。  
  
 次の例では C2245 が生成されます。  
  
```  
// C2245.cpp  
// compile with: /c  
class B {  
   void f(int i);  
};  
  
class A {  
   int m_i;  
   friend void B::f(char);   // C2245  
   // try the following line instead  
   // friend void B::f(int);  
};  
  
void B::f(int i) {  
   A a;  
   a.m_i = 0;  
}  
```
