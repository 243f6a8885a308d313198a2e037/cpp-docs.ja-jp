# コンパイラ エラー C2223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左の\-\>「識別子」は構造体か共用体に  
  
 `->` の左側のオペランドが、クラス、構造体、または共用体へのポインターではありません。  
  
 このエラーは、左側のオペランドが未定義の変数 \(つまり `int` 型\) であることが原因で発生する場合があります。
