# コンパイラ エラー C2242
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

列挙型、構造体、共用体の後に typedef 名を書くことはできません。  
  
 `typedef` 名は修飾名の最後に表示されます。
