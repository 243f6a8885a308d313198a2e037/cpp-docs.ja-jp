# <a name="compiler-error-c2156"></a>コンパイラ エラー C2156
プラグマは、関数の外に指定されなければなりません。  
  
 グローバル レベル (関数本体外) で指定する必要があるプラグマが関数内にあります。  
  
 次の例では C2156 が生成されます。  
  
```  
// C2156.cpp  
#pragma optimize( "l", on )   // OK  
int main() {  
   #pragma optimize( "l", on )   // C2156  
}  
```
