# コンパイラ エラー C2405
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

short 型の前方参照は、ラベルだけを参照するものである必要があります。  
  
 **short** 型の前方参照はラベルだけを参照するものである必要があります。  オフセットを加えることはできません。
