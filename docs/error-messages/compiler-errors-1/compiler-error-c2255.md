# <a name="compiler-error-c2255"></a>コンパイラ エラー C2255
'element' : クラス定義外では使用できません  
  
 たとえば、非メンバー関数が `friend`として宣言されているとします。  
  
 次の例では、C2255 が生成されます。  
  
```  
// C2255.cpp  
// compile with: /c  
class A {  
private:  
   void func1();  
   friend void func2();  
};  
  
friend void func1() {}   // C2255  
void func2(){}  
```
