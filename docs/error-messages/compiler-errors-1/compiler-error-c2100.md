# <a name="compiler-error-c2100"></a>コンパイラ エラー C2100
間接指定演算子 (*) の使い方が正しくありません。  
  
 間接演算子 ( `*` ) が使われて値に適用します。  
  
 次の例では、C2100 が生成されます。  
  
```  
// C2100.cpp  
int main() {  
   int r = 0, *s = 0;  
   s = &r;  
   *r = 200;   // C2100  
   *s = 200;   // OK  
}  
```
