# コンパイラ エラー C2419
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'context' 内で使われた MOD 演算子 '\/' の 2 番目の引数が 0 です。  
  
 `MOD` 演算子の右側のパラメーターが、このコンテキストでは 0 になります。
