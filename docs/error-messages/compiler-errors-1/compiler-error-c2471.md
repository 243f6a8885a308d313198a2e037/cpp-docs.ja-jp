# コンパイラ エラー C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

プログラム データベース 'file' を更新できません。  
  
 データベース ファイルに書き込むことができません。  
  
### この問題を解決するには、次のいずれかを行います。  
  
-   ディスクに十分な空き領域が存在することを確認してください。  
  
-   ファイルが読み取り専用でないことを確認してください。  
  
-   共有違反がないことを確認してください。
