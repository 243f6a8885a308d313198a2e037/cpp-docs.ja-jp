# コンパイラ エラー C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

適合していない文字列を連結しています  
  
 連結対象として、ワイド文字列と非ワイド文字列の両方が指定されました。  ワイド文字列と非ワイド文字列は連結できません。
