# コンパイラ エラー C2295
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

エスケープされた 'character': マクロ定義では正しくありません  
  
 マクロ定義には、指定された文字を持つエスケープ シーケンスを含めることはできません。
