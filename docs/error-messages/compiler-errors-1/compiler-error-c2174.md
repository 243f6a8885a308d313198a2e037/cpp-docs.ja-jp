# <a name="compiler-error-c2174"></a>コンパイラ エラー C2174
'function': 実際のパラメーターが型 'void': パラメーター番号 1、パラメーター リスト番号 2  
  
 パラメーター`number1`パラメーター リストに渡される`number2`は、`void`パラメーター。 パラメーター型を持てません`void`です。 代わりに、`void*` を使用してください。
