# コンパイラ エラー C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 実パラメーターが 'void' 型です : パラメーター number1, パラメーター リスト number2  
  
 パラメーター リスト `number2` に渡されたパラメーター `number1` が `void` 型のパラメーターです。  `void` 型のパラメーターは使用できません。  代わりに、`void*` を使用してください。
