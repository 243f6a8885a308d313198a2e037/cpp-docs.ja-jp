# <a name="compiler-error-c2224"></a>コンパイラ エラー C2224

'.identifier' の左側が構造体、共用体型をいる必要があります。

ピリオド (.) の左側のオペランドがクラス、構造体または共用体。

このエラーは、未定義の変数には、左側のオペランドによって発生することができます (したがって入力`int`)。
