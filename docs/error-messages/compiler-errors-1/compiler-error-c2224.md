# コンパイラ エラー C2224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-\>識別子' : 左側が構造体か共用体の型に対するポインターではありません。  
  
 ピリオド \(.\) の左側のオペランドが、クラス、構造体、または共用体ではありません。  
  
 このエラーは、左側のオペランドが未定義の変数 \(つまり `int` 型\) であることが原因で発生する場合があります。
