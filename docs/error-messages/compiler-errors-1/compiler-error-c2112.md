# コンパイラ エラー C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-' : ポインターを減算するには整数かポインターのオペランドが必要です。  
  
 異なる型を指すポインターの減算が行われました。
