# コンパイラ エラー C2188
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'number': ワイド文字型で保持するには大きすぎます  
  
 数はワイド文字型のサイズの上限を超えています。 より大きい型を選択します。
