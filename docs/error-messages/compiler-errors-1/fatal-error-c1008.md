# <a name="fatal-error-c1008"></a>致命的なエラー C1008

入力ファイルが指定されていません。

コンパイルする C または C++ ソース ファイルがコンパイラーに指定されませんでした。 **CL** コマンド ラインと環境変数のファイル名指定を確認してください。
