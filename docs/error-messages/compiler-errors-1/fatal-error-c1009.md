# <a name="fatal-error-c1009"></a>致命的なエラー C1009

コンパイラの制限: マクロの入れ子のレベルが深すぎます

コンパイラが同時にマクロが多すぎますを展開しようとするとします。 コンパイラでは、マクロの入れ子レベルの 256 の制限があります。 入れ子になったマクロを簡単なマクロに分割します。
