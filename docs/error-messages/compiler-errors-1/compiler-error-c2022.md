# コンパイラ エラー C2022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'数' : 文字として大きすぎます。  
  
 この 8 進数は、文字定数や文字列定数の中で円記号 \(\\\) の後ろに置かれていますが、文字を表すには値が大きすぎます。
