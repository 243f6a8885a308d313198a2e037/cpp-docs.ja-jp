# <a name="compiler-error-c2493"></a>コンパイラ エラー C2493
_ _based の無効な形式  
  
 A`__based`式は、ポインターに基づく必要があります。  
  
 次の例では、C2493 が生成されます。  
  
```  
// C2493.cpp  
// compile with: /c  
char mybase;  
int __based(mybase) ptr;   // C2493  
  
// OK  
char * mybase;  
int __based(mybase) * ptr;  
```
