# <a name="fatal-error-c1603"></a>致命的なエラー C1603
'number' バイトの範囲外のインライン アセンブラー分岐対象  
  
 Jcxz 命令または JECXZ 命令と、指定されたターゲットのラベルの間の距離が 128 バイトを超えています。 ラベルが命令の近くになるように、コードを更新します。
