# 致命的なエラー C1603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

インライン アセンブラー分岐対象の範囲が 'number' バイト超えています。  
  
 JCXZ 命令または JECXZ 命令と指定したターゲット ラベルの間の距離が 128 バイトを超えています。  ラベルが命令の近くになるようにコードを更新してください。
