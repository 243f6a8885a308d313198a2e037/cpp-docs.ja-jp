# 致命的なエラー C1003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

エラーの数が上限を超えました。コンパイルは中断されます。  
  
 プログラム内のエラーが多すぎて回復できません。 コンパイラは強制終了する必要があります。
