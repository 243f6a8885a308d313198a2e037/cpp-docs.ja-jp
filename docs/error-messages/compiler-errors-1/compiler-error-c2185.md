# <a name="compiler-error-c2185"></a>コンパイラ エラー C2185

'identifier': 無効です based の割り当て。

レジスタ変数または自動 (ローカル) 変数が宣言される`__based`します。 グローバル変数のみを宣言する`__based`します。
