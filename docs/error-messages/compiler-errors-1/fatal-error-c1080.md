# 致命的なエラー C1080
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンパイラの制限 : コマンド ライン オプションが制限の number 文字を超えています。  
  
 コンパイラに渡された引数が 256 文字を超えています。
