# <a name="fatal-error-c1201"></a>致命的なエラー C1201
クラス テンプレート定義内の構文エラーの後で続行することはできません  
  
 クラス テンプレートの定義の解析中に予期しないエラーが発生しました。  
  
 他のエラーを修正し、再コンパイルします。 その失敗すると、注意してください、エラーの状況をしようとして問題を特定し、再現可能なテスト_ケースを作成する場合は、連絡[マイクロソフト製品サポート サービス](/visualstudio/ide/talk-to-us)します。
