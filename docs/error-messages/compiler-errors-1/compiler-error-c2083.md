# コンパイラ エラー C2083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構造体\/共用体を比較できません  
  
 構造体または共用体は、別のユーザー定義型と直接比較されます。 比較演算子が定義されているか、スカラー型への変換が存在するのでない限り、これは行えません。
