# コンパイラ エラー C2320
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

アクセス指定子 'specifier' の後には ':' が必要です  
  
 `public`、`protected`、`private` の各キーワードの後にはコロンを付ける必要があります。
