# <a name="compiler-error-c2118"></a>コンパイラ エラー C2118
負の添字  
  
 配列のサイズを定義する値は、配列の最大サイズより大きいまたは 0 未満です。  
  
 次の例では、C2118 が生成されます。  
  
```  
// C2118.cpp  
int main() {  
   int array1[-1];   // C2118  
   int array2[3];   // OK  
}  
```
