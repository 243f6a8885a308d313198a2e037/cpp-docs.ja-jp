# <a name="compiler-error-c2839"></a>コンパイラ エラー C2839
無効な戻り値の型 'type' の 'operator ->' をオーバー ロード  
  
 `->`演算子は、クラス、構造体、または共用体、またはいずれかへの参照を返す必要があります。
