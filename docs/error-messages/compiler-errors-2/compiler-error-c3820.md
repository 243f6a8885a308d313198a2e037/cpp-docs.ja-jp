# コンパイラ エラー C3820
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'name': 初期化子はマネージにする必要があります  
  
 マネージ型の *name* 初期化子は、同様にマネージにする必要があります。  
  
### このエラーを解決するには  
  
1.  マネージ初期化子メソッドを指定します。
