# <a name="compiler-error-c3820"></a>コンパイラ エラー C3820

'name': 初期化子はマネージドにする必要があります

マネージド型の *name* 初期化子は、同様にマネージドにする必要があります。

### <a name="to-correct-this-error"></a>このエラーを解決するには

1. マネージド初期化子メソッドを指定します。
