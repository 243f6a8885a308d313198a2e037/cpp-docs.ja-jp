# <a name="compiler-error-c3240"></a>コンパイラ エラー C3240
'function': 'type' のオーバー ロードされた非抽象メンバー関数でなければなりません  
  
 基本データ型には、定義されている関数が含まれています。 関数は仮想である必要があります。  
  
## <a name="example"></a>例  
 次の例では、C3240 を生成します。  
  
```  
// C3240.cpp  
// compile with: /c  
__interface I {  
   void f();  
};  
  
struct A1 : I {   
   void f() {}  
};  
  
struct A2 : I {   
   void f() = 0;  
};  
  
template <class T>   
struct A3 : T {  
   void T::f() {}  
};  
  
template <class T>   
struct A4 : T {  
   void T::f() {}  
};  
  
A3<A1> x;   // C3240  
A3<I> x2;  
A4<A2> x3;  
```
