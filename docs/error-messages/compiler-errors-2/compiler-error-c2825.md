# <a name="compiler-error-c2825"></a>コンパイラ エラー C2825
var: クラスまたは名前空間が続くと必要があります ':: '  
  
 失敗したときは、修飾名を形成するしました。  
  
 たとえば、コードに関数名が始まる関数宣言が含まれていないことを確認:: です。  
  
## <a name="example"></a>例  
 次の例では、C2825 が生成されます。  
  
```  
// C2825.cpp  
typedef int i;  
int main() {  
   int* p = new int;  
   p->i::i();   // C2825  
   // try the following line instead  
   // p->i::~i();  
}  
```
