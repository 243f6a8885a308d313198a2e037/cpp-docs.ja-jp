# <a name="compiler-error-c2882"></a>コンパイラ エラー C2882
'name': 式の名前空間の識別子の不正な使用  
  
 式の中で名前空間の名前を使用しようとするとします。  
  
 次の例では、C2882 が生成されます。  
  
```  
// C2882.cpp  
// compile with: /c  
namespace A {  
   int k;  
}  
  
int i = A;   // C2882, can't assign A to i  
```
