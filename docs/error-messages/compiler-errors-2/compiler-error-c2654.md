# <a name="compiler-error-c2654"></a>コンパイラ エラー C2654
'identifier': メンバー関数の外でメンバーがアクセスしようとしました  
  
 宣言内でのメンバーにアクセスします。 メンバー データは、メンバー関数でのみアクセスできます。  
  
 宣言内で変数を初期化しようとするときは、このエラーを発生することができます。 この目的のためには、コンス トラクターを使用します。
