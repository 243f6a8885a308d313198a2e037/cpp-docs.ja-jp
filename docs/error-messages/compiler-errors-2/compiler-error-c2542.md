# <a name="compiler-error-c2542"></a>コンパイラ エラー C2542
'identifier': クラスのオブジェクトが初期化にコンス トラクターを持たない  
  
 初期化に一致するパラメーター リストを持つコンス トラクターはありません。  
  
 このエラーは、初期化の正しくないパラメーターで発生することができます。
