# コンパイラ エラー C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : オブジェクトをイニシャライズするためのコンストラクターがありません。  
  
 初期化に指定したのと同じパラメーター リストを持つコンストラクターがありません。  
  
 このエラーは、初期化のパラメーターが間違っていることが原因で発生する場合があります。
