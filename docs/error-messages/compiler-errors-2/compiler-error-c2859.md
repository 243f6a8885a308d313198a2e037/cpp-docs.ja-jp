# コンパイラ エラー C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' は、このプリコンパイル済みヘッダーが作成されたときに使用されたファイル '型' ではありません。プリコンパイル済みヘッダーを再作成してください。  
  
 プロジェクト データベース ファイルとプリコンパイル済みヘッダー ファイルは、情報の整合性を確保するため、共に作成する必要があります。  プロジェクトを再度ビルドして、プリコンパイル済みヘッダーを再作成します。  
  
 プリコンパイル済みヘッダーについては、「[\/Y \(プリコンパイル済みヘッダー\)](../../build/reference/y-precompiled-headers.md)」を参照してください。
