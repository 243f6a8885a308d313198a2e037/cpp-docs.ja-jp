# <a name="compiler-error-c3166"></a>コンパイラ エラー C3166
'pointer': 'type' のメンバーとして内部 _ _gc ポインターへのポインターを宣言することはできません  
  
無効なポインター宣言が見つかりました (、`__nogc`へのポインター、`__gc`ポインターです。)。 
  
C3166 は古い形式のコンパイラ オプションを使用して到達のみ **/clr:oldSyntax**です。
