'symbol': シンボルは、using 宣言で使用することはできません

A`using`ディレクティブがない、[名前空間](../../cpp/namespaces-cpp.md)キーワード。 これは、場合、コンパイラはコードを[宣言を使用して](../../cpp/using-declaration.md)なく[ディレクティブを使用して](../../cpp/namespaces-cpp.md#using_directives)します。
