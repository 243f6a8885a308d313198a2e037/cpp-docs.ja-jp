# コンパイラ エラー C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 関数が、ビット フィールドのメンバーとして宣言されています。  
  
 関数がビット フィールドのメンバーとして宣言されています。  
  
 このエラーは、コンストラクター初期化子リストに構文エラーがあることが原因で発生する場合があります。
