# <a name="compiler-error-c3697"></a>コンパイラ エラー C3697
'修飾子': この修飾子では使用できません ' ^'  
  
 追跡ハンドル (^) は、意図していない修飾子に適用されました。  
  
 次の例では、C3697 が生成されます。  
  
```  
// C3697.cpp  
// compile with: /clr  
using namespace System;  
int main() {  
   String ^__restrict s;   // C3697  
   String ^ s2;   // OK  
}  
```
