# <a name="compiler-error-c3130"></a>コンパイラ エラー C3130
内部コンパイラ エラー: が pdb ファイルに挿入されたコード ブロックを書き込みに失敗しました  
  
 このエラーは、コンパイラが .pdb ファイルに挿入されたコード ブロックを書き込みに失敗した場合に発生します。 エラーの最も一般的な原因は、ディスク領域の不足です。
