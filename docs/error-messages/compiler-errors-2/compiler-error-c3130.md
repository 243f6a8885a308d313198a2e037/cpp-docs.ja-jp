# <a name="compiler-error-c3130"></a>コンパイラ エラー C3130

内部コンパイル エラー: が PDB に挿入されたコード ブロックを書き込みに失敗しました

このエラーは、コンパイラが .pdb ファイルに、挿入されたコード ブロックを書き込みに失敗した場合に発生します。 失敗の最も一般的な理由は、ディスク領域の不足です。
