# コンパイラ エラー C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

内部コンパイラ エラー: 挿入されたコードの PDB への書き込みに失敗しました。  
  
 このエラーが発生するのは、挿入されたコード ブロックを .pdb ファイルに書き込むことができなかった場合です。  通常、このエラーの原因はディスク容量の不足です。
