# コンパイラ エラー C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' : 演算子は、与えられたオペランドには使うことができません。  
  
 この演算子は、与えられたオペランドには使用できません。  必要なオペランド用にオーバーロードされた演算子を用意してください。  
  
 このエラーは、オペランドに不正な型が指定されていると発生する場合があります。  
  
 このエラーは、単一のパラメーターを使用する変換演算子またはコンストラクターを定義すると解決できます。
