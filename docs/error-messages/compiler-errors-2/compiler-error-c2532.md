# <a name="compiler-error-c2532"></a>コンパイラ エラー C2532
'identifier': 参照の修飾子が正しくありません  
  
 参照が変更されました。 参照は、別のオブジェクトを参照してくださいを変更することはできません。 代わりにポインターを使用します。
