# コンパイラ エラー C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 参照の修飾子が正しくありません。  
  
 参照が変更されています。  参照は、ほかのオブジェクトを参照するようには変更できません。  この場合は、ポインターを使ってください。
