# コンパイラ エラー C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コマンド ライン オプション '\/Yc \(\/Fdfilename\)' は '\/Fdfilename' で使われているプリコンパイル済みヘッダーと整合しません。  
  
 プリコンパイル済みヘッダーの使用オプション \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) で、前のプリコンパイル済みヘッダーの作成オプション \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\) と異なるプログラム データベースが指定されています。
