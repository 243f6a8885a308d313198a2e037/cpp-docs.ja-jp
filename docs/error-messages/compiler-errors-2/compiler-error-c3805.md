# コンパイラ エラー C3805
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'トークン' : 予期しないトークン。 '}' か識別子が必要です。  
  
 プロパティを定義するときに、無効なトークンが見つかりました。  無効なトークンを削除してください。
