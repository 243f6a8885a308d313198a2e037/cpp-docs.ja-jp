# <a name="compiler-error-c2529"></a>コンパイラ エラー C2529
'name': 参照への参照は無効です  
  
 このエラーは、ポインターの構文を使用して、ポインターへの参照を宣言することによって解決される可能性があります。  
  
 次の例では、C2529 が生成されます。  
  
```  
// C2529.cpp  
// compile with: /c  
int i;  
int &ri = i;  
int &(&rri) = ri;   // C2529  
```
