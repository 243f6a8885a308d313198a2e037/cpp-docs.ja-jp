# <a name="compiler-error-c3858"></a>コンパイラ エラー C3858
'type': 現在のスコープでは再宣言することはできません  
  
 型は、同じスコープで 2 回宣言することはできません。  
  
 次の例では、C3858 が生成されます。  
  
```  
// C3858.cpp  
// compile with: /LD  
template <class T>  
struct Outer  
{  
   struct Inner;  
};  
  
template <class T>  
struct Outer<T>::Inner;   // C3858  
// try the following line instead  
// struct Outer<T>::Inner{};  
```
