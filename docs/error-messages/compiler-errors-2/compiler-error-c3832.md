# コンパイラ エラー C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type library' : タイプ ライブラリが 32 ビット ポインター用にビルドされた可能性があります。'ptrsize' 修飾子を変更してください。  
  
 [\#import](../Topic/%23import%20Directive%20\(C++\).md) ディレクティブの `ptrsize` 属性で指定されている明示的な情報が、コンパイラがタイプ ライブラリから検出した情報と一致しません。
