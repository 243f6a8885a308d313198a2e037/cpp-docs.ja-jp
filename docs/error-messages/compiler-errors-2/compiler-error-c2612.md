# <a name="compiler-error-c2612"></a>コンパイラ エラー C2612
末尾の 'char' ベース/メンバー初期化子リストについています。  
  
 最後の基底クラスまたは初期化子リスト内のメンバーの後に文字が表示されます。  
  
 次の例では、C2612 が生成されます。  
  
```  
// C2612.cpp  
class A {  
public:  
   int i;  
   A( int ia ) : i( ia ) + {};   // C2612  
};  
```
