# コンパイラ エラー C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無効な ProgID 'progid'  
  
 `#import` ステートメントで無効な progid が指定されています。  Windows レジストリを調べて、有効な progid を指定しているかどうかを確認してください。
