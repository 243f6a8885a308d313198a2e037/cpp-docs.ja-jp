# <a name="compiler-error-c2517"></a>コンパイラ エラー C2517
'identifier': 右の ':: ' は定義されていません  
  
 スコープ解決演算子の右側にある id (`::`) クラス、構造体、または、左側の共用体の定義済みのメンバーである必要があります。 クラス、構造体または共用体の名前がない場合は、グローバル スコープを持つ右側の識別子を宣言する必要があります。
