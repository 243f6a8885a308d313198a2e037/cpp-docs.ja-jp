# <a name="compiler-error-c3050"></a>コンパイラ エラー C3050
'type1': ref クラスは 'type1' から継承できません  
  
 `System::ValueType`参照型の基本クラスをすることはできません。  
  
 次の例では C3050 が生成されます。  
  
```  
// C3050.cpp  
// compile with: /clr /LD  
ref struct X : System::ValueType {};   // C3050  
ref struct Y {};   // OK  
```
