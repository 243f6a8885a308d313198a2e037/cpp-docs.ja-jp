# <a name="compiler-error-c2869"></a>コンパイラ エラー C2869
'name': 名前空間として既に定義されています  
  
 名前空間として既に使用される名前を再利用することはできません。  
  
 次の例では、C2869 が生成されます。  
  
```  
// C2869.cpp  
// compile with: /c  
namespace A { int i; };  
  
class A {};   // C2869, A is already used  
```
