# <a name="compiler-error-c2722"></a>コンパイラ エラー C2722

':: 演算子 ': 次の演算子コマンド; が無効です'operator 演算子' を使用して、

`operator`ステートメントの再定義`::new`または`::delete`します。 `new`と`delete`演算子は、グローバルなため、スコープ解決演算子 (`::`) は意味がありません。 削除、`::`演算子。
