# <a name="compiler-error-c2860"></a>コンパイラ エラー C2860
'void' '(void)' を除く、引数の型にすることはできません  
  
 型`void`他の引数を指定して、引数の型として使用できません。  
  
 次の例では、C2860 が生成されます。  
  
```  
// C2860.cpp  
// compile with: /c  
void profunc1(void, int i);   // C2860  
void func10(void);   // OK  
```
