# <a name="compiler-error-c2510"></a>コンパイラ エラー C2510
'identifier' : '::' の左側は、クラス、構造体、共用体のいずれかでなければなりません  
  
 スコープ解決演算子 (`::`) の左側には、クラス、構造体、または共用体の名前のいずれかが必要です。
