# コンパイラ エラー C3187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\_\_func\_\_' : は関数の本体内でしか使用できません。  
  
 定義済みのマクロは、関数本体の外部で使用できません。  
  
 このエラーを修正するには、マクロを関数の本体の内部に移動します。
