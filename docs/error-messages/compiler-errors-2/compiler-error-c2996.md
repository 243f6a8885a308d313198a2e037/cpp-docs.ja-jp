# コンパイラ エラー C2996
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'関数': 再帰的な関数テンプレートの定義  
  
 関数定義は、そのルートのテンプレート クラスのインスタンスを作成しようとします。 再帰的なテンプレートのインスタンス化は許可されていません。
