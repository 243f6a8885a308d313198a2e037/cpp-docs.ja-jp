# コンパイラ エラー C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'specifier' : ストレージ クラスの指定子が演算子と型の間で誤って指定されています。  
  
 ユーザー定義型の変換はすべてのストレージ クラスに適用されるため、型変換にはストレージ クラスを指定できません。
