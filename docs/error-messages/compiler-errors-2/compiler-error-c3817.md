# <a name="compiler-error-c3817"></a>コンパイラ エラー C3817
'宣言': プロパティは、関数にのみ適用できます  
  
 `property`キーワードが関数定義に適用できるだけです。  
  
 C3817 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。
