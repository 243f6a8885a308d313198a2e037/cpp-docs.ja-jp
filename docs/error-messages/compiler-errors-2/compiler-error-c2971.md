# <a name="compiler-error-c2971"></a>コンパイラ エラー C2971
'class': テンプレート パラメーター 'param': 'arg': 非型引数としてローカル変数を使用することはできません  
  
 テンプレート引数としては、名前またはローカル変数のアドレスを使うことはできません。  
  
 次の例では、C2971 が生成されます。  
  
```  
// C2971.cpp  
template <int *pi>   
class Y {};  
  
int global_var = 0;  
  
int main() {  
   int local_var = 0;  
   Y<&local_var> aY;   // C2971  
   // try the following line instead  
   // Y<&global_var> aY;  
}  
```
