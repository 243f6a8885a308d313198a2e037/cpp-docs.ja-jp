# <a name="compiler-error-c2756"></a>コンパイラ エラー C2756
'template type': 既定のテンプレート引数は部分的特殊化では使用できません  
  
 部分的特殊化用のテンプレートに既定の引数を含めることはできません。  
  
 次の例では、C2756 を生成し、その修正方法を示しています。  
  
```  
// C2756.cpp  
template <class T>  
struct S {};  
  
template <class T=int>  
// try the following line instead  
// template <class T>  
struct S<T*> {};   // C2756  
```
