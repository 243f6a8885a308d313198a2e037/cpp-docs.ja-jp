# <a name="compiler-error-c2881"></a>コンパイラ エラー C2881
'namespace1': 'namespace2' に対するエイリアスとして既に使用されて  
  
 2 つの名前空間の別名として同じ名前を使うことはできません。  
  
 次の例では、C2881 が生成されます。  
  
```  
// C2881.cpp  
// compile with: /c  
namespace A {  
   int k;  
}  
  
namespace B {  
   int i;  
}  
  
namespace C = A;  
namespace C = B;   // C2881 C is already an alias for A  
```
