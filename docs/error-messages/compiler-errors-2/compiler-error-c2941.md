# コンパイラ エラー C2941
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': type\-class\-id がローカル 'item' として再定義されています  
  
 ジェネリック クラスまたはテンプレート クラスを項目として使用することはできません。
