# <a name="compiler-error-c2918"></a>コンパイラ エラー C2918

'name': WinRT 型の発行サーフェスではインデックス付きのプロパティは使用できません

WinRT 型の発行サーフェスではインデックス付きのプロパティはサポートされません。
