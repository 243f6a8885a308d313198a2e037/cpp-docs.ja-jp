# <a name="compiler-error-c2651"></a>コンパイラ エラー C2651
'データ型': 'operator' の左側がクラス、構造体または共用体にする必要があります  
  
 クラスにある場合は、テンプレート パラメーターを使用して、整数型ではなくクラスを持つクラス テンプレートを特殊化です。
