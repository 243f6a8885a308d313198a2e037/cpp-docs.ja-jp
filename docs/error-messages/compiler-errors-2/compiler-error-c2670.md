# コンパイラ エラー C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'識別子' : 関数テンプレートでは、パラメーター '数値' を型 '型' から変換できません。  
  
 パラメーターは、要求された型に変換できませんでした。  
  
 このエラーは、明示的な変換を作成すると解決される場合があります。
