# <a name="compiler-error-c2670"></a>コンパイラ エラー C2670
'identifier': 関数テンプレートは型 'type' からパラメーターの番号を変換することはできません  
  
 パラメーターを必要な型に変換できませんでした。  
  
 明示的な変換を作成する場合、このエラーを修正することがあります。
