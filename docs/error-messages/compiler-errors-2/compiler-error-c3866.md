# <a name="compiler-error-c3866"></a>コンパイラ エラー C3866
関数呼び出しに引数リストがありません。  
  
 非静的メンバー関数の内部デストラクターまたはファイナライザーの呼び出しは、引数リストがありませんでした。  
  
 次の例では、C3866 が生成されます。  
  
```  
// C3866.cpp  
// compile with: /c  
class C {  
   ~C();  
   void f() {  
      this->~C;   // C3866  
      this->~C();   // OK  
   }  
};  
```
