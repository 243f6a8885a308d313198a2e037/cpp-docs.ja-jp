# コンパイラ エラー C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

属性 'attribute' を処理できません。  
  
 `System.Security.Permissions.SecurityAttribute` から継承するユーザー定義の属性が、セキュリティ属性の定義に使用されています。  これらの属性は、定義時と同じアセンブリでは使用できません。
