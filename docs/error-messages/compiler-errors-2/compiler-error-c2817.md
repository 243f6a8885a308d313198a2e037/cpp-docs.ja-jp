# <a name="compiler-error-c2817"></a>コンパイラ エラー C2817
'operator delete' の戻り値の型は 'void' である必要があります。  
  
オーバー ロードされた[演算子 delete](../../standard-library/new-operators.md#op_delete)関数が値を返すことはできません。
