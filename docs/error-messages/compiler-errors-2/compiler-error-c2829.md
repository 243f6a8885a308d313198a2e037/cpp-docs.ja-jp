# コンパイラ エラー C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator operator' に、可変個仮引数リストを指定することはできません。  
  
 可変個のパラメーター リストを取ることができる演算子は次の 2 つだけです。  
  
-   関数呼び出し演算子 `()`  
  
-   `new`
