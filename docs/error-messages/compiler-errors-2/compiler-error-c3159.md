# <a name="compiler-error-c3159"></a>コンパイラ エラー C3159

'pointer': 値型へのポインターの配列を宣言することはできません

値型へのポインターの配列を宣言することはできません。

C3159 は古い形式のコンパイラ オプションを使用して到達のみ **/clr:oldSyntax**します。
