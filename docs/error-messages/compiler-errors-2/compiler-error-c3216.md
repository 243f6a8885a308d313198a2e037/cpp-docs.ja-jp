# <a name="compiler-error-c3216"></a>コンパイラ エラー C3216
制約は、'type' ではなくジェネリック パラメーターでなければなりません  
  
 制約の形式が間違っています。  
  
 次の例では C3216 が生成されます。  
  
```  
// C3216.cpp  
// compile with: /clr  
interface struct A {};  
  
generic <class T>  
where F : A   // C3216  
// Try the following line instead:  
// where T : A    // C3216  
ref class C {};  
```  
  
 次の例では、考えられる解決策を示しています。  
  
```  
// C3216b.cpp  
// compile with: /clr /c  
interface struct A {};  
  
generic <class T>  
where T : A  
ref class C {};  
```
