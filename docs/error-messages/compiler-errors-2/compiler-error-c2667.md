# <a name="compiler-error-c2667"></a>コンパイラ エラー C2667
'function': 最適な変換をある数のオーバー ロードのいずれも  
  
 オーバー ロードされた関数呼び出しがあいまいで、解決できません。  
  
 オーバー ロードされた関数のいずれかの関数呼び出しの実際のパラメーターの照合に必要な変換を必要とされる他のすべてのオーバー ロードされた関数によって変換よりも厳密にする必要があります。  
  
 関数テンプレートの部分的な順序付けの詳細については、サポート技術情報資料 Q240869 を参照してください。
