# <a name="compiler-error-c2892"></a>コンパイラ エラー C2892
ローカル クラスはメンバー テンプレートを含めることはできません。  
  
 Template 宣言されたメンバー関数は、関数で定義されているクラスでは有効ではありません。  
  
 次の例では、C2892 が生成されます。  
  
```  
// C2892.cpp  
int main() {  
   struct local {  
      template<class T>   // C2892  
      void f() {}  
   };  
}  
```
