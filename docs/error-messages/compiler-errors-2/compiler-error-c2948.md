# コンパイラ エラー C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

明示的なインスタンス生成。ストレージ クラス指定子 'specifier' は特殊化では許されていません  
  
 既に明示的にインスタンス化されているテンプレート クラスの特化には、`extern` などのストレージ クラス指定子を使用できません。
