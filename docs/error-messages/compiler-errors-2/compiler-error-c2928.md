# <a name="compiler-error-c2928"></a>コンパイラ エラー C2928

明示的なインスタンス生成。'identifier' はテンプレート クラス 'class' の関数または静的なデータ メンバーのいずれでもありません

関数でも `class` 変数でもない `static` のメンバーは、明示的にインスタンス化できません。
