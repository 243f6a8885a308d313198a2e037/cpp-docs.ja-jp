# <a name="compiler-error-c3396"></a>コンパイラ エラー C3396

'attribute' : カスタム属性が 'namespace' で見つかりませんでした

C3396 は、最新の共通言語ランタイムを使用していないことを示します。  解決するには、CLR を再インストールします。
