# <a name="compiler-error-c3812"></a>コンパイラ エラー C3812
'property' はプロパティの宣言の最初のトークンである必要があります。  
  
 プロパティを宣言するときに、`__property`キーワードは、線上の最初のトークンである必要があります。  
  
 C3812 は古い形式のコンパイラ オプションを使用して到達のみ **/clr:oldSyntax**です。
