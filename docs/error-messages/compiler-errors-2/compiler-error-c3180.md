# コンパイラ エラー C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'型名' : 名前がメタデータの限度である 'limit' 文字を超えています。  
  
 メタデータにあるマネージ型の名前が切り捨てられました。  切り捨てにより、この型は `#using` ディレクティブ \(または別の言語で同義のもの\) で使用できなくなります。  
  
 型名の制限には、名前空間の限定が含まれます。
