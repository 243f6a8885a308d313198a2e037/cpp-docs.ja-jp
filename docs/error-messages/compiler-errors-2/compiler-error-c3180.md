# <a name="compiler-error-c3180"></a>コンパイラ エラー C3180

'type name': 名前がメタデータ制限の 'limit' 文字を超えています

コンパイラでは、メタデータ内のマネージ型の名前が切り捨てられます。 切り捨てによりは、型を使用できない、`#using`ディレクティブ (または別の言語と同等)。

型名の制限には、任意の名前空間の修飾が含まれています。
