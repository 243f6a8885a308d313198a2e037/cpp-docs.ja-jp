# <a name="compiler-error-c2725"></a>コンパイラ エラー C2725
'exception' : 値または参照によってマネージ オブジェクトまたは WinRT オブジェクトをスローまたはキャッチできません  
  
 マネージ例外または WinRT 例外の型が正しくありません。  
  
## <a name="example"></a>例  
 次の例では、C2725 を生成し、その修正方法を示しています。  
  
```  
// C2725.cpp  
// compile with: /clr  
ref class R {  
public:  
   int i;  
};  
  
int main() {  
   R % r1 = *gcnew R;  
   throw r1;   // C2725  
  
   R ^ r2 = gcnew R;  
   throw r2;   // OK     
}  
```  
  
## <a name="example"></a>例  
 次の例では、C2725 を生成し、その修正方法を示しています。  
  
```  
// C2725b.cpp  
// compile with: /clr

   try {}

   // catch( System::Exception ^e) {}  
}  
```
