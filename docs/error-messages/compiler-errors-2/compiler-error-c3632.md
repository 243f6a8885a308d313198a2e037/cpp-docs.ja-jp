# <a name="compiler-error-c3632"></a>コンパイラ エラー C3632

'event': 無効なスタイルのコンストラクトのイベント

[_ _event](../../cpp/event.md)の宣言はすべての構成要素では無効です。

C3632 は古い形式のコンパイラ オプションを使用して到達のみ **/clr:oldSyntax**します。
