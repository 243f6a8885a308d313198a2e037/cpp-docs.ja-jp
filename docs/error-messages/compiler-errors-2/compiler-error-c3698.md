# <a name="compiler-error-c3698"></a>コンパイラ エラー C3698
'type': 'operator' の引数としてこの型は使用できません  
  
 マネージ オブジェクトの宣言が正しくありません。  
  
 次の例では、C3698 が生成されます。  
  
```  
// C3698.cpp  
// compile with: /clr  
  
int main() {  
   array<int>^a = new array<int>^(20);   // C3698  
   array<int>^a2 = gcnew array<int>(20);   // OK  
}  
```
