# <a name="compiler-error-c3126"></a>コンパイラ エラー C3126
共用体のマネージ型 'type' には、' union' は定義できません。  
  
 マネージ型の内部共用体を定義することはできません。  
  
 次の例では、C3126 が生成されます。  
  
```  
// C3126_2.cpp  
// compile with: /clr /c  
ref class Test  
{  
   union x  
   {   // C3126  
      int a;  
      int b;  
   };  
};  
```
