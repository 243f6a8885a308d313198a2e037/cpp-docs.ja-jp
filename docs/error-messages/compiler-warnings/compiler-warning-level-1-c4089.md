# コンパイラの警告 (レベル 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 'number' 番目の実引数と対応する 'number' 番目の仮引数の型が異なります。  
  
 対応する仮パラメーターと実パラメーターの型が異なります。  実パラメーターは、変更されずに渡されます。  関数により、実パラメーターは関数定義で指定された型にキャストされます。
