# <a name="compiler-warning-level-1-c4089"></a>コンパイラの警告 (レベル 1) C4089
'function': 実パラメーター 'number'、'number' の仮パラメーターの種類  
  
 対応する仮引数と実パラメーターは、異なる型を持ちます。 実際のパラメーターは、変更せずに渡されます。 関数は、実パラメーターは、関数定義で指定された型にキャストします。
