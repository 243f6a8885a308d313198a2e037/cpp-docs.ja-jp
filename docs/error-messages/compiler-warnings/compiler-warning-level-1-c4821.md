# <a name="compiler-warning-level-1-c4821"></a>コンパイラの警告 (レベル 1) C4821
Unicode エンコードの種類を指定できません。シグネチャ (BOM) つきファイルを保存してください。  
  
 コンパイラは、ファイルのエンコードの種類を特定できませんでした。 この警告を解決するのには、バイト順マーカーを使用して、ファイルを保存します。 参照してください[エンコード付きでファイルを管理する](http://msdn.microsoft.com/en-us/ab3a74da-5c8f-4680-8882-f1def4909e82)の詳細。
