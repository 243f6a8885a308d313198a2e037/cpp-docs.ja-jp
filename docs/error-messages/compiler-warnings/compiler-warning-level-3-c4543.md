# コンパイラの警告 (レベル 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

属性 'no\_injected\_text' で省略された挿入テキスト  
  
 ソース コードに [no\_injected\_text](../../windows/no-injected-text.md) 属性が使用されています。これは、属性を使用してコードが挿入されないことを意味します。  
  
 この警告は、属性でコードを挿入できないことを示します。
