# <a name="compiler-warning-level-1-c4153"></a>コンパイラの警告 (レベル 1) C4153
関数へのポインターがデータへのポインターに変換されました。  
  
 関数ポインターがデータ ポインターとの間で変換されています。 この変換は、Microsoft 拡張機能 (/Ze) では許可されますが、ANSI C では許可されません。
