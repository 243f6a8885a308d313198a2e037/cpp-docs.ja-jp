# <a name="compiler-warning-level-1-c4375"></a>コンパイラの警告 (レベル 1) C4375
パブリックでないメソッド 'method2' は 'method1' をオーバーライドしません  
  
 別の型を実装する型には、オーバーライド メソッドが定義されているが、上書きが公開されません。 したがって、メソッドは、基本データ型メソッドをオーバーライドしません。
