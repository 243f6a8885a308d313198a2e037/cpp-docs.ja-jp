# コンパイラの警告 (レベル 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class' : コピー コンストラクターを生成することができません。  
  
 コンパイラは、クラスの既定のコピー コンストラクターを生成できませんでした。基本クラスにプライベート コピー コンストラクターがある可能性があります。
