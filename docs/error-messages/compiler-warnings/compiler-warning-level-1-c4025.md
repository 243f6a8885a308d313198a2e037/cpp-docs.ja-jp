# <a name="compiler-warning-level-1-c4025"></a>コンパイラの警告 (レベル 1) C4025
'number': _based ポインターが可変引数を伴う関数に渡されました。パラメーター番号  
  
 可変個の引数のある関数に _based ポインターを渡すと、ポインターが正規化され、予期しない結果が発生します。 可変個の引数のある関数に _based ポインターを渡さないでください。
