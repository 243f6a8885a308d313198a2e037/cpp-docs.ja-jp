# <a name="compiler-warning-level-2-c4007"></a>コンパイラの警告 (レベル 2) C4007

'function': '属性' する必要があります

関数の必須の属性が明示的に記載されていません。 たとえば、関数**メイン**必要があります、`__cdecl`属性。 コンパイラは、属性を強制します。
