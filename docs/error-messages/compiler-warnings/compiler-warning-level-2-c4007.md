# コンパイラの警告 (レベル 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 'attribute' にする必要があります。  
  
 関数の必須属性が明示的に指定されていません。  たとえば、**main** 関数には `__cdecl` 属性が必要です。  コンパイラが強制的に属性を指定します。
