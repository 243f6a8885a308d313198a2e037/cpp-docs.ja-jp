# <a name="compiler-warning-level-3-c4622"></a>コンパイラの警告 (レベル 3) C4622
オブジェクト ファイル 'file' でデバッグ情報はプリコンパイル済みヘッダーを作成したときに上書きされました  
  
 コンパイルされたときに、指定したファイルの CodeView 情報が失われました、 [/Yu](../../build/reference/yu-use-precompiled-header-file.md) (プリコンパイル済みヘッダーを使用する) オプション。  
  
 オブジェクト ファイルの名前を変更 (を使用して[/Fo](../../build/reference/fo-object-file-name.md)) を作成するか、プリコンパイル済みヘッダーを使用してファイルを開き、新しいオブジェクト ファイルを使用してリンクします。
