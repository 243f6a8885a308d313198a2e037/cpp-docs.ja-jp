# <a name="compiler-warning-level-1-c4411"></a>コンパイラの警告 (レベル 1) C4411

'identifier': 置換レジスタ シンボル

識別子は、ローカル シンボルの置換レジスタに解決される、そのため、別のシンボルを持つオペランドで使用する可能性があります。
