# コンパイラの警告 (レベル 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 置換レジスタとなるローカルシンボルなので、別のシンボルをもつオペランドに使われます。  
  
 指定された識別子は変位レジスタとなるローカル シンボルのため、ほかのシンボルを持つオペランドに使用されると思われます。
