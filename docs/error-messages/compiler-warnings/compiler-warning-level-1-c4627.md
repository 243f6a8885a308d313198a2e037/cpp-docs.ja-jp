# <a name="compiler-warning-level-1-c4627"></a>コンパイラの警告 (レベル 1) C4627
'\<識別子 >': プリコンパイル済みヘッダーの使用を探しているときにスキップ  
  
 プリコンパイル済みヘッダーが使用されている場所を探している間に、コンパイラが発生しました、`#include`ディレクティブを*\<識別子 >*ファイルをインクルードします。 コンパイラは無視、`#include`ディレクティブが警告を発行**C4627**プリコンパイル済みヘッダーが含まれていない場合、 *\<識別子 >*ファイルをインクルードします。
