# コンパイラの警告 (レベル 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof 演算子がサイズが 0 となったオペランドに適用されました。  
  
 `sizeof` 演算子が、サイズ 0 になるオペランド \(空の構造体、共用体、クラス、列挙型、または `void` 型\) に適用されました。
