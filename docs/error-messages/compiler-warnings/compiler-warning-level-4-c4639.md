# コンパイラの警告 (レベル 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MSXML エラーです。XML ドキュメント コメントは処理されません。理由  
  
 この警告は、さまざまな理由で発生します。  
  
 この警告を解決するには、次の方法があります。  
  
-   再コンパイルします。  
  
-   共通言語ランタイムを再インストールすることによって MSXML を再インストールします。  
  
-   警告が発生しているドキュメント コメントを編集するか削除し、再コンパイルします。  
  
 C4639 が発生すると、それ以降のすべての XML 処理が無効になり、.xdc ファイルは生成されません。
