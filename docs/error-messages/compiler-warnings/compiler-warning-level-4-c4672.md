# コンパイラの警告 (レベル 4) C4672
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' があいまいです。 1 つ目を 'identifier2' とします。  
  
 **try** ブロックでスローされる指定オブジェクトがあいまいです。 オブジェクトがスローされる場合、そのオブジェクトを特定できません。
