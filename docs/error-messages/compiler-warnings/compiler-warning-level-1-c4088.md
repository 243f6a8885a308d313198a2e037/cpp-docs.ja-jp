# <a name="compiler-warning-level-1-c4088"></a>コンパイラの警告 (レベル 1) C4088
'function': 実パラメーター 'number'、'number' の仮パラメーターのポインターが一致しません  
  
 対応する仮引数と実パラメーターでは、さまざまなレベルの間接参照があります。 実際のパラメーターは、変更せずに渡されます。 呼び出された関数は、その値をポインターとして解釈されます。
