# <a name="compiler-warning-level-1-c4379"></a>コンパイラの警告 (レベル 1) C4379
このコンパイラによって、共通言語ランタイムのバージョンがサポートされていません。 このバージョンを使用すると、予期しない結果が生じる可能性があります。  
  
 以前のバージョンをコンピューターに、共通言語ランタイムのバージョンではない現在があります。 C4379 を解決するには、コンパイラに同梱されている共通言語ランタイムのバージョンをインストールします。
