# コンパイラの警告 (レベル 1) C4379
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

共通言語ランタイムのバージョン 'バージョン' はこのコンパイラによってサポートされていません。このバージョンを使用すると、予期しない結果を発生させる可能性があります  
  
 コンピューターには、以前のバージョンの共通言語ランタイムはありますが、現在のバージョンはありません。  C4379 を解決するには、使用しているコンパイラに付属しているバージョンの共通言語ランタイムをインストールしてください。
