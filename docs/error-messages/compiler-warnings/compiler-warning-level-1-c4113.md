# コンパイラの警告 (レベル 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' はパラメーター リストが 'identifier2' と異なります。  
  
 関数ポインターに別の関数ポインターが割り当てられましたが、両者の仮パラメーター リストが一致しません。  割り当ては変更されずにコンパイルされます。
