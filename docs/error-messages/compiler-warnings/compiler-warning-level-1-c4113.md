# <a name="compiler-warning-level-1-c4113"></a>コンパイラの警告 (レベル 1) C4113

パラメーター リスト 'と 'identifier2' が 'identifier1' とは異なる

関数ポインターは別の関数ポインターに割り当てられますが、関数の仮パラメーター リストが一致しません。 割り当ては、変更しなくてもコンパイルされます。
