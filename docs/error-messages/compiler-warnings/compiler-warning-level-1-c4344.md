# コンパイラの警告 (レベル 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

動作変更 : 明示的テンプレート引数を使用して 'function' への呼び出しを行います。  
  
 明示的なテンプレート引数を使用した関数の呼び出しでは、明示的な引数が指定されていない場合に別の関数を呼び出します。
