# <a name="compiler-warning-level-1-c4024"></a>コンパイラの警告 (レベル 1) C4024
'function': 仮引数と実際のパラメーター 'number' の種類  
  
 対応する仮引数と実パラメーターは、異なる型を持ちます。 コンパイラは、変更なしの実際のパラメーターを渡します。 受信側の関数では、パラメーターの型を予期される型に変換します。
