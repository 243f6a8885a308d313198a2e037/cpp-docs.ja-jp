# コンパイラの警告 (レベル 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : の型が 'number' の仮引数および実引数と異なります。  
  
 対応する仮パラメーターと実パラメーターの型が異なります。  実パラメーターは変更されずに渡されます。  パラメーターを受け取った関数は、その型を必要な型に変換します。
