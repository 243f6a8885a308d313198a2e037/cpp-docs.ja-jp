# コンパイラの警告 (レベル 4) C4263
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : メンバー関数はどの基本クラスの仮想メンバー関数もオーバーライドしません。  
  
 クラスの関数定義の名前が基本クラスの仮想関数と同じ名前ですが、引数の数または型が異なっています。  基本クラスの仮想関数は効果的に隠ぺいされます。  
  
 既定では、この警告はオフに設定されています。  詳細については、「[Compiler Warnings That Are Off by Default](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md)」を参照してください。  
  
 次の例では警告 C4263 が生成されます。  
  
```  
// C4263.cpp  
// compile with: /W4  
#pragma warning(default:4263)  
#pragma warning(default:4264)  
class B {  
public:  
   virtual void func();  
};  
  
class D : public B {  
   void func(int);   // C4263  
};  
  
int main() {  
}  
```
