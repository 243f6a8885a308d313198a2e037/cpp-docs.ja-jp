# コンパイラの警告 (レベル 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

関数の 2 回目の宣言で引数の数が増加しています。  
  
 関数が異なる仮パラメーターで再宣言されています。  最初の宣言で指定された仮パラメーターが使用されます。
