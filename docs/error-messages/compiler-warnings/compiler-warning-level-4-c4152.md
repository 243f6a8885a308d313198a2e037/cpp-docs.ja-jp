# <a name="compiler-warning-level-4-c4152"></a>コンパイラの警告 (レベル 4) C4152
非標準の拡張機能、式における関数/データ間のポインター変換  
  
 関数ポインターがデータ ポインターとの間で変換されています。 この変換は、Microsoft 拡張機能 (/Ze) では許可されますが、ANSI C では許可されません。
