# <a name="compiler-warning-level-1-c4402"></a>コンパイラの警告 (レベル 1) C4402
PTR 演算子を使用する必要があります。  
  
 PTR 演算子が参照するか、インライン アセンブラー コード内の型にキャストするときになしのオペランドでは、型を使用します。  
  
 PTR 演算子と見なされます。
