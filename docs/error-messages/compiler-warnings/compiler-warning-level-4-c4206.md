# コンパイラの警告 (レベル 4) C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**非標準の拡張機能が使用されています : 翻訳単位が空です。**  
  
 プリプロセス後のファイルが空です。  
  
 この拡張仕様を取り入れると、ほかのコンパイラへのコードの移植が難しくなります。  ANSI 互換オプション \([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\) ではエラーが生成され、C のソース コードだけに適用されます。
