# コンパイラの警告 (レベル 1) C4264
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'virtual\_function' : 仮想メンバー関数用に基本クラス 'class' から使用可能なオーバーライドはありません; 関数は隠されます。  
  
 C4264 は、常に [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md) の後に生成されます。  
  
 既定では、この警告はオフに設定されています。  詳細については、「[Compiler Warnings That Are Off by Default](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md)」を参照してください。
