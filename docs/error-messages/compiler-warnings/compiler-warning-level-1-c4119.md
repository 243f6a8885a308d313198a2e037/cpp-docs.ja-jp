# コンパイラの警告 (レベル 1) C4119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

異なったシンボリックベース 'base1' と 'base2' が指定されています。  
  
 2 つの based ポインターはベースが異なるため、互換性がありません。 コンパイラにより、これらのポインターを相互に変換することはできません。
