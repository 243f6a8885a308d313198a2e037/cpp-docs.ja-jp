# <a name="compiler-warning-level-4-c4053"></a>コンパイラの警告 (レベル 4) C4053
'?:' 演算に void 型の式がオペランドとして使われています。  
  
 `?:` 演算子に型 `void`の式が指定されています。 `void` オペランドの値が定義されていません。
