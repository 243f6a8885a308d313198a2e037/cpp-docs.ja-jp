# <a name="compiler-warning-level-1-c4116"></a>コンパイラの警告 (レベル 1) C4116
かっこ内の名前のない種類の定義  
  
 構造体、共用体、または名前のない列挙型は、かっこで囲んだ式で定義されます。 型定義は無意味です。  
  
 C の関数呼び出しでは、定義はグローバル スコープを持ちます。 C++ の関数呼び出しでは、定義は、呼び出される関数と同じスコープをいます。
