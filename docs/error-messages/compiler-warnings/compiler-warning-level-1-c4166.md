# コンパイラの警告 (レベル 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**コンストラクターとデストラクターの呼び出し規約が無効です。**  
  
 コンストラクターとデストラクターは、明示的に **\_\_clrcall** を指定する場合を除き、プラットフォームの既定以外の呼び出し規則を持つことはできません。
