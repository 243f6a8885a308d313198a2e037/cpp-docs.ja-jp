# <a name="compiler-warning-levels-1-and-4-c4223"></a>コンパイラの警告 (レベル 1 および 4) C4223
使用される標準の拡張機能: 左辺値では非配列ポインターに変換  
  
 標準 C での非左辺値配列をポインターに変換できません。 既定の Microsoft 拡張機能 ([/Ze](../../build/reference/za-ze-disable-language-extensions.md))、することができます。
