# <a name="compiler-warning-level-1-c4406"></a>コンパイラの警告 (レベル 1) C4406
ディレクティブは無視のオペランド  
  
 ディレクティブは、オペランドを受け取りませんが、オペランドが指定されました。  
  
 指定したオペランドは無視されます。
