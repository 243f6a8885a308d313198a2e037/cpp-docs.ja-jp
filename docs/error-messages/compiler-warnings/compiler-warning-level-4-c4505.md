# コンパイラの警告 (レベル 4) C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 参照されていないローカル関数は削除されました。  
  
 関数がローカルで、しかもモジュール本体において参照されていないため、実行されないコードになっています。  
  
 この関数のコードは生成されませんでした。
