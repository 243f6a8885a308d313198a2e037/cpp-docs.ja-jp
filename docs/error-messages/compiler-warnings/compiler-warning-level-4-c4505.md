# <a name="compiler-warning-level-4-c4505"></a>コンパイラの警告 (レベル 4) C4505

'function': 参照されていないローカル関数は削除されました

指定された関数は、ローカルおよびモジュールの本体で参照していません。したがって、関数は、実行されないコードです。

コンパイラは、この関数のコードを生成しませんでした。
