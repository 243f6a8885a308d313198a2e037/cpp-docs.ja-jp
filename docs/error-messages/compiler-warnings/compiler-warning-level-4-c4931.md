# コンパイラの警告 (レベル 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

number ビット ポインター用にタイプ ライブラリがビルドされていることを想定します。  
  
 [\#import](../Topic/%23import%20Directive%20\(C++\).md) ディレクティブの **ptrsize** 属性で明示的な情報が指定されていないため、タイプ ライブラリのポインター サイズは ***number*** になりました。  
  
 既定では、この警告はオフに設定されています。  詳細については、「[Compiler Warnings That Are Off by Default](../Topic/Compiler%20Warnings%20That%20Are%20Off%20by%20Default.md)」を参照してください。
