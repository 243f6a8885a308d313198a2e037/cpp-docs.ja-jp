# コンパイラの警告 (レベル 1) C4305
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : truncation from 'type1' to 'type2'  
  
 識別子が小さい型に変換されるため、情報が失われます。  
  
 \[コネクション ポイントの実装\] ウィザードを使用している場合は、以下のサポート技術情報の文書を参照してください。  
  
-   「BUG: ATL Connection Point Wizard Generated Code for Event with VARIANT Argument Gives C4305 Warning \(Q250847\)」
