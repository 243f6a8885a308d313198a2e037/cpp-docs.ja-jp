# <a name="compiler-warning-level-1-c4098"></a>コンパイラの警告 (レベル 1) C4098
'function': void 関数の値を返す  
  
 戻り値の型で宣言された関数[void](../../cpp/void-cpp.md)が、`return`値を返すステートメントです。 コンパイラは型の値を返します`int`です。
