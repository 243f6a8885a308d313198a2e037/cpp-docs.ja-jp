# コンパイラの警告 (レベル 1) C4143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'same\_seg' プラグマはサポートされていません ; 代わりに \_\_based を使用してください。  
  
 **\#pragma same\_seg** は現在サポートされていません。 代わりに [\_\_based](../Topic/Based%20Pointers%20\(C++\).md) キーワードを使用します。
