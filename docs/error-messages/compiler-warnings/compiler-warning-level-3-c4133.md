# コンパイラの警告 (レベル 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type' : 'type1' と 'type2' の間で型に互換性がありません。  
  
 この警告は、型が異なる 2 つのポインターで減算しようとしたことが原因で発生する場合があります。  
  
 この警告を回避するには、適切な型キャストを指定します。
