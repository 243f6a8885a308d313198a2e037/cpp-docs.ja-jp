# <a name="compiler-warning-level-2-c4948"></a>コンパイラの警告 (レベル 2) C4948
'アクセサー' の戻り値の型が対応する setter の最後のパラメーターの型と一致しません  
  
どのようなデータ型が取得されると、インデックス付きプロパティの設定に不整合が検出されました。  
  
C4948 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。
