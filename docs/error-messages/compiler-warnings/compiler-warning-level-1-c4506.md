# コンパイラの警告 (レベル 1) C4506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

インライン関数 'function' は定義されていません。  
  
 関数はインライン関数として宣言されていますが、定義されていません。  
  
 関数のインライン展開は実行されません。  
  
 インライン展開を指定した外部関数が `extern` キーワードによって宣言されているかどうかを確認してください。
