# コンパイラの警告 (レベル 1) C4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

リンケージが extern キーワードなしで指定されています。  
  
 リンケージが、`extern` キーワードなしで指定されています。  非外部型に対するリンケージの指定は不適切です。  
  
 `extern` キーワードが指定されていると見なされます。
