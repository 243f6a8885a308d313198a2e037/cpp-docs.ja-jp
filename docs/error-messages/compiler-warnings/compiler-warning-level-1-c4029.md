# コンパイラの警告 (レベル 1) C4029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

宣言されたパラメーター リストが定義と一致しません。  
  
 関数宣言で指定されている仮パラメーターの型が、関数定義で指定されている仮パラメーターの型と一致しません。  関数定義のパラメーター リストが使用されます。
