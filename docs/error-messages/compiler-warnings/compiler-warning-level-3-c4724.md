# コンパイラの警告 (レベル 3) C4724
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

剰余の 2 番目のオペランドは、コンパイル時に 0 と評価され、不定の結果を返します。  
  
 剰余演算の 2 番目のオペランドはコンパイル時に 0 と評価され、不定の結果が返されます。
