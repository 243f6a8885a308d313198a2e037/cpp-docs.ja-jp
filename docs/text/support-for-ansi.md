# ANSI のサポート
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

MFC フレームワークは全体として、徐々に Unicode 文字セットだけをサポートする方向に向かっていますが、MFC クラスおよびメソッドのほとんどは ANSI 文字セットをサポートしています。  [!INCLUDE[windowsver](../Token/windowsver_md.md)] と Windows コモン コントロール Version 6.1 の継続的な拡張により、いくつかの ANSI クラスおよびメソッドのサポートの使用は推奨されなくなりました。 詳細については、「[サポートされなくなった ANSI API](../mfc/deprecated-ansi-apis.md)」および「[Unicode のサポート](../text/support-for-unicode.md)」を参照してください。
