# C 単項演算子
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

単項演算子はオペランドの前に置かれ、右から左に関連付けられます。  
  
## 構文  
 *unary\-expression*:  
 *postfix\-expression*  
  
 **\+\+**  *unary\-expression*  
  
 `--`  *unary\-expression*  
  
 *unary\-operator cast\-expression*  
  
 **sizeof**  *unary\-expression*  
  
 **sizeof \(**  *type\-name*  **\)**  
  
 *unary\-operator*: 次のいずれか  
 **&**  
 **\* \+ –** `~` **\!**
