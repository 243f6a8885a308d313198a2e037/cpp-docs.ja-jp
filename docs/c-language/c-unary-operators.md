# <a name="c-unary-operators"></a>C 単項演算子

単項演算子はオペランドの前に置かれ、右から左に関連付けられます。

## <a name="syntax"></a>構文

*unary-expression*: *postfix-expression*

**++**  *unary-expression*

`--`  *unary-expression*

*unary-operator cast-expression*

**sizeof**  *unary-expression*

**sizeof (**  *type-name*  **)**

*unary-operator*: **& \* + -** `~` **!** のいずれか
