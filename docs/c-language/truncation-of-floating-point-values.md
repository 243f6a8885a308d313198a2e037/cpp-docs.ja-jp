# <a name="truncation-of-floating-point-values"></a>浮動小数点値の切り捨て
**ANSI 3.2.1.4** 浮動小数点数をより狭い浮動小数点数に変換するときの切り捨てまたは丸めの方向  
  
 アンダーフローが発生すると、浮動小数点変数の値はゼロに丸められます。 オーバーフローは、指定した最適化に応じて、実行時エラーが発生する場合も、予測不可能な値が生成される場合もあります。
