# <a name="largest-array-size"></a>最大配列サイズ

**ANSI 3.3.3.4, 4.1.1**配列の最大サイズ、つまり **size_t** のサイズを保持するために必要な整数の型

`size_t` は、32 ビット x86 プラットフォームでは `unsigned int` です。 64 ビット プラットフォームでは、`size_t` typedef は `unsigned __int64` です。
