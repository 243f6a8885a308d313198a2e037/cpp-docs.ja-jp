# 外部リンケージのない有意文字
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.1.2** 外部リンケージを持たない有意文字の数  
  
 識別子は、247 文字まで有効です。  コンパイラは、識別子で使用できる文字数を制限することなく、単純に制限値を超えた文字を無視します。
