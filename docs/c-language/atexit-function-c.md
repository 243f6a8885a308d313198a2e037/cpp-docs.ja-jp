# <a name="atexit-function-c"></a>atexit 関数 (C)
**ANSI 4.10.4.3** 引数の値が 0、EXIT_SUCCESS、または EXIT_FAILURE 以外の場合に `atexit` 関数が返すステータス  
  
 `atexit` 関数は、成功した場合は 0、失敗した場合は 0 以外の値を返します。
