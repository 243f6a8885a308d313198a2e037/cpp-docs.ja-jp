# <a name="registers-availability-of-registers"></a>レジスタ: レジスタの可用性

**ANSI 3.5.1** register ストレージ クラス指定子を使用してオブジェクトをレジスタに実際に配置できる範囲

コンパイラは、レジスタ変数に対するユーザーの要求を考慮しません。 代わりに、最適化時に独自の選択を行います。
