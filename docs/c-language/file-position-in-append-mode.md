# 追加モードのファイル位置
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** 追加モードのストリームのファイル位置インジケーターが、最初にファイルの先頭または末尾に配置されているかどうか  
  
 ファイルが追加モードで開かれると、ファイル位置インジケーターの最初の位置はファイルの末尾になります。
