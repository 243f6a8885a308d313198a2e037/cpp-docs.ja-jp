# <a name="clock-function-c"></a>clock 関数 (C)
**ANSI 4.12.2.1** `clock` 関数の時代 (年号)  
  
 `clock` 関数の時代 (年号) は、C プログラムの実行が開始されると (0 から) 始まります。 1/**CLOCKS_PER_SEC** (Microsoft C の 1/1000 と同じ) で計測される時刻を返します。
