# 配列の格納
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

配列型に関連付けられたストレージは、そのすべての要素に必要なストレージです。  配列の要素は、最初の要素から最後の要素まで、連続した増加メモリ位置に保存されます。
