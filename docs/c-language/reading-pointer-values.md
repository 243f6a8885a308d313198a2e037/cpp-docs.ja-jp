# ポインター値の読み取り
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.6.2** `fscanf` 関数での **%p** 変換の入力  
  
 **%p** 書式指定文字を指定すると、`fscanf` 関数は、ポインターを 16 進 ASCII 値から正しいアドレスに変換します。
