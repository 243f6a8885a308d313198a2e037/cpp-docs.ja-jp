# C 一次式
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

式のオペランドは "1 次式" と呼ばれます。  
  
## 構文  
 *primary\-expression*:  
 *identifier*  
  
 *constant*  
  
 *string\-literal*  
  
 **\(**  *expression*  **\)**  
  
 *expression*:  
 *assignment\-expression*  
  
 *expression*  **,**  *assignment\-expression*
