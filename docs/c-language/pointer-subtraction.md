# <a name="pointer-subtraction"></a>ポインターの減算
**ANSI 3.3.6, 4.1.1** 同じ配列の要素への 2 つのポインター間の差を保持するために必要な整数型、**ptrdiff_t**  
  
 **ptrdiff_t** は **signed int** です。
