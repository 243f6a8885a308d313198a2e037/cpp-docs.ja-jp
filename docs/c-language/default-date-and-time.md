# 既定の日付と時刻
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.8.8** 翻訳の日時が使用できない場合の \_DATE\_ と \_TIME\_ の定義  
  
 オペレーティング システムが翻訳の日時を提供しないと、\_DATE\_ および \_TIME\_ の既定値は `May 03 1957` および `17:00:00` になります。
