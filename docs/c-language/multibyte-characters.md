# マルチバイト文字
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.1.2** マルチバイト文字のシフト状態  
  
 マルチバイト文字は、Microsoft C も含めて一部の実装で、基本文字セットで表されない外国語文字を表すために使用されます。  ただし、Microsoft C は状態依存のエンコーディングをサポートしません。  したがって、シフト状態はありません。  詳細については「[マルチバイト文字とワイド文字](../Topic/Multibyte%20and%20Wide%20Characters.md)」を参照してください。
