# ファイルのバッファリング
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** ファイルのバッファリングの特性  
  
 標準入出力関数によってアクセスされるディスク ファイルは、完全にバッファリングされます。  既定では、バッファーには 512 バイトが保持されます。
