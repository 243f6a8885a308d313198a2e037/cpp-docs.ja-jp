# 剰余
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.5** 整数除算の剰余の符号  
  
 剰余の符号は、被除数の符号と同じです。  次に例を示します。  
  
```  
 50 / -6 == -8  
 50 % -6 ==  2  
-50 /  6 == -8  
-50 %  6 == -2  
```
