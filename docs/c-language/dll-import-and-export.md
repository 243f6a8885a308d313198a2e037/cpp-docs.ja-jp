# DLL のインポートとエクスポート
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 固有の仕様 →**  
  
 dllimport と dllexport ストレージ クラス修飾子は C 言語への Microsoft 特定の拡張機能です。  これらの修飾子は、クライアント \(実行可能ファイルまたは別の DLL\) に DLL のインターフェイスを定義します。  これらの修飾子の使用に関する特定の情報については、「[dllexport、dllimport](../cpp/dllexport-dllimport.md)」を参照してください。  
  
 **END Microsoft 固有の仕様**
