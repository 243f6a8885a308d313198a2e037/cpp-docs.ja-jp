# 文字列リテラルの概要
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

*string\-literal*:  
 **'** *s\-char\-sequence*  opt **'**  
  
 **L'** *s\-char\-sequence*  opt             **'**  
  
 *s\-char\-sequence*:  
 *s\-char*  
  
 *s\-char\-sequence s\-char*  
  
 *s\-char*:  
 エスケープシーケンス \(二重引用符 \("\)、円記号 \(\\\)、または改行文字\) を除くソース文字セットのメンバー
