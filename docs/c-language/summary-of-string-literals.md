# <a name="summary-of-string-literals"></a>文字列リテラルの概要
*string-literal*:  
 **'** *s-char-sequence* opt**'**  
  
 **L'** *s-char-sequence* opt**'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 *escape-sequence* (二重引用符 (")、円記号 (\\)、または改行文字) を除くソース文字セットのメンバー。
