# テキスト ファイルの切り捨て
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** テキスト ストリームに書き込むと関連付けられたファイルがそれ以降切り捨てられるかどうか  
  
 テキスト ストリームへの書き込みで、それ以降、ファイルが切り捨てられることはありません。
