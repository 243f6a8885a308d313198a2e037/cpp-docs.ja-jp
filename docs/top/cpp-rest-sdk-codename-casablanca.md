# C++ REST SDK (Codename &quot;Casablanca&quot;)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ REST SDK \(コード名 "Casablanca"\) を使用すると、クラウド ベースのサービスに接続する最新の非同期 C\+\+ コードをより簡単に記述できます。  
  
 Casablanca プロジェクトは Visual Studio に付属しなくなりました。  ドキュメントおよびソースがホストされ、CodePlex で更新される場合があります。  このプロジェクトの詳細については、「[C\+\+ REST SDK \(codename "Casablanca"\) \(C\+\+ REST SDK \(コード名 "Casablanca"\)\)](http://casablanca.codeplex.com/)」を参照してください。
