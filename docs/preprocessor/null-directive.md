# <a name="null-directive"></a>Null ディレクティブ
Null プリプロセッサ ディレクティブは、1 つの番号記号 (**#**) 行に単独で記述します。 これは実行されません。  
  
## <a name="syntax"></a>構文  
  
```  
  
#  
  
```
