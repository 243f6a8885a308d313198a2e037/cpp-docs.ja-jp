# auto_search
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 固有の仕様**  
  
 タイプ ライブラリが \#import を使用して参照され、自身が別のタイプ ライブラリを参照している場合、コンパイラが他のタイプ ライブラリに対して暗黙の \#import を実行できることを指定します。  
  
## 構文  
  
```  
auto_search  
```  
  
## 解説  
 **END C\+\+ 固有の仕様**
