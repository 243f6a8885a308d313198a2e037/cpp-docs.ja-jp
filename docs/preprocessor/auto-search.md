# <a name="autosearch"></a>auto_search
**C 固有の仕様**  
  
 タイプ ライブラリが #import を使用して参照され、自身が別のタイプ ライブラリを参照している場合、コンパイラが他のタイプ ライブラリに対して暗黙の #import を実行できることを指定します。  
  
## <a name="syntax"></a>構文  
  
```  
auto_search  
```  
  
## <a name="remarks"></a>コメント  
 **END C 固有の仕様**
