# no_search_namespace
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 固有の仕様**  
  
 [no\_namespace](../Topic/no_namespace.md) 属性と同じ機能ですが、[auto\_search](../preprocessor/auto-search.md) 属性で \#import ディレクティブを使用するタイプ ライブラリに対して使用されます。  
  
## 構文  
  
```  
no_search_namespace  
```  
  
## 解説  
 **END C\+\+ 固有の仕様**
