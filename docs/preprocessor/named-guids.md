# <a name="namedguids"></a>named_guids
**C 固有の仕様**  
  
 定義し、旧スタイルの形式の GUID 変数を初期化するようにコンパイラに指示**LIBID_MyLib**、 **CLSID_MyCoClass**、 **IID_MyInterface**、および**DIID_MyDispInterface**です。  
  
## <a name="syntax"></a>構文  
  
```  
named_guids  
```  
  
## <a name="remarks"></a>コメント  
 **END C 固有の仕様**
