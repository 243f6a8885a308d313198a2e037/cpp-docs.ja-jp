# <a name="namedguids"></a>named_guids
**C++ 固有の仕様**  
  
定義の形式の古いスタイルの GUID 変数を初期化するコンパイラに指示`LIBID_MyLib`、 `CLSID_MyCoClass`、 `IID_MyInterface`、および`DIID_MyDispInterface`します。  
  
## <a name="syntax"></a>構文  
  
```  
named_guids  
```  
  
## <a name="remarks"></a>Remarks  

**END C 固有の仕様**
