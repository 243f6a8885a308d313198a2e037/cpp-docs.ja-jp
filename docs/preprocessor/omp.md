# <a name="omp"></a>omp
省略可能なディレクティブの句と共に、1 つ以上の OpenMP ディレクティブを受け取ります。  
  
## <a name="syntax"></a>構文  
  
```  
#pragma omp directive  
```  
  
## <a name="remarks"></a>Remarks  
 
参照してください[OpenMP ディレクティブ](../parallel/openmp/reference/openmp-directives.md)詳細についてはします。
