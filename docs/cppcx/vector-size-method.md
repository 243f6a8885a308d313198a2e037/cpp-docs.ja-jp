# Vector::Size メソッド
現在のベクター オブジェクトの要素数を返します。  
  
## 構文  
  
```  
  
virtual property unsigned int Size;  
```  
  
## 戻り値  
 現在の Vector 内の要素数。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
