# Map::Size メソッド
マップの [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) 要素の数を返します。  
  
## 構文  
  
```  
  
virtual property unsigned int Size;  
```  
  
## 戻り値  
 マップの要素数。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
