# SizeT::SizeT コンストラクター
指定された値を持つ SizeT の新しいインスタンスを初期化します。  
  
## 構文  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## パラメーター  
 value1  
 符号なし 32 ビット値。  
  
 value2  
 符号なし 32 ビット値へのポインター。
