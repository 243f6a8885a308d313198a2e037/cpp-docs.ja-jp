# Agile::~Agile デストラクター
Agile クラスの現在のインスタンスを破棄します。  
  
## 構文  
  
```cpp  
  
~Agile();  
```  
  
## 解説  
 このデストラクターは、現在の Agile オブジェクトによって表されるオブジェクトも解放します。
