# Array::Value プロパティ
現在の配列へのハンドルを取得します。  
  
## 構文  
  
```  
  
property Array^ Value;  
```  
  
## 戻り値  
 現在の配列へのハンドル。  
  
## 必要条件  
 **ヘッダー:** vccorlib.h  
  
 **名前空間:** Platform
