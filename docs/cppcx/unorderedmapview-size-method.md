# UnorderedMapView::Size メソッド
UnorderedMapView 内の [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) 要素の数を返します。  
  
## 構文  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## 戻り値  
 Unordered MapView の要素数。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
