# Platform::Details::HeapAllocationTrackingLevel 列挙型
この列挙型は内部使用のみを意図しており、開発で使用するためのものではありません。  
  
## 構文  
  
```cpp  
  
enumm class HeapAllocationTrackingLevel;  
```  
  
## コメント  
 この列挙型は、ブラウザーで検査できるため、この列挙型は完全を期すためだけに提供されています。
