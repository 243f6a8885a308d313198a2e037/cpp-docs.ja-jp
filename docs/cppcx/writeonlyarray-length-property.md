# WriteOnlyArray::Length プロパティ
呼び出し元が割り当てた配列内の要素の数を返します。  
  
## 構文  
  
```cpp  
property unsigned int Length{  
   unsigned int get() const;  
}  
```  
  
## 戻り値  
 配列の要素数。  
  
## 必要条件  
 **ヘッダー:** vccorlib.h  
  
 **名前空間:** Platform
