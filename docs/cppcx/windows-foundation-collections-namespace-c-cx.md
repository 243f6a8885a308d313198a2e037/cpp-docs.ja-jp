# Windows::Foundation::Collections 名前空間 (C++/CX)
[!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] は、Vector、VectorView、Map、MapView コレクション クラスの使用を単純化する関数で Windows::Foundation::Collections 名前空間を補完します。  
  
## 構文  
  
```  
  
namespace Windows {  
    namespace Foundation {  
        namespace Collections;  
    }  
}  
```  
  
### 関数  
  
|名前|説明|  
|--------|--------|  
|[back\_inserter 関数](../cppcx/back-inserter-function.md)|コレクションの末尾に値を挿入するために使用できる反復子を返します。|  
|[begin 関数](../cppcx/begin-function.md)|コレクションの先頭を指す反復子を返します。|  
|[end 関数](../cppcx/end-function.md)|コレクションの末尾を越えて指す反復子を返します。|  
|[to\_vector 関数](../cppcx/to-vector-function.md)|コレクションを std::vector として返します。|  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Windows::Foundation::Collections
