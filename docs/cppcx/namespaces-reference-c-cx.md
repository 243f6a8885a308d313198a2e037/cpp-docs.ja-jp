# 名前空間参照 (C++/CX)
ドキュメントのこのセクションにある技術情報では、[!INCLUDE[cppwrt](../cppcx/includes/cppwrt-md.md)] \([!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]\) のコンパイラをサポートする名前空間について説明します。  
  
## コンパイラが指定した名前空間  
 [!INCLUDE[wrt](../cppcx/includes/wrt-md.md)] を対象とするプログラムのコーディングを簡略化するために、[!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)] コンパイラおよびそれをサポートするヘッダー ファイルが広範囲の型を定義する名前空間を提供します。 名前空間は、組み込み数値型と、文字列、配列、およびコレクションと、[!INCLUDE[wrt](../cppcx/includes/wrt-md.md)] エラーを表す Visual C\+\+ の例外と、[!INCLUDE[wrt](../cppcx/includes/wrt-md.md)] の標準型への言語固有の拡張を定義します。
