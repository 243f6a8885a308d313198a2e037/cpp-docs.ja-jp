# InputIterator::operator-&gt; 演算子
現在の InputIterator により指定される要素のアドレスを取得します。  
  
## 構文  
  
```  
pointer operator->() const;  
```  
  
## 戻り値  
 現在の InputIterator により指定される要素のアドレス。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
