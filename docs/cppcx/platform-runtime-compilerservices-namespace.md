# Platform::Runtime::CompilerServices 名前空間
この名前空間は内部使用のみを意図しており、開発で使用するためのものではありません。  
  
## 構文  
  
```cpp  
  
namespace Platform {  
   namespace CompilerServices{  
}}  
```
