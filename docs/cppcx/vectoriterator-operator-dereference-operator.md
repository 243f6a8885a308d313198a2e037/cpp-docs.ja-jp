# VectorIterator::operator* 演算子
現在の VectorIterator により指定される要素のアドレスを取得します。  
  
## 構文  
  
```  
reference operator*() const;  
```  
  
## 戻り値  
 現在の VectorIterator により指定される要素。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
