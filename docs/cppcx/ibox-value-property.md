# IBox::Value プロパティ
このオブジェクトに元から格納されていた値を返します。  
  
## 構文  
  
```cpp  
property T Value {T get();}  
```  
  
```csharp  
  
```  
  
#### パラメーター  
 `T`

## プロパティ値\/戻り値  
 このオブジェクトに元から格納されていた値を返します。
