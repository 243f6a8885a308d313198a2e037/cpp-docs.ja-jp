# IntPtr::ToInt32 メソッド
現在の IntPtr 値を 32 ビット整数に変換します。  
  
## 構文  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## 戻り値  
 32 ビット整数。
