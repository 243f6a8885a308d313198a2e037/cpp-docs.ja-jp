# VectorViewIterator::operator* 演算子
現在の VectorViewIterator により指定された要素への参照を取得します。  
  
## 構文  
  
```  
reference operator*() const;  
```  
  
## 戻り値  
 現在の VectorViewIterator により指定された要素。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
