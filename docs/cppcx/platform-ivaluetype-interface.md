# <a name="platformivaluetype-interface"></a>Platform::IValueType インターフェイス
`Platform::IValueType` は、値クラスと値構造体によって実装されるインフラストラクチャ インターフェイスです。 コードで明示的には使用されません。  
  
## <a name="syntax"></a>構文  
  
```cpp  
interface class IValueType  
```
