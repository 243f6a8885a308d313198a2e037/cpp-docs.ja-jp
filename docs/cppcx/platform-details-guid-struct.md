# Platform::Details::__GUID 構造体
この構造体は内部使用のみを意図しており、開発で使用するためのものではありません。  
  
## 構文  
  
```cpp  
  
ref struct __GUID;  
```  
  
## コメント  
 この構造体は、ブラウザーで検査できるため、完全を期すためだけに提供されています。
