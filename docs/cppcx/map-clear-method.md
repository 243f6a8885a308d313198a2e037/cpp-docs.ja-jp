# Map::Clear メソッド
現在のマップ オブジェクトから、すべてのキー\/値ペアを削除します。  
  
## 構文  
  
```  
  
virtual void Clear();   
```  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
