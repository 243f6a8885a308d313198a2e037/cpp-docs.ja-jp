# Vector::GetView メソッド
Vector の読み取り専用ビュー、つまり IVectorView を返します。  
  
## 構文  
  
```  
  
Windows::Foundation::Collections::IVectorView<T>^   
   GetView();  
```  
  
## 戻り値  
 IVectorView オブジェクト。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
