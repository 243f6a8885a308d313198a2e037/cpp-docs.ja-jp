# String::ToString メソッド (C++/CX)
値が現在の文字列と同じである String オブジェクトを返します。  
  
## 構文  
  
```cpp  
  
String^ String::ToString()  
```  
  
## 戻り値  
 値が現在の文字列と同じである文字列オブジェクト。
