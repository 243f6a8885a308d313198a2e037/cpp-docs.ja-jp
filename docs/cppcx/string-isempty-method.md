# String::IsEmpty メソッド
現在の String オブジェクトが空かどうかを示します。  
  
## 構文  
  
```cpp  
  
bool IsEmpty()  
```  
  
## 戻り値  
 現在の String オブジェクトが `true` または空の文字列 \(""\) の場合は `null`。それ以外の場合は `false`。
