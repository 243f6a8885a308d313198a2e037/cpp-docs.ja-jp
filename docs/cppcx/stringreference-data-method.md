# StringReference::Data メソッド
この `StringReference` のコンテンツを char16 値の配列として返します。  
  
## 構文  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## 戻り値  
 char16 UNICODE テキスト文字の配列。
