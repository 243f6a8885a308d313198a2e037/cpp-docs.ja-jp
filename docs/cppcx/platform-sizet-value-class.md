# Platform::SizeT 値クラス
オブジェクトのサイズを表します。 SizeT は符号なしのデータ型です。  
  
## 構文  
  
```cpp  
public ref class SizeT sealed : ValueType  
```  
  
## メンバー

|----------|--------|  
|[SizeT::SizeT コンストラクター](../cppcx/sizet-sizet-constructor.md)|指定された値で、クラスの新しいインスタンスを初期化します。|
