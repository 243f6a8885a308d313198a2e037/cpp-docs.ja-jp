# StringReference::GetHSTRING メソッド
`__abi_HSTRING` として文字列の内容を返します。  
  
## 構文  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## 戻り値  
 文字列データを格納する `__abi_HSTRING`。
