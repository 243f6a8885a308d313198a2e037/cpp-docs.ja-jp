# UnorderedMap::Clear メソッド
現在の UnorderedMap オブジェクトから、すべてのキー\/値ペアを削除します。  
  
## 構文  
  
```cpp  
  
virtual void Clear();   
```  
  
## 必要条件  
 [Platform::Collections::UnorderedMap クラス](../cppcx/platform-collections-unorderedmap-class.md)**ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
