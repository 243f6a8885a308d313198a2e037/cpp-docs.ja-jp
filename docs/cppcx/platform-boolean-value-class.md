# Platform::Boolean 値クラス
ブール値を表します。`bool` と同じ。  
  
## 構文  
  
```cpp  
public value struct Boolean  
```  
  
## メンバー  
 ブール値には、[Platform::Object クラス](../cppcx/platform-object-class.md) から派生した Equals\(\)、GetHashCode\(\)、ToString\(\) メソッドと、[Platform::Type クラス](../cppcx/platform-type-class.md) から派生した GetTypeCode\(\) メソッドがあります。
