# COMException::COMException コンストラクター
COMException クラスの新しいインスタンスを初期化します。  
  
## 構文  
  
```cpp  
COMException( int hresult )  
```  
  
#### パラメーター  
 hresult  
 例外で表されるエラー HRESULT。
