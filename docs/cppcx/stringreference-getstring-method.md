# StringReference::GetString メソッド
`Platform::String^` として文字列の内容を返します。  
  
## 構文  
  
```scr  
__declspec(no_release_return) __declspec(no_refcount)  
    ::Platform::String^ GetString() const  
```  
  
## 戻り値  
 文字列データを格納する `Platform::String^`。
