# UnorderedMap::Remove メソッド
指定したキー\/値ペアを UnorderedMap オブジェクトから削除します。  
  
## 構文  
  
```cpp  
  
virtual void Remove(  
   K key);  
```  
  
#### パラメーター  
 `key`  
 キー\/値ペアのキー部分。`key` の型は型名 *K* です。  
  
## 必要条件  
 **ヘッダー:** collection.h  
  
 **名前空間:** Platform::Collections
